Definitions - F_CARRIER_TENDER_PERFORMANCE

This fact table stores the carriers tender details.

SHIPMENT_INITIAL_COST
This is the shipment cost that was derived by GC3 when tendering the shipment to the first carrier.
FINAL_ACCEPTED_COST
This is the cost at which the tender was accepted by the carrier. If the first carrier accepted the tender, thsi wil be same as SHIPMENT_INITIAL_COST.
