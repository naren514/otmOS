Definitions - E_SHIPMENT_KEY

A mapping table that stores the GC3 shipment GID and the new calculated Shipment GID for the history database. This is used to avoid the key reuse issue.

AA_SHIPMENT_GID
This is the new unique key.
SHIPMENT_GID
GC3 Shipment GID. Duplicates of this key can exist.
GC3_INSERT_DATE
The INSERT_DATE from GC3. The date when the shipment was created in GC3.
