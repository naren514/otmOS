Definitions - E_LOAD_LOG

This table stores the datetime when the tables were last loaded.

APPLICATION_GID
Currently this will have a value of GC3. In future we colud be loading data from other user aplications.
TABLE_NAME
Advanced analytics table name.
LOAD_TIME
The last load run time.
LOAD_STATUS
The status of the load run. Valid values are 'SUCCESS', 'ERROR'
ERROR_MESSAGE
The Oracle error message, if the status was 'ERROR'.
NUM_OF_RECORDS
Number of records loaded during the last load run.
