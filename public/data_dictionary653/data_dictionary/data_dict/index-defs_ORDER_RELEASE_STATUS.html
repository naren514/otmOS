Definitions - ORDER_RELEASE_STATUS

The status of the order release for a base order. This table links to the defined status(es) in the Status_Type and Status_Value table.

ORDER_RELEASE_GID
A unique identifier created by the system for the order release. the gid contains the domain portion.
STATUS_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
STATUS_VALUE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
