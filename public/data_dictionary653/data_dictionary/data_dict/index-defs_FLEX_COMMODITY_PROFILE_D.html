Definitions - FLEX_COMMODITY_PROFILE_D

This table is used to identify user-defined commodities for the items. The commodities can be specified by any or all of the governmental specifications in HTS, SITC, STCC and NMFC.

FLEX_COMMODITY_PROFILE_GID
FLEX_COMMODITY_PROFILE_GID contains the unique identifier for the flex commodity profile.
COMMODITY_GID
COMMODITY_GID contains the unique identifier for the commodity.
NMFC_CLASS_GID
NMFC_CLASS_GID contains the unique identifier for the National Motor Freight Classification (NMFC).
STCC_GID
STCC_GID contains the unique identifier for the Standard Commodity Code (STCC).
HTS_GID
HTS_GID contains the unique identifier for the Harmonized Tariff System (HTS).
SITC_GID
SITC_GID contains the unique identifier for the Standard International Trade Classification (SITC).
USER_DEFINED_COMMODITY_GID
USE_DEFINED_COMMODITY_GID contains the unique identifier for the user-defined commodity.
