Definitions - DOMAIN_ROLE

This table lists the predefined "roles" that may be assigned to a given domain. These roles include "PLANNING", "CUSTOMER", and "SERVPROV".

DOMAIN_ROLE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
DOMAIN_ROLE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
