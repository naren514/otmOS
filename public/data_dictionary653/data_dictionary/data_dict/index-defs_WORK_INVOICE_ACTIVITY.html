Definitions - WORK_INVOICE_ACTIVITY

lines on the Work Invoice; list of activities performed by driver for the shipment

WORK_INVOICE_GID
Id for the work invoice
SEQUENCE
Line number of the activity record
ACTIVITY_DATE
Date activity was performed
SPECIAL_SERVICE_GID
Special Service Gid for the activity performed
SOURCE_LOCATION_GID
Source location of the activity
DEST_LOCATION_GID
Dest location of the activity
PAYABLE_INDICATOR_GID
Indicates the payable state of this activity (ex: Payable, Not Payable, Display as Zero, etc.)
DISTANCE
Distance for the activity, if applicable
DURATION
Duration for the activity, if applicable
WEIGHT
Weight for the activity, if applicable
VOLUME
Volume for the activity, if applicable
SHIP_UNIT_COUNT
Count for the activity (such as number of ship units), if applicable
SHIPMENT_SPECIAL_SERVICE_SEQ
Shipment special service sequence
