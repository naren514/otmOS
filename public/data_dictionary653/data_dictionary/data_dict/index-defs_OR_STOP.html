Definitions - OR_STOP

This table defines intermediate stops on an order.

OR_STOP_GID
Primary key of the table
OR_STOP_SEQ
Stop sequence number.
LOCATION_GID
Defines the stop location.
EARLY_ARRIVAL_TIME
When IS_APPT is Y, this field is early appointment time.
LATE_ARRIVAL_TIME
When IS_APPT is Y, this field is late appointment time.
IS_APPT
When IS_APPT is Y, EARLY/LATE_ARRIVAL_TIME are appointment times.
IS_APPT_REQUIRED
If appointment is required.
IS_APPT_CONFIRM_REQUIRED
If appointment confirmation is required.
IS_APPT_CONFIRMED
If appointment is confirmed.
LEG_POSITION
define where to insert this stop in a multi-stop situation
