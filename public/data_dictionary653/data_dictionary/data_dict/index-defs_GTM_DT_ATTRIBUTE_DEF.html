Definitions - GTM_DT_ATTRIBUTE_DEF

This table is a master table holds the information about attributes and its type, the java class that implement the copy logic.

GTM_DT_ATTRIBUTE_DEF_GID
gtm_dt_attribute_def_gid contains unique identifier for data transfer attribute
GTM_DT_ATTRIBUTE_DEF_XID
gtm_dt_attribute_def_xid contains unique identifier for data transfer attribute
ASSOCIATE_TRANSLATION_KEY
This field stores string that defines the association type. Ex Trade Transaction or Trade Transcation Line.
ATTRIBUTE_TRANSLATION_KEY
This field stores string that defines the qualifier name. ex :Remark or Refnum
SOURCE_LIST_TYPE1
This field indicates the type of the qualifier for source attribute value. The values can be G/D/P/F. G in case the attribute is a simple attribute. D is used for drop list. P is used for picklist.F is used for Fixed List.
SOURCE_QUERY_CLASS1
This filed indicates the query class name of the source qualifier. This is required in case the value of the field source_list_type is G or P
SOURCE_LIST_TYPE2
This field indicates the type of the qualifier for source attribute value. The values can be G/D/P/F. G in case the attribute is a simple attribute. D is used for drop list. P is used for picklist.F is used for Fixed List.
SOURCE_QUERY_CLASS2
This filed indicates the query class name of the source qualifier. This is required in case the value of the field source_list_type is G or P
TARGET_LIST_TYPE1
This field indicates the type of the qualifier for target attribute value. The values can be G/D/P/F. G in case the attribute is a simple attribute. D is used for drop list. P is used for picklist.F is used for Fixed List.
TARGET_QUERY_CLASS1
This filed indicates the query class name of the target qualifier. This is required in case the value of the field target_list_type is G or P
TARGET_LIST_TYPE2
This field indicates the type of the qualifier for target attribute value. The values can be G/D/P/F. G in case the attribute is a simple attribute. D is used for drop list. P is used for picklist.F is used for Fixed List.
TARGET_QUERY_CLASS2
This filed indicates the query class name of the target qualifier. This is required in case the value of the field target_list_type is G or P
SOURCE1_TRANSLATION_KEY
To store transaltion key for source1
SOURCE2_TRANSLATION_KEY
To store transaltion key for source2
TARGET1_TRANSLATION_KEY
To store transaltion key for target1
TARGET2_TRANSLATION_KEY
To store transaltion key for target2
ASSOCIATION_TYPE
Identifies the objects between which the data configuration is applied. Takes the values SH-TT, OR-TT, OB-TT, TT-DE.
IS_SOURCE1_MANDATORY
Takes the values Y/N. Determines if the source 1 requires input mandatorily or optional.
IS_SOURCE2_MANDATORY
Takes the values Y/N. Determines if the source 2 requires input mandatorily or optional.
IS_TARGET1_MANDATORY
Takes the values Y/N. Determines if the target 1 requires input mandatorily or optional.
IS_TARGET2_MANDATORY
Takes the values Y/N. Determines if the target 2 requires input mandatorily or optional.
SOURCE_FIXED_VALUES1
Takes the list of fixed values if the SOURCE_LIST_TYPE1 is F.
SOURCE_FIXED_VALUES2
Takes the list of fixed values if the SOURCE_LIST_TYPE2 is F.
TARGET_FIXED_VALUES1
Takes the list of fixed values if the TARGET_LIST_TYPE1 is F.
TARGET_FIXED_VALUES2
Takes the list of fixed values if the TARGET_LIST_TYPE2 is F.
