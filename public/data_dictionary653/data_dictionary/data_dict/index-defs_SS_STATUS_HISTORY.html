Definitions - SS_STATUS_HISTORY

This table is used to store the status of the shipment. The status is received via an external transmission. This table stores the event that triggered the status. The events can be: Arrived, Delivered, Delayed, Cargo Problems, Departed, Delivery Problem, Est_Arrival, etc.

SHIPMENT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SERVPROV_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SHIPMENT_STOP_NUM
This indicates the shipment stop number for a stop based event
EVENT_LOCATION_GID
The location identifier of an location based event. This will be null for non-location based events
