Definitions - VESSEL

Stores vessels and information related to each vessel.

VESSEL_GID
Vessel Id
VESSEL_NAME
User specified name for the vessel; typically used if different from the GID
OWNER
Vessel owner
OPERATOR
vessel operator
REGISTRY_PORT_LOC_GID
vessel port of registry
FLAG_COUNTRY_CODE_GID
vessel country
OFFICIAL_NUMBER
official number for vessel
CALL_SIGN
vessel call sign
IMO_NUMBER
7 digit number assigned to all ships
CLASS_SOCIETY_NUMBER
class society number for vessel
YEAR_BUILT
year vessel was built
IS_IN_SERVICE
Must be Y/N to indicate if vessel is in service.
