Definitions - SHIPMENT_REFNUM_QUAL

The master table that stores the descriptions of the qualifiers for the shipment.

SHIPMENT_REFNUM_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SHIPMENT_REFNUM_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
IS_VISIBLE_IF_MASTER_CARR
Identifies whether the reference numbers with this qualifier should display on related shipments.
DEFAULT_REFNUM_BN_TYPE_GID
The BN type used to generate a refnum when the refnum field for a record in the corresponding refnum table is not populated.
IS_REMOVE_ON_SERVPROV_CHANGE
Whether to remove a refnum of this type when a shipment has its servprov changed.
SHIPMENT_REFNUM_SEQUENCE_NO
The sequence number for Refnum Qualifier, to determine the sequence of Refnum Qualifiers on the Shipment Manager.
