Definitions - LOAD_CONFIG_SETUP_ORIENTATION

MAX_TOP_WEIGHT
Maximum allowable weight can be loaded on top of this type of unit when loaded with the orientation. This is only checked for non-like units. It does not apply when stacking like units.
MAX_LENGTH_OVERHANG
When loaded in the given orientation, maximum unsupported overhang along the length of the unit itself.
MAX_WIDTH_OVERHANG
When loaded in the given orientation, maximum unsupported overhang along the width of the unit itself.
