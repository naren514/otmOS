Definitions - RATE_GEO_TIME_PERIOD

This table represents an actual instance of a time-period on a particular lane (as defined by the owning rate offering) where more than one shipment has been counted for historical purposes.

RATE_GEO_GID
Owning rate geo
RATE_GEO_TIME_PERIOD_SEQ
System generated sequence number
START_DATE
Start date of the time period
END_DATE
End date of the time period
