Definitions - CR_SOLUTION

CR Solution results for the CR Solver run.

START_TIME
Start time of the CR Solver process.
END_TIME
End time of the CR Solver process.
NUM_ITERATIONS
Number of solver iterations before the solution was produced.
NUM_LVFS
Number of LV Forecasts brought into the solution.
TOTAL_VOLUME
Total volume on the LV Forecasts brought into the solution.
NUM_CRS
Number of CR routes created in the solution.
CR_VOLUME
Total lane volume on the CR routes created by the solution.
CR_COST
Total cost of the CRs created in the solution.
TOTAL_COST
Total cost for the solution, including CR route cost and residual cost.
NUM_RESOURCES_USED
Number of fleet resources used for CR routes created in the solution.
AVG_CR_VOLUME
Average volume on CR routes created in the solution.
NUM_LVFS_ON_CRS
Number of LVFs at least partially covered by CR routes.
SAVINGS
Total savings of the solution.
SAVINGS_PERCENT
Total savings compared with cost if every forecast were at market cost.
STATUS_CODE
Code indicating status of the CR solve.
