Definitions - NMFC_ARTICLE_CLASS_MAP

This table is a linkage between NMFC_ARTCLE and NMFC_CLASS tables.

NMFC_ARTICLE_GID
NMFC_ARTICLE_GID contains the unique identifier for the National Motor Freight Classification (NMFC) article for the item being shipped.
NMFC_CLASS_GID
NMFC_CLASS_GID contains the unique identifier for the National Motor Freight Classification (NMFC) class for the item being shipped.
