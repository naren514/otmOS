Definitions - MATCH_RULE_REFNUM

The value of the specified match refnum qualifier will be matched against the corresonding match with refnum value for the match refnum qualifer table (m_refnum_qual_table_name) and match with refnum qualifier table (mw_refnum_qual_table_name).

MATCH_RULE_GID
Id of the match rule
MATCH_RULE_REFNUM_SEQ_NO
match rule refnum sequence
M_REFNUM_QUAL_TABLE_NAME
It indicate the match refnum qualifier table (e.g., invoice_tn_refnum_qual).
M_REFNUM_QUAL_GID
It indicate the match refnum qualifier ID(e.g., invoice_tn_refnum_qual)
MW_REFNUM_QUAL_TABLE_NAME
It indicate the match with refnum qualifier table (e.g., shipment_refnum_qual, su_refnum_qual).
MW_REFNUM_QUAL_GID
It indicate the match with refnum qualifier id (e.g., shipment_refnum_qual).
