Definitions - ORDER_RELEASE_ACCESSORIAL

This table consists the links to the Accessorial table to define the accessories required to fulfill the order at the order release level. It supercedes the accessories defined at the order base level. For e.g., a pump may be required to pump out the chemicals from a delivery truck. Accessorials are different from special services in that the customers are charged for the accessories.

ORDER_RELEASE_GID
A unique identifier created by the system for the order release. the gid contains the domain portion.
ACCESSORIAL_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
