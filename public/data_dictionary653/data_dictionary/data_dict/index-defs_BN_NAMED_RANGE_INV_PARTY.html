Definitions - BN_NAMED_RANGE_INV_PARTY

Defines involved parties that should be notified when a named range event is raised.

BN_NAMED_RANGE_GID
The named range to which this involved party applies
INVOLVED_PARTY_QUAL_GID
The involved party qualifier.
INVOLVED_PARTY_CONTACT_GID
The contact to which the message should be sent.
COM_METHOD_GID
The communication method.
