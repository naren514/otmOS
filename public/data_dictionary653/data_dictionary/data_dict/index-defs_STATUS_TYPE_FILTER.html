Definitions - STATUS_TYPE_FILTER

This table defines a user-defined filter for status types that is applied to a specific object type (business object)

STATUS_TYPE_FILTER_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
STATUS_TYPE_FILTER_XID
GC3 Unique identifier for the entity. Is used for creating the GID
OBJECT_TYPE
Business object type
