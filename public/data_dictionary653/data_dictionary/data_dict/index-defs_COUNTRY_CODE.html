Definitions - COUNTRY_CODE

COUNTRY_CODE stores a three-digit code for every country, which is standard data for the countries. This table also stores (for reference only) two-digit codes for those countries that have them. GC3 always uses the three-digit code country code.

COUNTRY_CODE3_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COUNTRY_CODE3_XID
GC3 Unique identifier for the entity. Is used for creating the GID
COUNTRY_NAME
COUNTRY_NAME contains the country name.
COUNTRY_ZONE_GID
COUNTRY_ZONE_GID contains the unique identifier for the country zone. Zones can include: AFR ANT ASI AUS CAM CAR EAF EUR FAR MID NAM NPL SAF SAM SAO
IS_EU_MEMBER_COUNTRY
Indicates if country is member of European Union.
VALIDATE_POSTAL_CODE
This flag should be set to 'Y' if address validation has been loaded into the POSTAL_CODE_VALIDATE table for this particular country.
CAR_REGISTRATION_CODE
1-3 character car registration code.
IS_INSPECTION_REQUIRED
Inspection is required for this country. Default is 'N'
IS_COLLECT_ALLOWED
Collect is allowed for this country. Default is 'Y'
BOL_TYPE
BOL Type. Copied down to shipment
BOL_ISSUANCE_TYPE
BOL Issuance Type. Copied down to shipment
IS_MEMO_BL_ALLOWED
Indicate if this country allows Memo B/L
IATA_REGION
The IATA region code for this country. Will be 1, 2, or 3
