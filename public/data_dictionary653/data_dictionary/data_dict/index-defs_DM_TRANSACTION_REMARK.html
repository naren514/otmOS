Definitions - DM_TRANSACTION_REMARK

Qualified remarks about the demurrage transaction (qualifiers in table REMARK_QUAL). This may have similar information to the DM_TRANSACTION_NOTE table, except that the remarks are qualified, while notes are freeform. Example: Qual: Order Out reason Remark: Equipment Demaged..

DM_TRANSACTION_SEQ_NO
DM_TRANSACTION_SEQ_NO contains the unique identifier for the demurrage transaction.
REMARK_SEQUENCE
REMARK_SEQ_NO contains a sequence number for a remark on an demurrage transaction.
REMARK_QUAL_GID
Remark Qualifier
REMARK_TEXT
REMARK_TEXT contains the text for the remark on an demurrage transaction.
