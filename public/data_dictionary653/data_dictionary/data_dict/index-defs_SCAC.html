Definitions - SCAC

The Standard Carrier Alpha Codes is a code assigned to Transportation Companies to be used for identification.

SCAC_GID
The unique 2 to 4 letter Standard Carrier Alpha Code.
SCAC_XID
The 2 to 4 letter Standard Carrier Alpha Code.
COMPANY_NAME
The Company Name of the Standard Carrier Alpha Code.
DESCRIPTION
The Description of the Standard Carrier Alpha Code.
