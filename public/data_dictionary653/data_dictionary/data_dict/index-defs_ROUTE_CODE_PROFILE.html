Definitions - ROUTE_CODE_PROFILE

ROUTE_CODE_PROFILE table defines the profile of route codes, which allows flexibility to group route codes in a profile and specify if they are compatible for rebate functionality.

ROUTE_CODE_PROFILE_GID
ROUTE_CODE_PROFILE_GID is the unique identifier for the route code profile.
ROUTE_CODE_PROFILE_XID
ROUTE_CODE_PROFILE_XID is the unique external identifier for the route code profile.
ROUTE_CODE_PROFILE_NAME
ROUTE_CODE_PROFILE_NAME is text for the name of the route code profile
IS_COMPATIBLE
A true or false indicator that identifies whether the route codes are compatible.
