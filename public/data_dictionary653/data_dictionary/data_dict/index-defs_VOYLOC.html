Definitions - VOYLOC

This table defines the individual locations within the voyage path. Each location can be a Pickup or Delivery point, identified by a "D" for Pickup and "A" for Delivery in the database.

VOYAGE_GID
VOYAGE_GID contains the unique identifier for the voyage.
SEQ_NO
SEQ_NO contains the sequence number for the location of the voyage.
LOCATION_GID
LOCATION_GID contains the unique identifier for the location of the voyage.
D_OR_A
D_OR_A contains defines whether the location is a pickup (D) or delivery (A) point for the voyage.
D_OR_A_DATE
D_OR_A_DATE contains the date for the pickup (D) or delivery (A) for the voyage.
ESTIMATED_D_OR_A_DATE
estimated date at port sometimes sent by carrier
ACTUAL_D_OR_A_DATE
actual date at port sometimes sent by carrier
