Definitions - PACKAGING_FORM_CODE

THIS TABLE STORES THE EDI 211 PACKAGING FORM CODES. THE FORM CODE DESCRIBES THE PACKAGING FORM BUT NOT THE TYPE OF PACKAGING FORM. FOR EXAMPLE CTN WOULD MEAN CARTON AND THIS WOULD COVER ALL TYPES AND SIZES OF CARTON. LINKS TO THIS TABLE ARE CURRENTLY INCLUDED IN THE ITEM TABLE AS WELL AS THE SHIP_UNIT_SPEC TABLE.

PACKAGING_FORM_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PACKAGING_FORM_CODE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
