Definitions - SHIPMENT_AGENT_RSP

This table keeps the shipping agent contacts and their responsible shipments relationship and their approval/decline status for the shipments.

SHIPMENT_GID
One of the primary keys.
SHIPPING_AGENT_CONTACT_GID
One of the primary keys.
JOB_GID
Indicates which Job the shipment is in.
APPROVED
A:APPROVED, D:DECLINED, O:OUTSTANDING
ADJUSTMENT_REASON_GID
If APPROVED field is D, this field is required.
ADJUSTMENT_TEXT
If APPROVED field is D, this field is optional to keep the extra info about the declined shipment.
