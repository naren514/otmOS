Definitions - LOAD_CONFIG_SETUP

STACKING_INDEX
Stacking index of the ship unit spec. Units with higher stacking index should not be loaded on top of units with lower stacking index.
MAX_STACKING_HEIGHT
Defines the height this unit cannot be stacked above since unloading would be difficult otherwise.
MAX_STACKING_LAYERS
Number of layers this unit can be stacked all by itself.
IS_STACKABLE_ABOVE
To indicate if anything can be stacked on top of it.
IS_STACKABLE_BELOW
To indicate if anything can be stacked below it.
STACKING_RULE
This is an INCLUSIVE rule. SAME means SAME items can be stacked together. LIKE means SAME or LIKE items can be stacked together. Null means any items can be stacked together.
