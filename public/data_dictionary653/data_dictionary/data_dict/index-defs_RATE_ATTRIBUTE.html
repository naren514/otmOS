Definitions - RATE_ATTRIBUTE

Contains the valid listing of attributes, or rate characteristics. Each attribute is associated with a java class. Each java class will contain one or more fields, which determines the fields the user sees for entering this type of rate information. One or more attributes are then associated with the offering type within the RATE_ATTRIBUTE_JOIN table.

RATE_ATTRIBUTE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
RATE_ATTRIBUTE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
