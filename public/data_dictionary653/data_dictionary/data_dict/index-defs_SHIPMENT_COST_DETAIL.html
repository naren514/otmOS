Definitions - SHIPMENT_COST_DETAIL

The shipment_cost_detail table contains an explanation of how each shipment cost was calculated. The explanation is stored in the DETAIL column.

SHIPMENT_COST_SEQNO
Foreign key to shipment_cost table.
DETAIL_SEQNO
A one-up number to make the cost detail record unique.
DETAIL
The textual description of the cost detail as calculated and applied by the rating engine.
