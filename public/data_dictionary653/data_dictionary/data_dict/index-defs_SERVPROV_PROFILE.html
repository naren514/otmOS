Definitions - SERVPROV_PROFILE

This table allows creation of profiles to group the service providers. The compatible service providers can be grouped. For example US TL Carriers, Italian TL Carriers etc.

SERVPROV_PROFILE_GID
A unique identifier to define the group profile for multiple service providers. The gid contains the domain name.
SERVPROV_PROFILE_XID
Similar to the gid except for the domain name.
SERVPROV_PROFILE_NAME
A user defined profile (group) name.
IS_COMPATIBLE
A true or false indicator that identifies whether the service providers are compatible between the groups. This is used when planning the shipments.
