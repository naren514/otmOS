Definitions - DRIVER_CALENDAR_EVENT

The table captures all the Calendar events (excluding shipment assignments) for a Driver

DRIVER_CAL_EVENT_GID
The primary key for this table. If the user doesn't specify a GID, BNG will be used to generate a GID
DRIVER_GID
The Driver GID in OTM
EVENT_START_TIME
Start Time of the Activity for this particular Driver and Event
EVENT_END_TIME
End Time of the Activity for this particular Driver and Event
EVENT_LOCATION_GID
The location where this calendar event is scheduled
CAL_EVENT_TYPE_GID
The type of the Calendar event
