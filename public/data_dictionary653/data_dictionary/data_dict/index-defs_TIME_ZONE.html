Definitions - TIME_ZONE

This table is used to define the various time zones across the countries. This is used to schedule shipments and also to achieve full globalization in the GC3. It works with the Time_Zone_Locale table to identify the countries, states and cities that are part of the defined time zones.

TIME_ZONE_GID
TIME_ZONE_GID contains the unique identifier for a time zone. Examples include Australia/Sydney and Europe/Istanbul.
TIME_ZONE_XID
TIME_ZONE_XID contains the unique external identifier for the time zone.
TIME_ZONE_NAME
TIME_ZONE_NAME contains the name of the time zone used for localization.
