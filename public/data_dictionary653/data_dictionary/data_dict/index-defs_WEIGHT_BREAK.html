Definitions - WEIGHT_BREAK

The weight break table holds the discounts a carrier may give to a customer who is shipping large quantities. Each weight break is then associated with a profile in the WEIGHT_BREAK_PROFILE table. Ex: Of a Weight Profile: ABC Carriers. Ex of a weight break: 100-200 lbs. $2.00 /lb 201-300 lbs. $1.75 /lb

WEIGHT_BREAK_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
WEIGHT_BREAK_XID
GC3 Unique identifier for the entity. Is used for creating the GID
WEIGHT_BREAK_PROFILE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
