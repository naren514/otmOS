Definitions - GTM_RULE_RESTRICTION

GTM_CONTROL_TYPE_GID
Determines if this restriction is related to a license, embargo, or document
CONTROL_CODE
This will be a reference to a license_type_gid, embargo_regulation_gid, or document_gid
GTM_VALUE_SET_GID
Value Set ID assigns the value qualifiers from it to a transaction/transactionline for VALUE SET category
GTM_VALUE_QUALIFIER_GID
Value Qualifier ID to get the associated formula for VALUE FORMULA category
GTM_FORMULA_EXPRESSION_GID
Formula Expression ID to be associated to a particular value qualifier for VALUE FORMULA category
VALUE_FORMULA_RANK
Value Formula rank helps to decide the priority in case multiple formulae are associated to a particular value qualifier
