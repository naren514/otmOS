Definitions - CR_PROJECT

A Cooperative Route Project for definining an aggregation.

START_DATE
Start of date range for aggregation.
END_DATE
End of date range for aggregation.
RATE_DISTANCE_GID
Rate Distance for determining distances within this project.
CR_FORECAST_RULE_GID
The rule indicating the basis for lane volume forecasting.
FORECAST_ADJUSTMENT_PERCENT
Denotes the % value by which to adjust the basis for lane volume forecasting. Can be negative.
FORECAST_ADJUSTMENT_STDDEV
Denotes the multiples of standard deviation by which to adjust the basis for lane volume forecasting. Can be negative.
SCENARIO_SOLUTION_GID
Denotes the scenario whose solution has been chosen as the solution for this project.
