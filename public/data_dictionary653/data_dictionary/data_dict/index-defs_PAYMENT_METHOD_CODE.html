Definitions - PAYMENT_METHOD_CODE

This table defines the various methods by which the payment can be made on an order. For example: CC = Collect, CD = Collect on Delivery, MX = Mixed, NC = Service Freight no Charge, PP = Prepaid, TP = Third Party

PAYMENT_METHOD_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PAYMENT_METHOD_CODE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
PAYMENT_METHOD_CODE_TYPE
Distinguish if this code is Commercial or Standard, or Both.
