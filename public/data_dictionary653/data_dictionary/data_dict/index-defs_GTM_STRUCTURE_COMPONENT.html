Definitions - GTM_STRUCTURE_COMPONENT

Trade Item Structure Component captures the component (quantity and % on item value) details of a trade item structure.

GTM_STRUCTURE_GID
GTM_STRUCTURE_GID contains the unique identifier for the trade item structure.
GTM_STRUCTURE_COMPONENT_SEQ
Trade Item Structure Component Sequence Number
ITEM_GID
Component Identification ID. Component can be identified either by Trade Item ID or by Product Classification Code
DESCRIPTION
Component Description
GTM_PROD_CLASS_CODE_GID
Component Identification ID. Component can be identified either by Trade Item ID or by Product Classification Code
COUNTRY_OF_ORIGIN
Components country of origin
