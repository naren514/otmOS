Definitions - OB_LINE_STATUS

The status of the line item within a base order. This table links to the defined status(es) in the Status_Type and Status_Value table.

OB_LINE_GID
OB_LINE_GID contains the unique identifier for the order base line item.
STATUS_TYPE_GID
STATUS_TYPE_GID contains the unique identifier for the status type for the order base.
STATUS_VALUE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
