Definitions - RATE_PREFERENCE

This table contains the unique carrier preference identifiers, along with the effective and expiration dates. The child table that contains the carriers' rate offerings that belong the preference is called RATE_PREFERENCE_DETAIL.

RATE_PREFERENCE_GID
Unique identifier of preferred carriers for a lane or zone.
RATE_PREFERENCE_XID
External identifier of the preferred carriers for a lane or zone.
RATE_PREFERENCE_NAME
Name of the preferred carriers for a lane or zone.
X_LANE_GID
The lane for which this preference is in effect.
RATE_ZONE_PROFILE_GID
The zone for which this preference is in effect.
EFFECTIVE_DATE
The date the preferred carriers list becomes effective.
EXPIRATION_DATE
The date the preferred carriers list expire.
