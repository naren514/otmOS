Definitions - DM_RULE

Demurrage Rule provides the capability to define possible rules for the determination of demurrage, detention and storage chargeable days and charges.

DM_RULE_GID
DM_RULE_GID contains the unique identifier for the demurrage rule.
DESCRIPTION
Description of the Demurrage Rule
IS_ACTIVE
When N, indicates demurrage rule is inactive and should not be used by the algorithm.
EFFECTIVE_DATE
The Effective Date for this Demurrage Rule
EXPIRATION_DATE
The Expiration Date for this Auto Approval Rule.
CR_DM_SCAC_GID
Criteria to define rules based on demurrage reporting SCAC. In other words the enable rule to be defined specific to each service provider.
CR_START_SCAC_GID
Criteria to define rules based on demurrage start event reporting SCAC.
CR_END_SCAC_GID
Criteria to define rules based on demurrage end event reporting SCAC.
CR_EQUIPMENT_MARK_LIST_GID
Criteria to define rules based on equipment marks list.
CR_EQUIPMENT_NUMBER_LOW
Criteria to define rules based on equipment number range (low).
CR_EQUIPMENT_NUMBER_HIGH
Criteria to define rules based on equipment number range (high).
CR_EQUIPMENT_GROUP_PROFILE_GID
Criteria to define rules based on equipment group profile.
CR_FLEX_COMMODITY_PROFILE_GID
Criteria to define rules based on commodity.
RATE_OFFERING_GID
Rate Offering linked to demurrage rule. Its helps to define/link demurrage charges.
RATE_GEO_GID
Rate Geo linked to demurrage rule. Its helps to define/link demurrage charges.
CR_DM_REGION_GID
Criteria to define rules based on demurrage location. One can use region to use the same rule for multiple location, station etc.
CR_START_REGION_GID
Criteria to define rules based on demurrage start event location. One can use region to use the same rule for multiple start location, station etc.
CR_END_REGION_GID
Criteria to define rules based on demurrage end event location. One can use region to use the same rule for multiple event location, station etc.
CR_AAR_CAR_TYPE
AAR Car Type.
CR_EQUIPMENT_OWNER_TYPE
The Type of Owner for this Equipment Type. C = Carrier Owned, P = Private, null = All.
