Definitions - RATE_ATTRIBUTE_JOIN

Joins together the offering type (such as air) to the attributes required for entering that type of offering. For example, air travel might require a user to enter maximum weight allowed, handling charges, and general contract info. Each of these attributes can be associated with one or more fields. These three attributes would then be associated with the "AIR" offering type, which would then determine the fields that appear to the user for entering rate information.
