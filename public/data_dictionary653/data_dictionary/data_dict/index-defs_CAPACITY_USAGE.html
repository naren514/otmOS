Definitions - CAPACITY_USAGE

Store a service provider's capacity usage by capacity group, equipment type, lane and time period

CAPACITY_USAGE_GID
Unique identifier for Capacity Usage. This consists of the XID and the Domain Name
CAPACITY_USAGE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
CAPACITY_LIMIT_GID
Unique Identifier for Capacity Limit records. This is a combination of CAPACITY_LIMIT_XID and DOMAIN_NAME
CAPACITY_GROUP_GID
Unique Identifier for each CAPCITY GROUP. This is a combination of XID and Domain Name
EQUIPMENT_TYPE_GID
A unique identifier to define and group the equipments within the system. The gid contains the domain portion.
X_LANE_GID
A unique identifier to define the X lanes in the GC3. The gid contains the domain portion. The X-Lane is the user interface in GC3 to define all the lanes.
