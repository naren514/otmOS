Definitions - INVOICE_SPLIT_RULE

Invoice_Split_Rule table contains the rules which are used to split Invoices/Bills into multiple invoices or bills for a single shipment. Invoice_Split_Rule table will define one of the valid ways that we can look at a shipment (buy or sell) and generate multiple bills or invoices from it based on the split rule that is being used.

INVOICE_SPLIT_RULE_GID
INVOICE_SPLIT_RULE_GID is the unique identifier for the INVOICE_SPLIT_RULE
INVOICE_SPLIT_RULE_XID
INVOICE_SPLIT_RULE_XID is the unique external identifier for the INVOICE_SPLIT_RULE
INVOICE_SPLIT_RULE_NAME
INVOICE_SPLIT_RULE_NAME contains the name of the rule for splitting invoice.
INVOICE_SPLIT_RULE_DESC
INVOICE_SPLIT_RULE_DESC contains the description about split rule.
INVOICE_SPLIT_RULE_TYPE
INVOICE_SPLIT_RULE_TYPE indicates whether this RULE is applicable for a bill or an invoice (valid values are I or B)
INVOICE_SPLIT_RULE_CRITERIA
invoice_split_rule_criteria will define the criteria which is base for splitting the invoice. Valid Inputs are:L,T,A,S; L represents - PER COST LINE, T represents - PER COST TYPE, A represents - PER ACCESORIAL.
