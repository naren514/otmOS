Definitions - TENDER_COLLABORATION

This tables stores the shipment tender information. Among other things, it contains the service provider name, expected response time and the actual response (Accepted/ Declined) from the service provider.

SHIPMENT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
TENDER_TYPE
Defines the type of tender. Values are 'Ordinary', 'Broadcast Tender', and 'Spot Bid'
ORIGIN_LOCATION
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
DESTINATION_LOCATION
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
RATE_GEO_GID
References the rate record that that is associated with the tender.
PLANNED_COST
Copied planning cost from shipment.
PLANNED_COST_BASE
Copied planning cost from shipment in base units.
PLANNED_COST_CURRENCY_GID
Unit of measure for the planned cost.
