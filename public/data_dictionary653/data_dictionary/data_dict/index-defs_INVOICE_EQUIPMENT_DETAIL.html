Definitions - INVOICE_EQUIPMENT_DETAIL

One record per piece of equipment per shipment on the invoice. Equipment that is related to the shipment for which this invoice is related.

INVOICE_GID
INVOICE_GID is the unique identifier for the invoice.
EQUIPMENT_PREFIX
EQUIPMENT_PREFIX identifies the prefix of the equipment that was used for the shipment that relates to this invoice.
EQUIPMENT_NUMBER
EQUIPMENT_NUMBER contains the number for the equipment.
ISO_EQUIPMENT_TYPE_IDENTIFIER
The ISO equipment type identifier.
DESCRIPTION_CODE
Description code.
EQUIPMENT_OWNER_IDENTIFIER
The identifier of the equipment owner.
OWNERSHIP_CODE
Ownership code.
LENGTH
LENGTH contains the length of the equipment.
LENGTH_UOM_CODE
LENGTH_UOM_CODE identifies the unit of measure for the LENGTH.
LENGTH_BASE
LENGTH_BASE contains the length of the equipment in the base unit of measure.
WIDTH
WIDTH contains the width of the equipment.
WIDTH_UOM_CODE
WIDTH_UOM_CODE identifies the unit of measure for the WIDTH.
WIDTH_BASE
WIDTH_BASE contains the width of the equipment in the base unit of measure.
HEIGHT
HEIGHT contains the height of the equipment.
HEIGHT_UOM_CODE
HEIGHT_UOM_CODE identifies the unit of measure for the HEIGHT.
HEIGHT_BASE
HEIGHT_BASE contains the height of the equipment in the base unit of measure.
WEIGHT
WEIGHT contains the weight of the equipment.
WEIGHT_UOM_CODE
WEIGHT_UOM_CODE identifies the unit of measure for the WEIGHT.
WEIGHT_BASE
WEIGHT_BASE contains the weight of the equipment in the base unit of measure.
VOLUME
VOLUME contains the volume of the equipment.
VOLUME_UOM_CODE
VOLUME_UOM_CODE identifies the unit of measure for the VOLUME.
VOLUME_BASE
VOLUME_BASE contains the volume of the equipment in the base unit of measure.
TARE_WEIGHT
Weight of equipment when empty.
TARE_WEIGHT_UOM_CODE
Unit of measure for the tare weight.
TARE_WEIGHT_BASE
The tare weight in base units.
DUNNAGE
Dunnage.
DUNNAGE_UOM_CODE
Dunnage unit of measure code.
DUNNAGE_BASE
Dunnage in base unit of measure.
UNIT_COUNT
Unit count.
WEIGHT_QUALIFIER
Weight qualifier.
MINIMUM_TEMPERATURE
MINIMUM_TEMPERATURE contains the minimum temperature for the equipment.
MIN_TEMP_UOM_CODE
The minimum temperature unit of measure.
MINIMUM_TEMPERATURE_BASE
The minimum temperature in base units.
MAXIMUM_TEMPERATURE
MAXIMUM_TEMPERATURE contains the maximum temperature for the equipment.
MAX_TEMP_UOM_CODE
The maximum temperature unit of measure.
MAXIMUM_TEMPERATURE_BASE
The maximum temperature in base units.
PERCENT_HUMIDITY_ALLOWED
PERCENT_HUMIDITY_ALLOWED contains the maximum percentage of humidity allowed for the equipment.
VENT_SETTING_CODE
The vent setting code.
WEIGHT_ALLOWANCE
WEIGHT_ALLOWANCE contains the regulatory maximum weight for the equipment.
WEIGHT_ALLOWANCE_UOM_CODE
The weight allowance unit of measure.
WEIGHT_ALLOWANCE_BASE
The weight allowance in base units.
