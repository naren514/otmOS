Definitions - GTM_STRUCTURE_REFNUM

GTM_STRUCTURE_REFNUM contains a series of business numbers stored with a trade item structure. These numbers are used to qualify the trade item structure. For example, LOTnumber etc

GTM_STRUCTURE_GID
GTM_STRUCTURE_GID contains the unique identifier for the trade item structure.
GTM_STRUCTURE_REFNUM_QUAL_GID
GTM_STRUCTURE_REFNUM_QUAL_GID contains the unique identifier for the trade item structure refnum qualifer. example LOT ID as the qualifier etc
GTM_STRUCTURE_REFNUM_VALUE
GTM_STRUCTURE_REFNUM_VALUE contains the qualfiier value. for example lot number -1456
