Definitions - FLEX_FIELD_DEF

This table defines ATTRIBUTE fields for OTM object tables. The defined ATTRIBUTE field can be used as PickList, DropList, FixedDropList, FlexPickList or FlexDropList on Manager Layout and Screen Set.

FLEX_FIELD_DEF_GID
Primary key of the table.
TABLE_NAME
a valid OTM table name which has flex fields
FIELD_NAME
FLex field name which this definition is for.
ML_TYPE
field type on manager layout, pl(pickList), dl(dropList), fxdl(fixedDropList), flxpl(flexPickList), flxdl(flexDropList)
ML_QUERY_TABLE_GID
The value will used by PickList or DropList on Manager Layout
ML_FINDER_SET_GID
The value will used by PickList on Manager Layout
ML_VALID_VALUE_TYPE_GID
Required for fixedDropList type. Valid Values from this ID will be options for this field on Manager Layout
ML_QUERY
SQL select statement to be used for this field on Manager Layout
SS_TYPE
field type on screen set, pl(pickList), dl(dropList), fxdl(fixedDropList), flxpl(flexPickList), flxdl(flexDropList)
SS_QUERY_TABLE_GID
The value will used by PickList or DropList on Screen Set
SS_FINDER_SET_GID
The value will used by PickList on Screen Set
SS_VALID_VALUE_TYPE_GID
Required for fixedDropList type. Valid Values from this ID will be options for this field on Screen Set
SS_QUERY
SQL select statement to be used for this field on Screen Set
