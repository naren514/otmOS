Definitions - REPORT_SYSTEM

Definition of an external report generation system

REPORT_SYSTEM_GID
Report System ID
REPORT_SYSTEM_XID
Report System XID
REPORT_SYSTEM_TYPE
Report System domain
BIP_HOST
Host name for remote BI Publisher server
BIP_PORT
Port number for remote BI Publisher server. Defaults to 9704 if left blank.
BIP_USER
BI Publisher user
BIP_PASSWORD
BI Publisher password
REPORT_PATH_PREFIX
Optional path prefix for remote reports. The REPORT.REPORT_PATH field is concatenated to the report path prefix to determine the full path name of the remote report.
HTTP_URL
For external system supporting a simple HTTP request (i.e. not BI Publisher), the URL for the report. The URL should have a {report} macro that will be replaced with the full report path request.
PROTOCOL
The protocol we want to use to connect to BI Publisher
