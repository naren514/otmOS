Definitions - VOUCHER

VOUCHER_GID
VOUCHER_GID contains the unique identifier for the voucher.
VOUCHER_XID
VOUCHER_XID contains the unique external identifier for the voucher.
VOUCHER_NUMBER
VOUCHER_NUMBER contains the number of the voucher.
INVOICE_GID
INVOICE_GID contains the unique identifier for the voucher.
WAS_AUTO_APPROVED
WAS_AUTO_APPROVED contains a Y for yes if the voucher was automatically approved or an N for no if not.
APPROVED_BY
APPROVED_BY contains the name of the person who has approved payment for the voucher.
AMOUNT_TO_PAY
AMOUNT_TO_PAY contains the dollar amount to be paid for the voucher.
AMOUNT_TO_PAY_BASE
AMOUNT_TO_PAY in base units.
AMOUNT_TO_PAY_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ADJUSTMENT_REASON_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
EXCHANGE_RATE_DATE
Indicates the date to be used for any currency conversions; if null, the latest conversion should be used.
EXCHANGE_RATE_GID
Indicates the set of exchange rates to use for currency conversions; if null, the default set should be used
AMT_TO_PAY_WITH_TAX
The Amount to Pay with Tax for the Voucher.
AMT_TO_PAY_WITH_TAX_CURR_GID
The Unique Currency Gid for the Amount to Pay with Tax.
AMT_TO_PAY_WITH_TAX_BASE
The Base amount for the Amount to Pay with Tax.
AMOUNT_TO_PAY_FN
functional currency amount for amount_to_pay
AMT_PAY_W_TAX_FN
functional currency amount for amt_to_pay_with_tax
