Definitions - ORDER_RELEASE_LINE

ORDER_RELEASE_LINE contains the individual line items for an order release. (See also OB_LINE.) For example, if the order base contains: 500 Nike shoes from Philadelphia to N.Y. at the latest by Jan. 1, 2001, the order release may contain a release of 400 Nike shoes by Dec. 1, 2000 and the line item for the order release may contain 400 Nike shoes from Philadelphia to N.Y. and 400 socks from Nashville to N.Y.

Indicates Dangerous Goods (DG) are being shipped under one of the special provisions listed in the IATA Dangerous Goods Regulation Guide, section 4.4. There appear to be 146 different provisions, and multiple provisions may apply to a single item in some cases.
