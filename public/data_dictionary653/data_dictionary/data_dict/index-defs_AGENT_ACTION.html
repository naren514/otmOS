Definitions - AGENT_ACTION

AGENT_ACTION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
AGENT_ACTION_XID
GC3 Unique identifier for the entity. Is used for creating the GID
DATA_QUERY_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
HELP_DESCRIPTION
Non-editable help for the action that is delivered as part of GC3.
COMPATIBLE_AGENT_EVENT_CLASS
The event class with which the the action is compatible.
IS_PRE_PERSISTENCE_COMPATIBLE
Indicates whether the action is compatible with pre-persistence lifetime events.
IS_POST_PERSISTENCE_COMPATIBLE
Indicates whether the action is compatible with post-persistence lifetime events.
IS_POST_REMOVAL_COMPATIBLE
Indicates whether the action is compatible with post removal lifetime events.
IS_LONG_RUNNING_TASK
If Y, this agent action is a long running task and should be monitored in the LONG_RUNNING_TASK table.
LRT_INTERRUPTABILITY
Interruptability support: NONE = task cannot be interrupted; INTERRUPTIBLE = task can be interrupted, likely ending in a failure state; RESUBMITTABLE: task can be resubmitted into it's origin queue for handling by another machine
LRT_INTERRUPT_ON_SQL
If 'Y', a task interruption will force the next SQL statement to fail.
