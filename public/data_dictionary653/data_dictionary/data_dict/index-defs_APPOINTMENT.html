Definitions - APPOINTMENT

Storing information of shipment appointment, including fixed appointments, feasible appointments and infeasible appointments. It also stores any blocked appointment time.

APPOINTMENT_GID
The unique identifier.
APPOINTMENT_XID
The ID of the location resource appointment.
LOCATION_RESOURCE_GID
The GID of the location resource.
STOP_NUM
The stop number of the shipment.
APPOINTMENT_START_TIME
Appointment start time.
APPOINTMENT_END_TIME
Appointment end time.
IS_FEASIBLE
Whether or not the appointment is feasible.
IS_FIXED
If the shipment has arrived at the stop, the appointment will be fixed.
IS_BLOCKED
Whether or not the appointment time is blocked.
CONFIRMATION_NUMBER
The appointment confirmation number.
INFEASIBLE_REASON
If the IS_FEASIBLE is false, the INFESIBLE_REASON will provide the reason for it.
DATA_QUERY_TYPE_GID
The Gid of Data_Query_Type. Current valid values in this column will either be SHIPMENT or SHIPMENT GROUP
MARKED_FOR_PURGE
A boolean field. When it is set to "Y", this row and its child data will be purged.
DOMAIN_NAME
The domain of the user.
