Definitions - RATE_OFFERING_TYPE

A list of valid values for the types of contracts. For example, the type of offering might be "AIR" to indicate air travel. The rate_offering_type is joined with rate attributes in the RATE_ATTRIBUTE_JOIN table to determine the fields the user sees for entering rate information.

RATE_OFFERING_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
RATE_OFFERING_TYPE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
