Definitions - OUT_XML_PROFILE_XPATH

This table, with the OUT_XML_PROFILE table, provides additional flexibility in specifying outbound xml profiles. This table provides the ability to spcecify the element to the excluded via a simplified XPath expression.

OUT_XML_PROFILE_GID
Reference to parent outbound xml profile.
SEQUENCE_NO
Sequence number.
XML_PATH
Specifies the XPath for the element.
