Definitions - GTM_BOND

This table stores the bond data.

GTM_BOND_GID
This is the bond identifier Gid for Gtm system.
GTM_BOND_XID
This is the bond identifier Xid for Gtm system.
GTM_BOND_NUMBER
This is the bond number provided by the government.
GTM_PRIOR_BOND_NUMBER
This is the previous bond number related to current bond.
GTM_BOND_TYPE1_GID
This is the type of bond.
GTM_BOND_CODE1
This is the primary bond code.
GTM_BOND_TYPE2_GID
This will always be same as first bond type but provided additionally in database to support foreign key constraint for gtm_second_bond_code.
GTM_BOND_CODE2
This is the additional bond code applicable for selected bond type.
IS_SUPPLEMENTAL
This identifies whether the bond is supplemental.
IS_EXTENSION
This identifies whether the bond is an extension.
EFFECTIVE_DATE
This is the date from when bond will be applicable.
EXPIRATION_DATE
This is the date when bond will expire.
LIABILITY_LMT
This is the liability limit of the bond.
LIABILITY_LMT_BASE
This is the liability limit of the bond in default currency.
LIABILITY_LMT_CURRENCY_GID
This is the currency unit for the liability limit of the bond.
DESCRIPTION
This is the description of the bond.
EXECUTION_DATE
This is the date when the bond is approved or sold.
