Definitions - NFRC_RULE

NFRC_RULE_GID
Unique identifier for NFRC_RULE
CHARGE_RULE_TYPE
Determines which sets of fields are displayed in the GUI and checked during charge rule application. J = Job, O = Order, B = Buy Shipment, S = Sell Shipment, A = All
EFFECTIVE_DATE
Compared against the start time of the primary leg shipment in the job. Used for all charge types.
EXPIRATION_DATE
Compared against the start time of the primary leg shipment in the job. Used for all charge types.
JOB_INVOLVED_PARTY_QUAL_GID
Compared against the list of associated involved parties for the job. Used for all charge types.
JOB_LOCATION_PROFILE_GID
Compared against the contact locations for the involved parties of a job. Used for all charge types.
OR_ORIGIN_REGION_GID
Compared against the order source location of all orders in a job. Used for O charge type.
OR_DESTINATION_REGION_GID
Compared against the order destination location of all orders in a job. Used for O charge type.
OR_ORDER_RELEASE_TYPE_GID
Compared against the order release type of all orders in a job. Used for O charge type.
OR_MIN_TOTAL_WEIGHT
Compared against the summed total weight of all orders in a job. Used for O charge type.
OR_MAX_TOTAL_WEIGHT
Compared against the summed total weight of all orders in a job. Used for O charge type.
OR_MIN_TOTAL_VOLUME
Compared against the summed total volume of all orders in a job. Used for O charge type.
OR_MAX_TOTAL_VOLUME
Compared against the summed total volume of all orders in a job. Used for O charge type.
OR_TRANS_HANDLING_UNIT_GID
Compared against the transport handling units (ship unit specs) for all orders in a job. Used for O charge type.
BUY_LOCATION_PROFILE_GID
Compared against all buy shipment stop locations for all shipments in a job. Used for B charge type.
BUY_MODE_PROFILE_GID
Compared against the transportation mode of the primary leg buy shipment. Used for B charge type.
BUY_PORT_OF_LOAD_LOC_PROF_GID
Compared against the port of load on the primary leg buy shipment. Used for B charge type.
BUY_PORT_OF_DSCH_LOC_PROF_GID
Compared against the port of discharge on the primary leg buy shipment. Used for B charge type.
SELL_LOCATION_PROFILE_GID
Compared against all sell shipment stop locations for all shipments in a job. Used for S charge type.
SELL_MODE_PROFILE_GID
Compared against the transportation mode of the primary leg sell shipment. Used for S charge type.
SELL_SERVPROV_PROFILE_GID
Compared against the service provider of the primary leg sell shipment. Used for S charge type.
SELL_PORT_OF_LOAD_LOC_PROF_GID
Compared against the port of load on the primary leg sell shipment. Used for S charge type.
SELL_PORT_OF_DSCH_LOC_PROF_GID
Compared against the port of discharged on the primary leg sell shipment. Used for S charge type.
ACCESSORIAL_CODE_GID
Assigned to the shipment(s) as a shipment_accessorial. Also, copied to the shipment_name field of the shipment(s).
PERSPECTIVE
Perspective of the resulting shipment(s). B = Buy, S = Sell, null = both (2 shipments, one Buy, one Sell)
IS_CALCULATE_COST
Boolean indicator for whether or not to attemnpt to calculate the cost. If true, the user interface will require a rate record id.
BUY_RATE_GEO_GID
The rate record used to calculate the buy shipment charge.
BUY_PAYMENT_METHOD_CODE_GID
The payment method code used with the buy shipment charge.
BUY_SERVPROV_GID
The service provider associated with the buy shipment charge. Overrides the servprov specfied on the rate specified in buy_rate_geo_gid.
SELL_RATE_GEO_GID
The rate record used to calculate the sell shipment charge.
SELL_PAYMENT_METHOD_CODE_GID
The payment method code used with the sell shipment charge
SELL_SERVPROV_GID
The service provider associated with the sell shipment charge. Overrides the servprov specified on the rate specified in sell_rate_geo_gid
BUY_GENERAL_LEDGER_CODE_GID
Code ID for the resulting buy shipments
SELL_GENERAL_LEDGER_CODE_GID
Code ID for the resulting sell shipments
GENERATION_TYPE
Defines if the rule should be considered by the NFRC rule engine or not
CONSOLIDATION_TYPE_PROF_GID
Consolidation type profile of matching jobs. NULL means consolidation type doesn't matter.
IS_PROFIT_SPLIT
Indicates the created NFRC shipment will be profit split or not.
