Definitions - CLAIM_COST

Stores costs associated with a claim

CLAIM_GID
Claim id that this cost is attached to
COST_SEQ_NO
Sequence number to keep costs unique
COST
Value of the claim cost
COST_CURRENCY_GID
Currency gid for the claim cost
COST_BASE
Base value for the claim cost
CLAIM_COST_TYPE_GID
Cost type of the claim cost. These values are a combination of PUBLIC types shippped with GC3 and types specified by the user. The cost type will indicate if this cost should be added or subtracted from the claim damage/loss value.
