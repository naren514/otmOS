Definitions - ORDER_MOVEMENT

Order movement represent part of the routing of an order. It is also a collection of ship units that are unplanned for certain part of a route.

ORDER_MOVEMENT_GID
the unique identifier of an order movement
IS_TEMPORARY
indicated whether this order movement can coexist with shipments. If this flag is true, the order movement will be removed once the order movement is planned on shipments. Otherwise the order movement's status is set to PLANNED-FINAL and stays in the system.
PERSPECTIVE
indicate whether this is BUY or SELL
SOURCE_LOCATION_GID
the source of the order movement. It can be null
IS_FIXED_SRC
indicate whether the source location on this order movement is the real source location or it is derived from the source location of its upstream shipments.
DEST_LOCATION_GID
the destination of the order movement. It can be null.
IS_FIXED_DEST
indicate whether the dest location of this order movement is real or it is derived from the downstream shipments.
EARLY_PICKUP_DATE
the earliest pickup of this order movement
OPERATION_EARLY_PICKUP_DATE
the date is calculated from upstream shipments' end time
LATE_PICKUP_DATE
the latest pickup time of this order movement
EARLY_DELIVERY_DATE
the earliest delivery of this order movement
LATE_DELIVERY_DATE
the latest delivery of this order movement
OPERATION_LATE_DELIVERY_DATE
this date is calculated from downstream shipments' start time
ORIGINAL_LEG_POSITION
The position of the original leg. Main leg's leg position is 0. Legs left to main leg have positions -1, -2, etc. Legs right to the main leg have positions 1, 2, ....
