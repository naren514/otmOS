Definitions - GTM_DT_CONFIG_ATTRIBUTE

This table holds the values and relation between data transfer configuration and data transfer attribute defination table

GTM_DT_CONFIG_GID
gtm_dt_config_gid contains unique identifier for data transfer configuration
GTM_DT_ATTRIBUTE_DEF_GID
gtm_dt_attribute_def_gid contains unique identifier for data transfer attribute
SEQUENCE_NUMBER
This stores the sequence number entered by the user for each attribute in UI
