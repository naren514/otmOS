Definitions - INVOICE_LINEITEM

One record per line item within an invoice. The line item has no direct relation to the S_SHIP_UNIT_LINE table, since invoice lines consist of a variety of things, including accessorial charges.

INVOICE_GID
INVOICE_GID is the unique identifier for the invoice.
LINEITEM_SEQ_NO
Line item sequence number.
DESCRIPTION
DESCRIPTION contains text describing the line item on the invoice.
UNIT_COUNT
Unit count for that line.
BILLABLE_INDICATOR_GID
indicator used to tell if this lineitem is billable (sell side bill only)
FLEX_COMMODITY_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
FLEX_COMMODITY_CODE
Flex commodity code.
WEIGHT
Total weight for line item.
VOLUME
Total volume for line item.
WEIGHT_UOM_CODE
Weight unit of measure.
VOLUME_UOM_CODE
Volume unit of measure.

Weight in base units.
VOLUME_BASE
Volume in base units.
WEIGHT_QUALIFIER
Weight qualifier.

FREIGHT_RATE_VALUE_BASE

FREIGHT_CHARGE_BASE

PREPAID_AMOUNT

PREPAID_AMOUNT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.

PAYMENT_METHOD_CODE_GID is the unique identifier for the method of payment for the invoice. Examples of payment include cash on delivery, check and electronic funds transfer.
ACCESSORIAL_CODE_GID
ACCESSORIAL_CODE_GID is the unique identifier for an accessorial code.
ACCESSORIAL_DESCRIPTION
ACCESSORIAL_DESCRIPTION contains text to describe the accessorial.
COMPARTMENT_ID_CODE
COMPARTMENT_ID_CODE contains the identification number for the equipment's compartment for this line item.

VAT_CODE_GID
