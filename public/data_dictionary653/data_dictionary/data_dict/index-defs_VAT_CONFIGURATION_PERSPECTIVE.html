Definitions - VAT_CONFIGURATION_PERSPECTIVE

GOODS_LOCATION_TYPE
Determines whether orgin and destination of goods is based on SHIPMENT origin or ORDER origin.
CUSTOMER_INV_PARTY_QUAL_GID
This is the involved party which determines the "customer" for the purpose of determining VAT. Usually BILL-TO
IS_ALLOCATE
Determines if VAT should be allocated. If not, order with VAT_OUTCOME with highest priority is used for the entire line.
