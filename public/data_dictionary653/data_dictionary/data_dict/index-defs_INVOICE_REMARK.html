Definitions - INVOICE_REMARK

Qualified remarks about the invoice (qualifiers in table REMARK_QUAL). This may have similar information to the INVOICE_NOTE table, except that the remarks are qualified, while notes are freeform. Example: Qual: Decline reason Remark: Equipment not available.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
REMARK_SEQ_NO
REMARK_SEQ_NO contains a sequence number for a remark on an invoice.
REMARK_QUAL_IDENTIFIER
Remark qualifier.
REMARK_TEXT
REMARK_TEXT contains the text for the remark on an invoice.
