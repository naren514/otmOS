Definitions - OB_SHIP_UNIT_CONTENT

This table links to the OB_SHIP_UNIT to identify and define the contents of the box, package or pallet. For e.g., the shipment unit may be a 100 lb. Pallet and the contents may be ## Nike shoes.

OB_SHIP_UNIT_GID
An order can be identified by the items within the order or the ship units. This field is a unique identifier to define the ship units. The gid contains the domain portion.
