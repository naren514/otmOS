Definitions - LOCATION_REMARK

Additional information about the location. For e.g., address line3, 4 can be added to the location address via this table. A location can have many remarks associated with it.

LOCATION_GID
LOCATION_GID contains the unique identifier for the location.
REMARK_SEQUENCE
REMARK_SEQUENCE contains the seuqnce number for the remark for this location.
REMARK_QUAL_GID
REMARK_QUAL_GID contains the unique identifier for the remark qualifier for the location.
REMARK_TEXT
REMARK_TEXT contains the text of the remark for the location.
