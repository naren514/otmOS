Definitions - OB_SPECIAL_SERVICE

This table links to the special services required for the base order. The special services differ from the accessories in that the customer does not pay for special services.

ORDER_BASE_GID
A unique identifier to define the order base in the system. It is system generated. The gid contains the domian portion.
SPECIAL_SERVICE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
