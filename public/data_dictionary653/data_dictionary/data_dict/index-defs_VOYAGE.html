Definitions - VOYAGE

This table contains the world wide vessel schedules. The schedule is updated weekly by a manual upload process. G-Log has subscription to the vessel schedule with ESG (Electronic Shipping Guide) that publishes the information at its site shipguide.com in a dbase4 format. This data is downloaded, converted and populated in the GC3 database.

VOYAGE_GID
VOYAGE_GID contains the unique identifier for the voyage.
VOYAGE_XID
VOYAGE_XID contains the unique external identifier for the voyage.
SERVPROV_GID
SERVPROV_GID contains the unique identifier for the service provider for the voyage.
VOYAGE_SERVICE_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
VESSEL_GID
Vessel for the voyage
VOYAGE_NAME
VOYAGE_NAME contains the name of the voyage.
DEP_REGION_NAME
DEP_REGION_NAME contains the name of the departure region for the voyage.
DEP_DATE
DEP_DATE contains the departure date for the voyage.
