Definitions - OBS_STATUS_HISTORY_REMARK

This table stores the remarks for the order base status events.

ORDER_BASE_GID
The order for which this remark is associated.
I_TRANSACTION_NO
The transaction number (along with ORDER_BASE_GID, creates relationship to parent table OBS_STATUS_HISTORY)
REMARK_SEQUENCE
A one-up number to make the key unique.
REMARK_TEXT
The remark text itself.
