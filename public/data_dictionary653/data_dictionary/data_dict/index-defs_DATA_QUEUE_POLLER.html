Definitions - DATA_QUEUE_POLLER

Defines a poller for a data queue

DATA_QUEUE_POLLER_GID
Globale ID for the data queue poller
DATA_QUEUE_POLLER_XID
Data Queue Poller ID
DESCRIPTION
A description of the function and logic of the poller
WHERE_FILTER
A where clause to attach to queries. This is used to filter records by columns other than QUEUE_STATE.
ORDER_BY
An order by clause to attach to queries. Defaults to sequence ordering.
JAVA_PLUGIN_GID
A java plugin used to retrieve records with the poller. The plugin must have a PLUGIN_USE_TYPE of 'DATA QUEUE POLLER'.
IS_PREEMPTIVE_POLLER
If Y, the user can preempt events in the queue
