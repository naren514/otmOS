Definitions - LOCATION_SERVPROV_PREF

The table is used to store time window preferences for appointment on a location for a service provider.

LOCATION_SERVPROV_PREF_GID
The unique identifier of location service provider preference
LOCATION_SERVPROV_PREF_XID
The XID of location service provider preference
LOCATION_GID
The Location GID
SERVPROV_GID
The Service Provider GID
MAX_SLOTS_PER_DAY
The maximum number of slots per day.
SERVICE_LOCATION_GID
Unique key constaint will be created on (location_gid, servprov_gid, service_location_gid) in CU5.
