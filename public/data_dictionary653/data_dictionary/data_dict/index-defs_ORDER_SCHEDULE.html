Definitions - ORDER_SCHEDULE

Defines delivery schedules for orders

ORDER_SCHEDULE_GID
The internal ID for the order schedule.
ORDER_SCHEDULE_XID
The external ID for the order schedule.
X_LANE_GID
The geography this order schedule covers.
ORDER_RELEASE_TYPE_GID
The type of order this schedule covers.
DESCRIPTION
The description of this order schedule.
EFFECTIVE_DATE
The date this order schedule becomes effective.
EXPIRATION_DATE
The date this order schedule becomes ineffective.
