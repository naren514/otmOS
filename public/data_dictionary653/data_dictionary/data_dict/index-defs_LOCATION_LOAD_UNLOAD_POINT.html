Definitions - LOCATION_LOAD_UNLOAD_POINT

Represents an actual load/unload point as it relates to a location along with data that is associated with that relationship.

LOCATION_GID
Location at which the load/unload point resides
LOAD_UNLOAD_POINT
Load/unload point such as a loading dock
DESCRIPTION
Optional description for the load/unload point
EQUIPMENT_GROUP_PROFILE_GID
Optional equipment group profile for the load/unload point; overrides any at the location
IS_LOAD
Indicates if this dock or point make be used for loading
LOAD_SEQUENCE
Optional, indicates where dock should be in a load sequence
IS_UNLOAD
Indicates if this dock or point make be used for loading
UNLOAD_SEQUENCE
Optional, indicates where dock should be in an unload sequence
