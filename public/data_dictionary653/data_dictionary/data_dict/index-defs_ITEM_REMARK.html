Definitions - ITEM_REMARK

This table contains any free entry text remarks for the item, that the user wants to add.

ITEM_GID
ITEM_GID contains the unique identifier for the item being shipped.
REMARK_SEQUENCE
REMARK_SEQUENCE contains the sequence number for the item remark.
REMARK_QUAL_GID
REMARK_QUAL_GID contains the unique identifier for the remark qualifier.
REMARK_TEXT
REMARK_TEXT contains the text of the remark about the item.
