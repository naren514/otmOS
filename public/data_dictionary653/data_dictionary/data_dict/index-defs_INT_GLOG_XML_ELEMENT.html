Definitions - INT_GLOG_XML_ELEMENT

This table lists the names of the various gc3 xml interfaces. Examples are: TransOrder, Location, and ItemMaster.

GLOG_XML_ELEMENT_GID
This is the External System Gid.
GLOG_XML_ELEMENT_XID
This is the interface type indicating the XML being supported.
OUT_XML_PROFILE_GID
Spcifies the default outbound XML profile to be used for the interface. This default is overridden by the profile defined for the External System.
DESCRIPTION
This is a description
ELEMENT_TYPE
Specify if the element is for inbound, outbound or both
IS_LONG_RUNNING_TASK
If Y, any transmission with this transaction element is a long running task and should be monitored in the LONG_RUNNING_TASK table.
