Definitions - DM_RULE_D

Demurrage Rule Detail provides the detail capability to define possible rules for the determination of demurrage, detention and storage chargeable days and charges

DM_RULE_GID
Demurrage Rule ID
DM_RULE_D_SEQ_NO
demurrage rule d seq no
ACTIVITY_TYPE_GID
Activity Type ID. Possible demurrage activity type are load-load, load-empty, empty-load, empty-empty-bad-asset, empty-empty-not-used-asset.
SPECIAL_SERVICE_GID
Determine the type of charge. Example could be Demurrage. These are modeled as special services.
FREE_DAYS_CALENDAR_GID
Free days calendar helps to capture the free days like weekends, holiday etc. This would not be considered to determine the debit days.
FREE_DAYS_ACTIVITY_GID
Free days Activity again which the free days are defined with the free days calendar.
FREE_DAYS_TO_LOAD
free days to load.
FREE_DAYS_TO_UNLOAD
free days to unload.
FREE_DAYS_TO_UNLOAD_RELOAD
free days to unload and reload.
FREE_DAYS_TO_SWITCH
free days for own switching
SECONDARY_CHARGE_RULE_GID
charge the extended use of asset as secondary charge shipment.
START_BS_STATUS_CODE_GID
demurrage chargeable days start event (status code) ID. In the case of RR owned freight car, demurrage would typically start with the either constructive placement or actual placement events.
END_BS_STATUS_CODE_GID
demurrage chargeable days end event (status code) ID. In the case of RR owned freight car, demurrage would typically end with the released event.
CHARGE_TYPE
Charge type, possible values are NFRC Shipment, Inbound Shipment, Outbound Shipment, Secondary Charge Shipment.
