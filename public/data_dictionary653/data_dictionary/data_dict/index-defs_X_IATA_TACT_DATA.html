Definitions - X_IATA_TACT_DATA

Staging table for IATA TACT rate data integration. This is just a field representation of the fixed width text data supplied as flat files by IATA.

CATEGORY
Identifies a logical group of records within the file, for example minimum charge, proportional rate, etc.
INTERNAL1
Internal use
ORIGIN_CITY
Origin City as shown in the current IATA City Code Directory
ORIGIN_COUNTRY
Origin Country as detailed in the latest edition of ISO 3166 (E) and shown in the current IATA City Code Directory
DEST_CITY
Destination City as shown in the current IATA City Code Directory
DEST_COUNTRY
Destination Country as detailed in the latest edition of ISO 3166 (E) and shown in the current IATA City Code Directory
UNIQUE_NOTE
Defines the note number reference detailing specific conditiosn/restrictions which aply to the rate/charge
SERVPROV_CODE
The carrier code of the participating carrier
SCR_ITEM_NUMBER
Defines the commodity description code as detailed in the IATA Specific Commodity Description Guide and other publications.
ULD_RATING_TYPE
Defines the type of ULD for which the rate/charge is applicable.
ULD_CHARGE_CODE
Defines conditions under which the rates/charges are applicable
CURRENCY_CODE
Currency code
DECIMAL_PLACE
Number of decimal places in the rate field
RATE
Numerical value of the rate/charge
WEIGHT_BREAKPOINT
Defines the minimum breakpoint at which the rate/charge is applicable. Also identifies if the amount contained in the rate field is a minimum charge.
