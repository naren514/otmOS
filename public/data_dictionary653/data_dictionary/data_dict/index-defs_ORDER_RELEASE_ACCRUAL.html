Definitions - ORDER_RELEASE_ACCRUAL

Stores allocated freight costs for accruals.

ORDER_RELEASE_ACCRUAL_GID
Unique Id for this accrual entry
ORDER_RELEASE_ACCRUAL_XID
Unique Id for this accrual entry
ORDER_RELEASE_GID
Order release id for which this accrual is related
SENT_DATE
Timestamp of when this accrual was sent from GC3, null means it has not been sent yet.
FREIGHT_COST
Total current allocated freight cost
FREIGHT_SENT_COST
Total allocated freight cost already sent via the accrual interface.
IS_REVERSAL
A flag to determine whether the Order Release Accrual is a reversal.
SHIPMENT_START_DATE
The Shipment Start Date.
FREIGHT_COST_FN
functional currency amount for freight_cost
FREIGHT_SENT_C_FN
functional currency amount for freight_sent_cost
DELTA
delta or difference for the accrual calculated as: new amount - amount previously sent
DELTA_FN
functional currency amount for delta
COST_TYPE_GID
if line accruals are enabled, this will be populated with the cost type for the accrual line
ACCESSORIAL_CODE_GID
if line accruals are enabled, this will be populated with the accessorial id for the accrual line
