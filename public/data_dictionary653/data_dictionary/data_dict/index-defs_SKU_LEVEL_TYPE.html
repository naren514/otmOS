Definitions - SKU_LEVEL_TYPE

Types of SKU stock level. Examples are minimum, maximum.

SKU_LEVEL_TYPE_GID
SKU inventory alert level type id. For example, a minimum level can indicate that replenishment request needs to be sent.
SKU_LEVEL_TYPE_XID
SKU inventory alert level type xid.
SKU_LEVEL_TYPE_DESCRIPTION
Description of SKU inventory alert level type.
