Definitions - SHIP_UNIT_LINE

SHIP_UNIT_LINE stores line item information for an order release. It is a parallel table to S_SHIP_UNIT_LINE.

SHIP_UNIT_GID
A unique identifier to define the shipping unit that will be used for the order release line. The gid contains the domain portion.
SHIP_UNIT_LINE_NO
The line number associated with the ship unit.
ORDER_RELEASE_LINE_GID
A unique identifier to define the multiple order release lines per order release. The gid has the domain portion.
WEIGHT
Calculated weight of the shipping unit.
WEIGHT_UOM_CODE
Unit of measure for WEIGHT column.
WEIGHT_BASE
WEIGHT in base units.
VOLUME
Calcuated volume of the shipping unit.
VOLUME_UOM_CODE
Unit of measure for VOLUME column.
VOLUME_BASE
VOLUME in base units.
ITEM_PACKAGE_COUNT
the number of packages in this SHIP_UNIT_LINE for one ship unit ( its S_SHIP_UNIT may represent more than one same ship units). (e.g., for 20 ship units that contains 4 cases of 24 cans of coca, each S_SHIP_UNIT_LINE row will have ITEM_GID: coca, ITEM_COUNT 4x24, ITEM_PACKAGE_SPEC_GID: case-of-24-cans, ITEM_PACKAGE_COUNT: 4. In the SHIP_UNIT table, the SHIP_UNIT_COUNT will be 20)
