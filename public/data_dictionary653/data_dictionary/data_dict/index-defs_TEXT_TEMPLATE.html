Definitions - TEXT_TEMPLATE

Text template for entry and reporting.

TEXT_TEMPLATE_GID
The global unique identifier that represents the block of text.
TEXT_TEMPLATE_XID
The external identifier that represents the block of text.
TEXT
The actual text related to this template.
DATA_QUERY_TYPE_GID
If set, the text template may only be used with documents of the given type.
IS_MODIFIABLE
If true, users should be able to override the text template on the document UI
