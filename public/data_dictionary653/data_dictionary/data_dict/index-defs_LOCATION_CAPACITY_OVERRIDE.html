Definitions - LOCATION_CAPACITY_OVERRIDE

CAPACITY_TYPE
Possible values: WEIGHT, VOLUME, SHIPMENT, TEU, FEU, and all Equipment_Reference_Unit_GIDs
ACTIVITY_GID
The sole purpose of this column is to make FK to LOCATION_CAPACITY_D. It should match the ACTIVITY_GID of ACTIVITY_CALENDAR.
TIME_BUCKET_DURATION
Duration of each bucket
MAXIMUM_QUANTITY
The maximum quantity per bucket the location can handle. Cannot be exceeded. Only valid when CAPACITY_TYPE is neither WEIGHT nor VOLUME.
MAXIMUM_WEIGHT
The maximum weight per bucket the location can handle. Cannot be exceeded. Only valid when CAPACITY_TYPE is WEIGHT.
MAXIMUM_VOLUME
The maximum volume per bucket the location can handle. Cannot be exceeded. Only valid when CAPACITY_TYPE is VOLUME.
TARGET_QUANTITY
The ideal quantity per bucket the location should handle. Can be exceeded, but pay above target penalty. Only valid when CAPACITY_TYPE is neither WEIGHT nor VOLUME.
TARGET_WEIGHT
The ideal weight per bucket the location should handle. Only valid when CAPACITY_TYPE is WEIGHT.
TARGET_VOLUME
The ideal volume per bucket the location should handle. Only valid when CAPACITY_TYPE is VOLUME.
BELOW_TARGET_PENALTY
Below target penalty per quantity, or per weight unit (TARGET_WEIGHT_UOM_CODE), or per volume unit (TARGET_VOLUME_UOM_CODE)
ABOVE_TARGET_PENALTY
Above target penalty per quantity, or per weight unit (TARGET_WEIGHT_UOM_CODE), or per volume unit (TARGET_VOLUME_UOM_CODE)
