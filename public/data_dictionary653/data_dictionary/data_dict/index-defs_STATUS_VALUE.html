Definitions - STATUS_VALUE

This table defines multiple status values that can be assigned to a status type. For example, for a Status type of Appointment, the values can be: Complete None Partial etc.

STATUS_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
STATUS_VALUE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
STATUS_VALUE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
