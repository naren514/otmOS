Definitions - RATE_DISTANCE

RATE_DISTANCE includes the valid list of values for distance types, such as minimum and maximum distances. The RATE_DISTANCE is then referenced in RATE_OFFERING. When this match is made it will indicate the distance criteria for which the contract is valid.

RATE_DISTANCE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
RATE_DISTANCE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
SHORT_DIST_CUTOFF_BASE
SHORT_DIST_CUTOFF in base units.
LONG_DIST_CUTOFF_BASE
LONG_DIST_CUTOFF in base units.
DISTANCE_EXTERNAL_ENGINE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
