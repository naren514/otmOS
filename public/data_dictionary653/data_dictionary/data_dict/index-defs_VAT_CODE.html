Definitions - VAT_CODE

VAT_CODE_GID
Global identifier for the Value Added Tax code (domain + '.' + xid).
VAT_CODE_XID
External identifier for the Value Added Tax code.
DESCRIPTION
Description of the Value Added Tax code.
IS_CUMULATIVE
Indicates if tax for this code should be calculated on top of any freight charge+tax previously calculated.
IS_PROVINCIAL
Indicates if this code is meant for provincial tax
