Definitions - AUTO_ASSIGN_RULE_DEF_D

Header information for each list within a single auto-assignment to be done in a rule.

AUTO_ASSIGN_RULE_GID
Auto-assign rule
DEFINITION_SEQUENCE
sequence within the rule for the actual auto-assignment
AUTO_ASSIGN_CRITERIA_GID
Component of the criteria
LIST_TYPE
way to process the list of items for the component; O=match one in list; N=match none in list
IS_MATCH_ALL
If Y, indicates all values from the business object must pass the rule check. N indicates any match passes the rule.
