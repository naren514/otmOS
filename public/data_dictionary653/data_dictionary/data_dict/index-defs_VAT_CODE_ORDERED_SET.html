Definitions - VAT_CODE_ORDERED_SET

Allows storage of a list of VAT codes which must be used by the tax calculation in a specific order.

VAT_CODE_ORDERED_SET_GID
Gid for the name of the ordered set of VAT Codes
VAT_CODE_ORDERED_SET_XID
Xid for the name of the ordered set of VAT Codes
DESCRIPTION
User description for the set of ordered VAT codes
