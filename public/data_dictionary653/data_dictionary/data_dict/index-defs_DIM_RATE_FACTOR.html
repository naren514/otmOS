Definitions - DIM_RATE_FACTOR

Holds weight and volume information, that can either be associated with a contract (RATE_OFFERING) or a lane (RATE_GEO).

DIM_RATE_FACTOR_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
DIM_RATE_FACTOR_XID
GC3 Unique identifier for the entity. Is used for creating the GID
UNIT_1_UOM_CLASS
W = Weight,V = Volume,D = Distance,U = Duration,S = Speed,T = Temperator,L = Length
UNIT_2_UOM_CLASS
W = Weight,V = Volume,D = Distance,U = Duration,S = Speed,T = Temperator,L = Length
