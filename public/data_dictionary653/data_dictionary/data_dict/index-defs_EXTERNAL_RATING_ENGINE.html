Definitions - EXTERNAL_RATING_ENGINE

This table stores information about any external rating engine that the user wants to use. Normally the clients will use the GC3 rating engine, but in some cases they may want to use their own or a third party rating engine. For example the SMC rating engine.

EXTERNAL_RATING_ENGINE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
EXTERNAL_RATING_ENGINE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
