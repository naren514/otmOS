Definitions - ACR_ENTRY_POINT

List of code entry points controllable by access control roles. These entry points are determined by OTM components and include servlets, actions, queries, topics, mbeans and topics.

ACR_ENTRY_POINT_GID
OTM Entry Point GID
ACR_ENTRY_POINT_XID
OTM Entry Point XID
ACR_ENTRY_POINT_TAG
A mapping into OTM, defining the component access - generally a Java class name with an optional parameter
ACR_ENTRY_POINT_TYPE
Type of OTM component
ACR_ENTRY_POINT_PARENT
For beans and managed beans, entry point to inherit access from
ACR_ENTRY_POINT_VERSION
Version information to identify old, invalid records.
