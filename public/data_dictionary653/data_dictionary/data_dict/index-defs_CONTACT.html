Definitions - CONTACT

This table is used to identify a contact person at a location for communication.

CONTACT_GID
CONTACT_GID contains the unique identifier for the contact for the location.
CONTACT_XID
CONTACT_XID contains the unique external identifier for the contact for the location.
FIRST_NAME
FIRST_NAME contains the given name of the contact person for the location.
LAST_NAME
LAST_NAME contains the surname of the contact person for the location.
JOB_TITLE
JOB_TITLE contains the title for the contact person at the location.
EMAIL_ADDRESS
EMAIL_ADDRESS contains the email address of the contact person for the location.
PHONE1
PHONE1 contains the primary telephone number for the location'scontact person.
PHONE2
PHONE2 contains the secondary telephone number for the location'scontact person.
FAX
FAX contains the facsimile number for the contact person for the location.
LANGUAGE_SPOKEN
LANGUAGE_SPOKEN contains the language the contact person at the location speaks.
IS_PRIMARY_CONTACT
IS_PRIMARY_CONTACT describes whether this contact person at the location is the primary contact person (if so Y) or not (if not N).
TELEX
TELEX contains the telephone number for telex communications for the location'scontact person.
EXTERNAL_SYSTEM_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
NOTES
NOTES contains text describing something regarding the contact person for the location or the location itself.
GL_USER_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ROUTE_THROUGH_LOCATION
Indicate whether notification should use location.
CONTACT_TYPE
Defines whether a contact is a person or group.
IS_BROADCAST
Valid only if the contact_type = 'GROUP'. This is a boolean flag which controls whether GC3 creates a single notification message and broadcasts (i.e. cc) it to each group member or whether GC3 creates a message customized for each member.
GROUP_SAVED_QUERY_GID
Valid only if the contact_type = 'GROUP'. The saved_query could then define which individual contacts belong to the contact_group.
LOCATION_GID
LOCATION_GID contains the unique identifier for the location.
IS_EMAIL_VIRUS_PROTECTED
If Y, document attachments may be sent to the contact without virus checking.
IS_NOTIFICATION_ON
No notification will be sent to the contact if this field is set to false ('N')
TIME_ZONE_GID
Time zone for the contact
FROM_ADDR_CONTACT_GID
FROM_ADDR_CONTACT_GID is the CONTACT_GID that contains a contact's 'From-Address' details. Valid only if IS_FROM_ADDRESS is 'N'.
IS_FROM_ADDRESS
IS_FROM_ADDRESS indicates whether a contact is a 'From-Address' contact.
MIDDLE_NAME
The middle name of the contact
IPP_PRINTER_GID
Used to assign an IPP printer as a recipient of notifications, particularly of document or report events.
ECM_USER
If the contact is acting as a reviewer or subscriber to documents within Oracle ECM, the ECM user mapping to the contact.
NAME_ADDRESS_UPDATE_DATE
Date on which the details - FIRST_NAME,LAST_NAME,COMPANY_NAME,ALTERNATE_NAME,LOCATION_GID are modified.RPLS checks this date before it starts screening.
