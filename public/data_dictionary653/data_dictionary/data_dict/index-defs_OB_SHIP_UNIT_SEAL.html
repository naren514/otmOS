Definitions - OB_SHIP_UNIT_SEAL

This table defines the lock or seal information for the box or pallet if needed. This can be used at the delivery time to check if the package is sealed or locked with the pre-defined number.

OB_SHIP_UNIT_GID
An order can be identified by the items within the order or the ship units. This field is a unique identifier to define the ship units. The gid contains the domain portion.
