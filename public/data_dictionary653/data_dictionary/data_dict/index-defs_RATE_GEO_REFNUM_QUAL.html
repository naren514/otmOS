Definitions - RATE_GEO_REFNUM_QUAL

Available qualifiers for rate geo reference numbers

RATE_GEO_REFNUM_QUAL_GID
Qualifier that may be used for a rate geo refnum
RATE_GEO_REFNUM_QUAL_XID
XID for qualifier that may be used for a rate geo refnum
RATE_GEO_REFNUM_QUAL_DESC
Description of this rate geo refnum qualifier
DEFAULT_REFNUM_BN_TYPE_GID
Business number generator bn type if this is to be created by the BNG
UPDATE_FLAG
Specifies if the refnum qualifier can exist on the business object multiple times, or is restricted to occur only once and whether the value is allowed to be updated. If the value is MANY, then the object can have multiple values for the same qualifier. If the value is either UPDATE_OK or UPDATE_NOT_OK, then the rule is in effect, meaning only one value is allowed for a given qualifier. In the case of UPDATE_NOT_OK, the value may not be modified.
