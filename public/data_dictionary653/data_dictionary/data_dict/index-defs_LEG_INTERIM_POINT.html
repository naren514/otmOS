Definitions - LEG_INTERIM_POINT

This table stores the details of the intermediate locations within a specified leg.

LEG_GID
LEG_GID contains the unique identifier for the leg.
SEQ_NO
SEQ_NO contains the sequence number for the leg.
LOCATION_GID
LOCATION_GID contains the unique identifier for the location for the leg.
SPECIAL_SERVICE_GID
The special service needed for this interim point. It will be added to the corresponding shipment stop special service.
