Definitions - SHIP_UNIT_SPEC

SHIP_UNIT_SPEC stores additional specification information for a shipment's ship unit.

SHIP_UNIT_SPEC_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SHIP_UNIT_SPEC_XID
GC3 Unique identifier for the entity. Is used for creating the GID
LENGTH_BASE
LENGTH in base units.
WIDTH_BASE
WIDTH in base units.
HEIGHT_BASE
HEIGHT in base units.
TARE_WEIGHT_BASE
TARE_WEIGHT in base units.
EFFECTIVE_VOLUME_BASE
EFFECTIVE_VOLUME in base units.
PACKAGING_FORM_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
MAX_WEIGHT_BASE
MAX_WEIGHT in base units.
UNIT_TYPE
Defines whether the ship_unit_spec is of type (P)ackaging Unit, (T)ransport Handling Unit, or (B).
CORE_DIAMETER
For cylindrical objects, this defines the diameter of the core
CORE_DIAMETER_UOM_CODE
For cylindrical objects, unit of measure for the diameter of the core
CORE_DIAMETER_BASE
For cylindrical objects, the base value of the diameter of the core
IS_CYLINDRICAL
Indicate if the ship unit spec is cylindrial or not.
IS_NESTABLE
To indicate if this item is nestable or not when doing volume estimation.
MAX_NESTING_COUNT
Max number of units nestable to reduce volume consumption.
NESTING_VOLUME
The additional volume consumption from nesting a like unit.
MAX_ON_WIDTH
Max width of items can be placed on the ship unit spec when used as a THU.
MAX_ON_LENGTH
Max length of items can be placed on the ship unit spec when used as a THU.
MAX_ON_HEIGHT
Max height of items can be placed on the ship unit spec when used as a THU.
PRIORITY
This column defines the priority associated with an Ship Unit Spec
