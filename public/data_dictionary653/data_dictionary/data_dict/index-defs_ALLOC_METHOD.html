Definitions - ALLOC_METHOD

ALLOC_METHOD_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ALLOC_METHOD_XID
GC3 Unique identifier for the entity. Is used for creating the GID
ALLOC_TARGET_GID
Identifies to which allocation target object this allocation method can correspond to.
IS_USE_SHIPMENT_SHIP_UNIT
This flag is used only when alloc_target_gid is ORDER. This is for specifying that we can use Shipment Ship Unit information for calculating the metrics when the shipment ship unit line level metrics are zero. This columns is added for the bug 9203554
