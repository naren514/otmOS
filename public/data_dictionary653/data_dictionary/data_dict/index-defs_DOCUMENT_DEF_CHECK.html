Definitions - DOCUMENT_DEF_CHECK

Document validation rule

DOCUMENT_DEF_GID
Document Type
SAVED_CONDITION_GID
Saved condition - must be met for validation to pass.
ERROR_TRANSLATION_GID
If validation fails, an error message to display.
SOLUTION_TRANSLATION_GID
If validation fails, a solution message to display.
REVIEW_STATUS
State of document to apply validation. Validation can be applied when a document is being previewed, reviewed, finalized or indepedent of state.
XPATH_ROOT
Root of XPath testing. This is based on the integration XML of the document's business object.
XPATH_TEST
XPath condition, relative to the XPath root, that must be met for validation. If the condition is on a child branch, it must validate for all matching branches.
AUTO_ASSIGN_RULE_GID
If validation fails, an auto assign rule to force a default value onto the document.
DOCUMENT_DEF_CHECK_GID
Unique ID for the validation rule
