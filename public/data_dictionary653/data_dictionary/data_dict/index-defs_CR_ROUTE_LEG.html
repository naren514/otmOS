Definitions - CR_ROUTE_LEG

Individual legs for cooperative routes.

CR_FORECAST_GID
Null if this leg represents a depot stop
START_DAY
Start day number (depot arrival day if depot stop)
START_TIME
Start time of day (in seconds). (Depot arrival time for depot start.)
START_REGION_GID
Start region (null if this leg is a depot stop)
START_LOCATION_GID
Depot location ID if this is a depot stop (null otherwise).
END_DAY
End day number (depot departure day if depot stop)
END_TIME
End time of day (in seconds). (Depot departure time for depot start.)
END_REGION_GID
End region (null if this leg is a depot stop)
END_LOCATION_GID
Depot location ID if this is a depot stop (null otherwise).
NEXT_LEG_DEADHEAD
Distance from this leg's destination region to the next leg's source region (or to the fleet depot if the next leg is a depot stop).
ALLOCATED_COST
Cost allocated to this leg from the CR route cost.
