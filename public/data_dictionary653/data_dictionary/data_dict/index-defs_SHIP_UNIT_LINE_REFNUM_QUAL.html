Definitions - SHIP_UNIT_LINE_REFNUM_QUAL

SHIP_UNIT_LINE_REFNUM_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SHIP_UNIT_LINE_REFNUM_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID.
DESCRIPTION
Description of the reference number qualifier.
IS_VISIBLE_IF_MASTER_CARR
Identifies whether the reference numbers with this qualifier should display on related shipments.Possible Values are Y/N.
UPDATE_FLAG
Specifies if the refnum qualifier can exist on the business object multiple times, or is restricted to occur only once and whether the value is allowed to be updated. If the value is MANY, then the object can have multiple values for the same qualifier. If the value is either UPDATE_OK or UPDATE_NOT_OK, then the rule is in effect, meaning only one value is allowed for a given qualifier. In the case of UPDATE_NOT_OK, the value may not be modified. Possible values are: UPDATE_OK/UPDATE_NOT_OK/MANY.
