Definitions - CONDITIONAL_BOOKING_FIELD

static data for conditional booking field specification

DATA_TYPE
The available values of this column should be the same as these defined in Planning_parameter_qualifer, i.e. S/B/I/N/D/G/U/C/P.
UPDATE_FIELD_NAME
this column capture the names of update field. The convention is ., i.e., SHIPMENT.TOTAL_VOLUME. Multiple values separated by comma are allowed.
