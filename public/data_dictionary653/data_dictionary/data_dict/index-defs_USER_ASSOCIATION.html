Definitions - USER_ASSOCIATION

The USER_ASSOCIATION table specifies the data that service providers can see, allowing them access only to their data. GC3 has a concept of SERVPROV user class that groups all the service providers. A SERVPROV in the USER_ASSOCIATION_QUAL denotes that the GL_USER_GID is a service provider. The service provider"s name is listed in the USER_ASSOCIATION_VALUE. For e.g., MSCARRIERS etc.

GL_USER_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
