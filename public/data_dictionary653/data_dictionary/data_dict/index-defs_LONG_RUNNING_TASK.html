Definitions - LONG_RUNNING_TASK

Long Running Task - provides visibility and control to long-running tasks

TASK_SEQUENCE
Unique sequence number for the LRT
START_TIME
Time the LRT was submitted. This may reflect the time it was placed in the workflow mediator or queues.
END_TIME
The time the LRT completed.
STATE
State of the LRT: MEDIATING = in the workflow mediator, QUEUED = in an event queue, RUNNING = in an active process, COMPLETED = fully completed, RESUBMITTED = resubmitted to the Process Control or Data Queue table, INTERRUPTED = interrupted, ABORTED = stopped by a server shutdown. (Note that RUNNING LRTs are automatically changed to ABORTED LRTs on server startup.)
IS_RESUBMITTABLE
if Y, the LRT can be resubmitted to its original Process Control or Data Queue. (E.g. Bulk Plan before shipment modifications)
IS_INTERRUPTIBLE
if Y, the LRT supports clean interruption (e.g. Bulkplan Termination)
APP_MACHINE_GID
The application server running the LRT.
LOG_PROCESS_ID
The log process ID of the LRT. This may not be unique across server runs, but corresponds to the Logging ID of the LRT during its runtime.
BUSINESS_PROCESS_PID
Business process ID of the LRT. This links to the Open Processes diagnostics as long as the LRT is still running on the server. Note on startup, this process ID is cleared. It can be used as an indicator of current vs. historical tasks.
SUBMITTED_BY_USER_GID
For User actions, the user that initiate the LRT. For Integrations, the user submitted on the transmission. For Process Control tasks and agent actions, the user associated with the workflow.
TASK_SOURCE
Type of LRT
ACTION_GID
If TASK_SOURCE='ACTION', the name of the User Action corresponding to the LRT.
AGENT_GID
If TASK_SOURCE='AGENT ACTION', the name of the Agent invoking the LRT.
AGENT_ACTION_GID
If TASK_SOURCE='AGENT ACTION', the name of the Agent Action corresponding to the LRT
TOPIC_ALIAS_GID
If TASK_SOURCE='PROCESS CONTROL', the name of the Process corresponding to the LRT.
I_TRANSMISSION_NO
If TASK_SOURCE='INTEGRATION', the transmission corresponding to the LRT.
PROCESS_CONTROL_HISTORY_ID
If TASK_SOURCE='PROCESS CONTROL', the unique Process Control History instance for this task request.
Q_INTEGRATION_IN_SEQ
If task came from Q_INTEGRATION_IN, a sequence for the queue record
TASK_DETAIL_NAME
If additional details are available for this task, a displayable name for the details (e.g. Bulk Plan Results).
TASK_DETAIL_URL
If additional details are available for this task, a URL to the additional details (e.g. the Bulk Plan Results screen).
TASK_DETAIL_PK
If additional details are available for this task, and table-driven, a string representation of the detail's primary key
