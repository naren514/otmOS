Definitions - INVOICE_STATUS

One record per status per invoice. Contains the statuses of an invoice. For example, whether the invoice was issued, approved, sent, etc.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
STATUS_TYPE_GID
STATUS_TYPE_GID contains the unique identifier for the status type on the invoice.
STATUS_VALUE_GID
STATUS_VALUE_GID contains the unique identifier for the status value on the invoice.
