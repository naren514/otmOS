Definitions - NMFC_CLASS

This table defines the NMFC governmental class name and number if the item is an NMFC Class.

NMFC_CLASS_GID
NMFC_CLASS_GID contains the National Motor Freight Classification (NMFC) class for the item being shipped.
NMFC_CLASS_XID
NMFC_CLASS_XID contains the unique external identifier for the National Motor Freight Classification (NMFC) class of the item being shipped.
NMFC_CLASS_NAME
NMFC_CLASS_NAME contains the National Motor Freight Classification (NMFC) name of the item being shipped.
