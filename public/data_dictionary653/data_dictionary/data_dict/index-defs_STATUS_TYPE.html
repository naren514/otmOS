Definitions - STATUS_TYPE

This table defines the status types that can or may be reported on a line item shipment. A status type may have multiple status values. For example the status type can be: Payment Planning Appointment Processed etc. See Status_Value table for details on the status values.

STATUS_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
STATUS_TYPE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
