Definitions - SHIPMENT_ACTION_QUAL

Allows users to specify the actions listed below: ASSIGN_SERVPROV ASSIGN_RATE CHECK_CREDIT TENDER TRACK_TRACE_GIIV MATCH_INVOICE CREATE_BILL These qualifiers are shown on an appropriate user interface to allow the user specifying the actions.

SHIPMENT_ACTION_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SHIPMENT_ACTION_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
