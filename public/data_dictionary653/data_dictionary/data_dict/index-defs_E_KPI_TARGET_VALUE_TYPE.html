Definitions - E_KPI_TARGET_VALUE_TYPE

This table defines the target value type for each KPIs target type. Example, KPI (Tender Performance) TARGET is FIXED 97%, or KPI (Tender Performance) FORECAST is FIXED 100%, or KPI (Cost Per Lane) TARGET is USER_DEFINED. For FIXED type, TARGET_VALUE should stores the fixed value. For MOVING_AVERAGE, TARGET_VALUE stores the number of months.
