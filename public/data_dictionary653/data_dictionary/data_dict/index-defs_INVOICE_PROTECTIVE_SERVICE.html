Definitions - INVOICE_PROTECTIVE_SERVICE

Codes to define temperature and ventilation requirements for cargo.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
PROT_SRV_SEQ_NO
Protective service sequence number.
PROT_SRV_CODE
Protective service code.
PROT_SRV_RULE_CODE
Protective service rule code.
PROT_SRV_TEMP
Temperature requirement.
PROT_SRV_TEMP_UOM_CODE
Unit of measure for the temperature requirement.
PROT_SRV_TEMP_BASE
Temperature requirement in base units.
