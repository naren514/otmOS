Definitions - ALLOC_SOURCE_OBJECT

For indentifying the source object record of the allocation. For example Shipment, Voucher, Bill

ALLOC_SOURCE_OBJECT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ALLOC_SOURCE_OBJECT_XID
GC3 Unique identifier for the entity. Is used for creating the GID, E.g. ShipmentCost, VoucherCost, BillCost
CLASS_NAME
The Specific java class name that represents and performs operations for this object.
SOURCE_OBJECT_TYPE
Identifies if the source object is of cost type or invemtory type or UOM type.
