Definitions - GTM_STR_COMPONENT_QUANTITY

A generic key/value representation to capture all possible quantity information for example weight, volume, quantity, counts, total weight, net weight etc.

GTM_STRUCTURE_GID
GTM_STRUCTURE_GID contains the unique identifier for the trade item structure.
GTM_STRUCTURE_COMPONENT_SEQ
Trade Item Structure Component Sequence Number
GTM_QUANTITY_TYPE_GID
A pointer to the quantity type. For example: Weight, Net Weight, Quantity etc
QUANTITY
Value for the specified quantity type.
