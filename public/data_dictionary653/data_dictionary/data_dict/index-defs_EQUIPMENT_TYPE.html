Definitions - EQUIPMENT_TYPE

This table categorizes the equipments from the Equipment table into various categories. The users in GC3 can define the individual equipments, or types of equipments or both.

EQUIPMENT_TYPE_GID
EQUIPMENT_TYPE_GID is the unique identifier for the equipment type.
EQUIPMENT_TYPE_XID
EQUIPMENT_TYPE_XID is the unique external identifier for the equipment type.
EQUIPMENT_TYPE_NAME
EQUIPMENT_TYPE_NAME contains text for the name of the equipment type.
EQUIPMENT_DESC
EQUIPMENT_DESC provides a textual description of the equipment type.
GROSS_WEIGHT
GROSS_WEIGHT contains the equipment type'sgross weight.
GROSS_WEIGHT_UOM_CODE
Unit of measure for GROSS_WEIGHT column.
GROSS_WEIGHT_BASE
GROSS_WEIGHT in base units.
TARE_WEIGHT_BASE
TARE_WEIGHT in base units.
NET_WEIGHT_BASE
NET_WEIGHT in base units.
VOLUME
VOLUME contains the volume of the equipment type.
VOLUME_UOM_CODE
Unit of measure for VOLUME column.
VOLUME_BASE
VOLUME in base units.
INTERIOR_LENGTH
INTERIOR_LENGTH contains the length of the equipment type'sinterior.
INTERIOR_LENGTH_UOM_CODE
Unit of measure for INTERIOR_LENGTH column.
INTERIOR_LENGTH_BASE
INTERIOR_LENGTH in base units.
INTERIOR_WIDTH
INTERIOR_WIDTH contains the width of the equipment type'sinterior.
INTERIOR_WIDTH_UOM_CODE
Unit of measure for INTERIOR_WIDTH column.
INTERIOR_WIDTH_BASE
INTERIOR_WIDTH in base units.
INTERIOR_HEIGHT
INTERIOR_HEIGHT contains the height of the equipment type'sinterior.
INTERIOR_HEIGHT_UOM_CODE
Unit of measure for INTERIOR_HEIGHT column.
INTERIOR_HEIGHT_BASE
INTERIOR_HEIGHT in base units.
EXTERIOR_LENGTH
EXTERIOR_LENGTH contains the length of the exterior of the equipment type.
EXTERIOR_LENGTH_UOM_CODE
Unit of measure for EXTERIOR_LENGTH column.
EXTERIOR_LENGTH_BASE
EXTERIOR_LENGTH in base units.
EXTERIOR_WIDTH
EXTERIOR_WIDTH contains the width of the exterior of the equipment type.
EXTERIOR_WIDTH_UOM_CODE
Unit of measure for EXTERIOR_WIDTH column.
EXTERIOR_WIDTH_BASE
EXTERIOR_WIDTH in base units.
EXTERIOR_HEIGHT
EXTERIOR_HEIGHT contains the height of the exterior of the equipment type.
EXTERIOR_HEIGHT_UOM_CODE
Unit of measure for EXTERIOR_HEIGHT column.
EXTERIOR_HEIGHT_BASE
EXTERIOR_HEIGHT in base units.
TEMPERATURE_CONTROL_GID
A unqie identifier to define the temperature controls value for the commodity. The gid contains the domian portion.
SUBSTITUTE_EQ_TYPE_PROFILE_GID
Indicates which equipment types can be substituted for this type in fleet equipment planning.
