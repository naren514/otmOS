Definitions - OBJECT_LOCK

Holds cross-machine object locks. These include business object locks along with other coordination. Note that these locks span multiple transactions and multiple threads of execution.

LOCK_TYPE
Type of lock. May be a DATA_QUERY_TYPE or some other coordination type.
OBJECT_ID
Lock ID. May be a business object GID or some other coordination key.
OWNER_MACHINE_GID
Machine currently owning the lock
OWNED_SINCE
Time the lock was grabbed
ACTIVITY
Activity currently using the lock
LOG_PROCESS_ID
A link to the System log entries for the lock owner
