Definitions - OFFSET_DEFINITION_D

This table holds different poossible placement dates come under given offset defnition along with their offsets.

OFFSET_DEFINITION_D_SEQ
is the unique identifier for each offset time entry.
OFFSET_DEFINITION_GID
is the unique identifier the offset definition.
DAY_OF_WEEK
It represents day of week selected. 1 for Monday and 2 for Tuesday ...and 7 for Sunday.
START_TIME
It represents start time selected in milliseconds
END_TIME
It represents enttime selected in milliseconds
OFFSET_DAYS
It represents offset days for selected day of week and time
OFFSET_TIME
It represents offset time in milliseconds
