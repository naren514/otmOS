GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.

The GC3 Sship Unit Line Item GID which is the basis for this duty calculation line item service request.

A success or failure code from the ITL Vendor'ssystem, related to this line item in the service message.

A unique identifier to define the multiple order release lines per order release. The gid has the domain portion.

A unique pointer to define the shipment ship unit. The gid contains the domain portion.

The unique sequence within the ship unit to define the individual line items.
OB_LINE_GID
A unique identifier to define the line items in the order. The gid contains the domain portion.
