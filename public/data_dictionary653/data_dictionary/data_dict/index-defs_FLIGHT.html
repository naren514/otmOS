Definitions - FLIGHT

This table contains the worldwide flight schedule. This information is updated monthly. The government flight schedule is obtained on a CD monthly (from OAG) and uploaded to this table.

FLIGHT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
FLIGHT_XID
GC3 Unique identifier for the entity. Is used for creating the GID
SRC_LOCATION_GID
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
DEST_LOCATION_GID
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
