Definitions - DOCUMENT_USE_PROFILE

A set of rights and assumptions defining the uploading and viewing of documents.

DOCUMENT_USE_PROFILE_GID
A global identifier for the document use profile
IS_ALLOWED_UPLOAD
If Y, users with this profile may upload documents to GC3
IS_TRUSTED_UPLOAD
If Y, users with this profile may upload documents to GC3 without needing a virus check.
MAX_UPLOAD_SIZE
Maximum number of bytes allowed in a document upload
IS_BROWSER_VIRUS_PROTECTED
If Y, users with this profile may view document content regardless of a virus check; the user's browser is assumed to provide virus checking
