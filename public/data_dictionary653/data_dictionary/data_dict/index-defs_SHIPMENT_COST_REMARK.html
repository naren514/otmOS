Definitions - SHIPMENT_COST_REMARK

Allows you to define remarks for why this shipment cost applies to this shipment.

SHIPMENT_COST_SEQNO
The unique number identifying the shipment_cost record this remark is associated with.
REMARK_SEQUENCE
The one-up number to make this remark unique.
REMARK_QUAL_GID
The remark qualifier, or remark type.
REMARK_TEXT
The remark text itself.
