Definitions - BULK_REPORTING

Progress table for monitoring bulk report generation

QUERY_NAME
Saved query used to generate bulk report
REPORT_GID
Report generated
START_TIME
Start time of bulk report generation
END_TIME
End time of bulk report generation
NUM_REPORTS_REQUESTED
# of business objects requested for bulk generation
NUM_REPORTS_GENERATED
# of business objects generated to date
NUM_REPORTS_EMAILED
# of e-mails generated to date
NUM_REPORTS_PRINTED
# of printouts via IPP generated to date
STATUS
Status of bulk reporting
SINCE
Time since last status change
APP_MACHINE_GID
Application server running the bulk generation
