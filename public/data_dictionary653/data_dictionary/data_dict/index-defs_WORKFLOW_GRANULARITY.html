Definitions - WORKFLOW_GRANULARITY

Contains list of values for units of time. This table is used by the WORKFLOW_TIME_WINDOW table. Ex: Seconds, Minutes, Hours, Days

WORKFLOW_GRANULARITY_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
WORKFLOW_GRANULARITY_XID
GC3 Unique identifier for the entity. Is used for creating the GID
