Definitions - INVOICE_LINEITEM_REFNUM_QUAL

Contains the qualifier and description of additional information that is associated with an invoice line item. The actual values would then be stored in INVOICE_LINEITEM_REFNUM.

INVOICE_LI_REFNUM_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
INVOICE_LI_REFNUM_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
INVOICE_LI_REFNUM_DESC
INVOICE_LI_REFNUM_DESC contains a textual description for the line item reference number on the invoice.
DEFAULT_REFNUM_BN_TYPE_GID
The BN type used to generate a refnum when the refnum field for a record in the corresponding refnum table is not populated.
