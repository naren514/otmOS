Definitions - STATUS_FUNCTOR

SAVED_CONDITION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SAVED_CONDITION_ADDITIVE
Identify if saved condition check and functor check are cumulative. 'Y' means cumulative, 'N' means saved condition overrides status functor.
SAVED_CONDITION_CAUSE_GID
Optional; specifies the translation ID of the error message to be returned if the saved condition causes the status functor to fail
