Definitions - DATA_QUEUE_TABLE

Defines common database attributes of a data queue table including sequence column and partition durations.

TABLE_NAME
Oracle table name of the data queue table
EXCEPTION_TABLE_NAME
Oracle table name of the data queue exception table
ORACLE_SEQUENCE
The oracle sequence generator for the data queue sequence column
ORACLE_EXCEPTION_SEQUENCE
The oracle sequence generator for the data queue exception table
PARTITION_DURATION
Duration of each partition
