Definitions - INVOICE_LINEITEM_REFNUM

INVOICE_LINEITEM_REFNUM lists the reference number for each line item on an invoice (one record per reference number). This table can be used to store additional qualified (categorized) information about a line item within the invoice. The qualifier/description of the information is stored in INVOICE_LINEITEM_REFNUM_QUAL.

INVOICE_GID
INVOICE_GID is the unique identifier for the invoice.
LINEITEM_SEQ_NO
LINEITEM_SEQ_NO contains a sequence number for the line item on the invoice.
INVOICE_LI_REFNUM_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
REFERENCE_NUMBER
Line item reference number.
