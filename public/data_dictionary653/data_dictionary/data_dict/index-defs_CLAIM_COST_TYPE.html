Definitions - CLAIM_COST_TYPE

Holds codes for the different types of costs that are associated with a claim such as LOSS_VALUE and SALVAGE. User may add codes in addition to those that are shipped with GC3.

CLAIM_COST_TYPE_GID
Code for the cost type of a claim
DESCRIPTION
Description of the cost type for a claim
COST_ACTIVITY
Indicates if the cost adds or subtracts from the total damage/loss value of the claim. 'ADD' indicates it is a loss; 'SUBTRACT' indicates it is a negative cost for the claim, such as money credited for salvage.
