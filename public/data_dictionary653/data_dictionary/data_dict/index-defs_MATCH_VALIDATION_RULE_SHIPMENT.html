Definitions - MATCH_VALIDATION_RULE_SHIPMENT

The value of the specified invoice refnum qualifier will be validated against the corresonding shipment refnum value during the invoice-shipment match validation process

MATCH_VALIDATION_RULE_GID
Id of the match validation rule
INVOICE_REFNUM_QUAL_GID
Invoice refnum qualifier for the values to be validated against the shipment
SHIPMENT_REFNUM_QUAL_GID
Shipment refnum qualifier for the values to be validated against the indicated invoice refnum
