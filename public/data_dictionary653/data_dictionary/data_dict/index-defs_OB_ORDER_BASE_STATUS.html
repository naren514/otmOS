Definitions - OB_ORDER_BASE_STATUS

The status of the base order. This table links to the defined status(es) in the Status_Type and Status_Value table.

ORDER_BASE_GID
ORDER_BASE_GID contains the unique identifier for the order base.
STATUS_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
STATUS_VALUE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
