Definitions - SHIPMENT_COST_REF

This table stores the breakdown of the shipment costs based on the shipment qualifiers. For example Item cost, order base cost, order release cost, order release line cost etc.

SEQNO
A system generated unique sequence number.
SHIPMENT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COST_REFERENCE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SHIPMENT_COST_QUAL_GID
A unique identifier for the various cost qualifiers like: Item, Order base Order release Order release line etc.
IS_PICKUP
True or False value to specify if the cost is a Pickup cost.
IS_DROPOFF
True or False value to specify if the cost is a Dropoff cost.
