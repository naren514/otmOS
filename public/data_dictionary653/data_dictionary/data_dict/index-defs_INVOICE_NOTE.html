Definitions - INVOICE_NOTE

Unlimited notes per invoice. Freeform, timestamped notes that can be used to indicate explanations/information throughout the lifetime of the invoice.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
TIMESTAMP
TIMESTAMP contains the day, month and year of the note on the invoice.
SUMMARY
Summary.
NOTE
NOTE contains the note(s) about the invoice.
IS_SYSTEM_GENERATED
Indicates whether this is system generated or not.
ENTERED_BY
ENTERED_BY contains the name and domain of the person who entered notes about the invoice.
