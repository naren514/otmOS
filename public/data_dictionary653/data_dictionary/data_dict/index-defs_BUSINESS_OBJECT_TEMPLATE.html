Definitions - BUSINESS_OBJECT_TEMPLATE

This table associates a template with data query type for a given domain

OBJECT_TEMPLATE_GID
OBJECT_TEMPLATE_GID is the gid in the table shipment, ob_order_base and order_release where the is_template column is "Y"
DATA_QUERY_TYPE_GID
The query object type.
TEMPLATE_SEQUENCE
Sequence to allow multiple template for a given query object.
DOMAIN_NAME
Domain of the User.
DATA_SOURCE
Specifies if the template applies to data from integration, UI or both
IS_ACTIVE
Boolean parameter to disable or enable the template
