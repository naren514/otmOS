Definitions - SHIPMENT_TEXT

Associates a shipment with a set of fixed text. A shipment can have more than one set of text, and the text can be shared by more than one shipment.

SHIPMENT_GID
The shipment for which the fixed text is associated.
TEXT_TEMPLATE_GID
The foreign key to the text_template table, in order to include fixed text with this shipment.
DOCUMENT_DEF_GID
Document that should hold text
TEXT_OVERRIDE
Transactional override for the template on the specific document
