Definitions - HNAME_COMPONENT

This table defines the hierarchy name of the location. The hierarchy can be any combination and levels of the name. For e.g., USA/PA/Philadelphia is a three level hierarchy with Country/State/City, USA/19406 is a two level hierarchy with Country/Postal Code.

HNAME_COMPONENT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
HNAME_COMPONENT_XID
GC3 Unique identifier for the entity. Is used for creating the GID
