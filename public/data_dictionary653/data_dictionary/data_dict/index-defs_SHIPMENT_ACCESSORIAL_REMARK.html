Definitions - SHIPMENT_ACCESSORIAL_REMARK

Allows you to define remarks to explain why this accessorial applies to the shipment.

SHIPMENT_GID
The shipment that this remark is associated with.
ACCESSORIAL_CODE_GID
The accessorial code related to the remark.
REMARK_SEQUENCE
The one-up sequence for the remark, to create a unique identifier.
REMARK_QUAL_GID
The remark qualifier, which identifies the type of remark this is.
REMARK_TEXT
The remark itself.
