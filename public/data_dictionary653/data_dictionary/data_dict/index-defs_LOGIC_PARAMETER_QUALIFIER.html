Definitions - LOGIC_PARAMETER_QUALIFIER

Parameter definitions for logic configuration

PARAMETER_GROUP_GID
The parameter group such as GENERAL, CONSOLIDATION, SEQUENCING for MULTISTOP config.
DATA_TYPE
S- String, B - Boolean, I - Integer, N - Number, D- Date, G - PickList, U - UOM, C - Currency, P - Percentage
UOM_TYPE
Required when data_type is U
GID_QUERY_CLASS
Java query class name. Required when data_type is G
VALID_VALUE_TYPE_GID
This columns helps to maintain the valid values for each Logic Config parameter in the VALID_VALUE_TYPE table
