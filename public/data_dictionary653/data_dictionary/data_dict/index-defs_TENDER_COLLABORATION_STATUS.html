Definitions - TENDER_COLLABORATION_STATUS

This table stores the status of the tender as received from the service provider. The status is received as transactions. This table can be linked to the I_Transaction and I_Transmission tables to obtain the details of the status.

STATUS_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
STATUS_VALUE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
