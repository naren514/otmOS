Definitions - SHIPMENT_SPECIAL_SERVICE

Allows you to define remarks for why a special service applies to the associated shipment.

SHIPMENT_GID
The shipment associated with the remark.
SHIPMENT_SPECIAL_SERVICE_SEQ
sequence to keep shipment special services unique
SPECIAL_SERVICE_GID
The special service associated with the remark.
STOP_NUM
If the special service is for a particular stop, this indicates the stop number
PROCESS_AS_FLOW_THRU
Override for record to be copied to the sell side during rating.
ADJUSTMENT_REASON_GID
Reason why charge was added.
COMPLETION_STATE
indicates completion state of this special service
PAYABLE_INDICATOR_GID
indicates the payable status of this special service; if values, overrides the payable status on the special service itself
BILLABLE_INDICATOR_GID
indicates the billable status of this special service; if values, overrides the billable status on the special service itself
ACTUAL_OCCUR_TIME
actual date and time special service occurred
PLANNED_DURATION
planned duration of the special service
IS_PLANNED_DURATION_FIXED
if Y, prevents OTM from overwriting the planned duration for the special service
ACTUAL_DURATION
if specified, actual duration of the special service
ACTUAL_DISTANCE
if specified, actual distance of the special service
ACTUAL_WEIGHT
if specified, actual weight of the goods applicable to this special service
ACTUAL_VOLUME
if specified, actual volume of the goods applicable to this special service
ACTUAL_SHIP_UNIT_COUNT
if specified, actual ship unit count applicable to this special service
ACTUAL_ITEM_PACKAGE_COUNT
if specified, actual package count applicable to this special service
IS_SYSTEM_GENERATED
indicates if this special service was created by the system, allows logic to determine which special services to leave in place when re-syncing the shipment special services
