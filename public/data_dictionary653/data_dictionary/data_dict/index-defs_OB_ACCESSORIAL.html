Definitions - OB_ACCESSORIAL

OB_ACCESSORIAL consists of the links to the ACCESSORIAL table to define the accessories required to fulfill the order. For example, a pump may be required to pump chemicals from a delivery truck. Accessorials are different from special services in that customers are charged for the accessories.

ORDER_BASE_GID
A unique identifier to define the order base in the system. It is system generated. The gid contains the domian portion.
ACCESSORIAL_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
