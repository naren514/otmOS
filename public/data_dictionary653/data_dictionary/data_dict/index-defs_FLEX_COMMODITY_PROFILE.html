Definitions - FLEX_COMMODITY_PROFILE

This table is used to resolve the many-to-many relationship between the flex commodity profiles and the qualifiers.

FLEX_COMMODITY_PROFILE_GID
FLEX_COMMODITY_PROFILE_GID contains the unique identifier for the flex commodity.
FLEX_COMMODITY_PROFILE_XID
FLEX_COMMODITY_PROFILE_XID contains the unique external identifier for the flex commodity.
FLEX_COMMODITY_PROFILE_NAME
FLEX_COMMODITY_PROFILE_NAME contains the name of the flex commodity.
FLEX_COMMODITY_QUAL_GID
FLEX_COMMODITY_QUAL_GID contains the unique identifier for the flex commodity qualifier.
