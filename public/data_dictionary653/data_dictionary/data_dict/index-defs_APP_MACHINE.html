Definitions - APP_MACHINE

SCHEDULED_SHUTDOWN_TIME
Scheduled shutdown time. Null, if no shutdown has been scheduled.
STATE
Server state: S_DOWN = server is down, S_STARTING = server is starting, S_UP = server is up, Q_USER_SESSION_DENY = quiescing: deny new user sessions, Q_USER_LRT_WARN = queiscing: warn user on any new long running tasks Q_USER_WARN = quiescing: warn user of imminent shutdown Q_LRT_QUIESCING = quiescing; stop polling any long running tasks, Q_QUIESCING = quiescing: stop new work Q_TERMINATING = quiescing: interrupt existing work, Q_WAITING = quiescing: waiting for server idle
NEXT_STATE
During shutdown, this is the next expected server state
NEXT_STATE_TIME
Estimated time of next server state change during shutdown
