Definitions - QUALITY_FACTOR_PROFILE

This table is used to specify the quality of the service. This can be associated with the leg table to specify for example, that the leg has an excellent and/or good service available by one or more carriers.

QUALITY_FACTOR_PROFILE_GID
QUALITY_FACTOR_PROFILE_GID contains the unique identifier for the quality factor profile for a service provider.
QUALITY_FACTOR_PROFILE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
