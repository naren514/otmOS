Definitions - OB_RELEASE_INSTRUCTION

This table is linked at the line item level. It identifies the line item and how much quantity to release against that line item in the base order. It provides further instructions to the create_order_release process in GC3. A build_shipment_process in GC3 then takes this information to build the shipments against the line item order.

ORDER_BASE_GID
A unique identifier to define the order base in the system. It is system generated. The gid contains the domian portion.
OB_LINE_GID
A unique identifier to define the line items in the order. The gid contains the domain portion.
QUANTITY_QUALIFIER_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SOURCE_LOCATION_GID
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
DEST_LOCATION_GID
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
WEIGHT_VALUE_BASE
WEIGHT_VALUE in base units.
VOLUME_VALUE_BASE
VOLUME_VALUE in base units.
TOTAL_WEIGHT
Total Weight for this release instruction.
TOTAL_WEIGHT_UOM_CODE
Unit of measure for TOTAL_WEIGHT column.
TOTAL_WEIGHT_BASE
TOTAL_WEIGHT in base units.
TOTAL_VOLUME
Total Volume for this release instruction.
TOTAL_VOLUME_UOM_CODE
Unit of measure for TOTAL_VOLUME column.
TOTAL_VOLUME_BASE
TOTAL_VOLUME in base units.
SHIP_UNIT_COUNT
The number of Ship Units for this release instruction.
TRANSPORT_HANDLING_UNIT_GID
The Ship Unit Specifiation for this release instruction.
OB_SU_RELEASE_INSTRUCTION_GID
If not null, this line is release at the SU level
