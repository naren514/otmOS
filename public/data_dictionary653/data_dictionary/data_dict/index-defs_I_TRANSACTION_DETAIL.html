Definitions - I_TRANSACTION_DETAIL

The I_TRANSACTION_DETAIL table provides additional details for the I_TRANSACTION when the Transaction applies to more than one object gid.

I_TRANSACTION_NO
Transaction Number.
SEQUENCE_NO
Sequence number.
STATUS
Specifies the status of the transaction for the object gid.
OBJECT_GID
Specifies the Gid of the Object that was affected by the Transaction.
NEW_TRANSACTION_NO
If the processing of the object required obtaining a new i_transaction_no, this field provides a reference to the number. Note: An i_transaction record may not exist for this new transaction.
OBJECT_SEQUENCE
Specifies the sequence number for the primary key for the object.
