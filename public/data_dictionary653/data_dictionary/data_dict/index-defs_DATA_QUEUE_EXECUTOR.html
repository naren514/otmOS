Definitions - DATA_QUEUE_EXECUTOR

Defines an executor for a data queue

DATA_QUEUE_EXECUTOR_GID
Global ID for the data queue executor
DATA_QUEUE_EXECUTOR_XID
Data queue executor ID
JAVA_PLUGIN_GID
JAVA_PLUGIN must have a PLUGIN_USE_TYPE of 'DATA QUEUE EXECUTOR'
REDO_ACTIVE_EVENTS
If Y, ACTIVE events should automatically be set to QUEUED on server restart
REDO_FAILED_EVENTS
If Y, FAILED events should automatically be set to QUEUED on server restart
