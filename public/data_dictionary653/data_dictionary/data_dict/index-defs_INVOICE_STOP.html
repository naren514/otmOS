Definitions - INVOICE_STOP

Contains the stops that are related to the shipment referred to on the invoice. The difference between an invoice stop and an invoice route is that a new route indicates a change in transportation carrier (and the location of the interchange), while a stop is simply a stop to pick up or deliver goods.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
STOP_SEQ_NO
Stop sequence number.
LOCATION_GID
LOCATION_GID contains the unique identifier for the location on the invoice.
LOCATION_REFNUM_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
LOCATION_REFNUM
LOCATION_REFNUM contains a reference number for the location on the invoice.
INVOICE_STOP_REASON_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
EQUIPMENT_PREFIX
Equipment prefix.
EQUIPMENT_NUMBER
EQUIPMENT_NUMBER contains the equipment number on the invoice.
START_DATE
Start date.
END_DATE
End date.
