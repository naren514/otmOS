Definitions - GTM_DECL_MESSAGE

This Table maintains the messages exchanged between the different parties involved in the process of Declaration.

GTM_DECL_MESSAGE_GID
GTM Declaration Message GID is a unique key across all domains.
GTM_DECL_MESSAGE_XID
GTM Declaration Message XID is a unique key in a particular domain.
MESSAGE_DATE
Date on which message is received or sent.
DESCRIPTION
Description for the message.
SENDER_ID
ID of the sender of the message.
RECEIVER_ID
ID of the receiver of the message.
COUNTRY_CODE3_GID
Country code
IS_INBOUND
This field specifies if the given message is inbound or not.
STATUS
It is the Status of the message. It may have values as Created, Sent, Received, Error, Processing or processed. The default value is Created.
BS_STATUS_GROUP_GID
Status Group ID.
BS_STATUS_CODE_GID
Message Function Code (original, additional, accepted, cancelled etc.)
BS_STATUS_REASON_CODE_GID
Status Reason code GID, eg. Reason Code or Error.
GTM_TRANSACTION_GID
