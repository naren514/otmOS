Definitions - GTM_TARIFF_CODE

Captures the tariff code details.

GTM_TARIFF_CODE_GID
Contains the unique identifier for the Tariff Code.
GTM_TARIFF_CODE_XID
Contains the unique external identifier for the tariff code.
GTM_PROD_CLASS_TYPE_GID
Contains the product classification type
GTM_DATA_VERSION_GID
Contains the data version of the tariff code.
TARIFF_CODE_SHORT_NAME
Contains the text for the short name of the tariff code.
TARIFF_CODE_NAME
Contains the text for the name of the tariff code.
EXTERNAL_REFERENCE_CODE
Contains the reference code for the tariff code in external system.
GTM_TARIFF_TYPE_GID
Contains the tariff type the code is related to.
IS_DEFAULT
Specifies if the tariff code is the default code.
IS_INCLUDE_COUNTRY
Specifies if the countries mentioned in the table TARIFF_COUNTRY are included in the rate calculation.
GTM_TRADE_AGREEMENT
Contains the trade agreement
GTM_TRADE_PREFERENCE_GID
Gtm trade preference ID.
