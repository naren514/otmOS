Definitions - GTM_DECL_MESSAGE_REMARK

This Table maintains the Remarks for the messages exchanged between the different parties involved in the process of Declaration.

GTM_DECL_MESSAGE_GID
GTM Declaration Message GID is part of the primary key and is a foreign key to the primary key of table GTM_DECL_MESSAGE.
REMARK_SEQUENCE
It is a part of primary key and is the unique number generated using BNG.
REMARK_QUAL_GID
Remark qualifier GID is a foreign key to the primary key of table Remark Qualifier.
REMARK_TEXT
Remark Text.
