Definitions - INT_EXT_SYS_TRANSLATION

EXTERNAL_SYSTEM_GID
This is the External System Gid.
GLOG_XML_ELEMENT_GID
This is the interface type indicating the XML being supported.
SEQUENCE_NUM
This is the sequence number used to indicate the order of the stylesheets applied.
STYLESHEET_NAME
This is the filename of the stylesheet profile to be applied.
APPLY_XSL_TO
Specifies if the XSL transformation should be applied to the Transmission or the GLogXMLElement. This only applies to Transmission with multiple Transactions.
ADD_XML_DECLARATION
Specifies if the XML Declaration should be suppressed.
MULTI_TRANSACTION_WRAPPER
Specifies the element name used to wrap all the Transactions. Default is the Transmission element. This only applies to Transmission with multiple Transactions.
STYLESHEET_PROFILE_GID
Stylesheet Profile to transform the xml message.
