Definitions - GTM_DECLARATION_TYPE

This table is for creating/maintaining different order type names like ASN, an OTM Shipment or Delivery.

GTM_DECLARATION_TYPE_GID
This field represents unique declaration type ID accross all domains.
GTM_DECLARATION_TYPE_XID
This field represents unique declaration type ID in a particular domain.
GTM_DECLARATION_TYPE_NAME
Name for a declaration type.
DESCRIPTION
This field describes the declaration type.
GTM_REGIME_GID
This field describes the regime of the declaration type.
CODE
This field represents the code provided by goverments or regimes for any type of declaration.
REQ_BS_STATUS_GRP_PF_GID
This field represents request status group profile. Which is a foreign key to BS_STATUS_GROUP_PROFILE table.
RESP_BS_STATUS_GRP_PF_GID
This field represents response status group profile. Which is a foreign key to BS_STATUS_GROUP_PROFILE table.
LOGIC_CONFIG_GID
This field represents logic configuration used. Which is a foreign key to LOGIC_CONFIG table.
