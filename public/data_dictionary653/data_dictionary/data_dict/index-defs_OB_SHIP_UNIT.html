Definitions - OB_SHIP_UNIT

The details for a base order can be either specified by the line item (OB_LINE and related tables) or by the shipping units. The shipping units include the specification of pallets or packaging information. The shipper may not know or may choose not to specify the contents within the palettes. Other information like weight, height, pickup times etc. may be specified.

OB_SHIP_UNIT_GID
An order can be identified by the items within the order or the ship units. This field is a unique identifier to define the ship units. The gid contains the domain portion.
OB_SHIP_UNIT_XID
Similar to the gid except for the domain portion.
ORDER_BASE_GID
A unique identifier to define the order base in the system. It is system generated. The gid contains the domian portion.
SHIP_UNIT_COUNT
The count of the ship units in this line.
SOURCE_LOCATION_GID
