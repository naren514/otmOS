Definitions - EQUIPMENT_GROUP_COMPARTMENT

This table stores the details about the individual portions within the equipment group. An equipment group may have one or more compartments. This table records the size (length, width, height), weight and volume information about each compartment. A link to temperature control for each compartment can also be made.

EQUIPMENT_GROUP_GID
EQUIPMENT_GROUP_GID is the unique identifier for the equipment group.
COMPARTMENT_NUM
COMPARTMENT_NUM identifies the compartment by number.
MAX_WEIGHT
MAX_WEIGHT specifies the maximum weight for the equipment group compartment.
MAX_WEIGHT_UOM_CODE
The unit of measure for the
MAX_WEIGHT_BASE
MAX_WEIGHT specifies the maximum weight for the equipment group compartment in base units.
MAX_VOLUME
MAX_VOLUME specifies the maximum capacity for the equipment group compartment.
MAX_VOLUME_UOM_CODE
The unit of measure for the max_volume column.
MAX_VOLUME_BASE
MAX_VOLUME specifies the maximum capacity for the equipment group compartment in base units.
LENGTH
LENGTH specifies the length of the equipment group compartment.
LENGTH_UOM_CODE
Unit of measure for LENGTH column.
LENGTH_BASE
LENGTH in base units.
WIDTH
WIDTH specifies the width of the equipment group compartment.
WIDTH_UOM_CODE
The unit of measure for the width column.
WIDTH_BASE
WIDTH in base units.
HEIGHT
HEIGHT specifies the height of the equipment group compartment.
HEIGHT_UOM_CODE
Unit of measure for HEIGHT column.
HEIGHT_BASE
HEIGHT in base units.
TEMPERATURE_CONTROL_GID
A unqie identifier to define the temperature controls value for the commodity. The gid contains the domian portion.
ALLOW_COMMODITY_MIXING
ALLOW_COMMODITY_MIXING specifies whether or not different commodity items can be shipped in the same compartment in an equipment group. Valid values are Y for yes and N for no.
DOOR_HEIGHT
Defines the height of the door for the comparment.
CURVE_GID
The curve function for the roof of the compartment.
START_LENGTH_POSITION
Defines the starting position for the compartment length wise
START_WIDTH_POSITION
Defines the starting position for the compartment width wise
START_HEIGHT_POSITION
Defines the starting position for the compartment height wise
COMPARTMENT_TYPE_GID
A unique identifier to define the compartment type
HAS_METER
Indicates if the compartment has METER or not(The METER is a device that is used the measure how much product is discharged at any given stop).
COMMODITY_GID
Commodity that is compatible with the compartment
IS_BULK
Indicates if it is a BULK. If Y then BULK If N then PACKAGED
