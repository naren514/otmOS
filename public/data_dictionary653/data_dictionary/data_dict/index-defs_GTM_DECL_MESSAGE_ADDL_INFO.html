Definitions - GTM_DECL_MESSAGE_ADDL_INFO

This Table maintains additional information related to the messages exchanged between the different parties involved in the process of Declaration.

GTM_DECL_MESSAGE_GID
GTM Declaration Message GID is part of the primary key and is a foreign key to the primary key of table GTM_DECL_MESSAGE.
SEQUENCE
It is a part of primary key and is the unique number generated using BNG.
TYPE
It is the message type given by government.
CODE
It is the message code given by government.
SEVERITY
It is the message severity given by government.
ADDL_TEXT
It is the message description given by government.
