Definitions - FLEX_COMMODITY_QUAL

This table identifies the enumerated values for qualifying the commodities. This is just a way to separate the qualifiers. The information can be directly stored in the Flex_commodity_Profile_D table.

FLEX_COMMODITY_QUAL_GID
FLEX_COMMODITY_QUAL_GID contains the unique identifier for the flex commodity qualifier.
FLEX_COMMODITY_QUAL_XID
FLEX_COMMODITY_QUAL_XID contains the unique external identifier for the flex commodity.
