Definitions - MATCH_VALIDATION_RULE_ORDER

The value of the specified invoice refnum qualifier will be validated against the corresonding order refnum value for the orders on the shipment during the invoice-shipment match validation process

MATCH_VALIDATION_RULE_GID
Id of the validation rule
INVOICE_REFNUM_QUAL_GID
Values of this invoice refnum qualifier should be validated against values of the order refnum qual or value of an order release field
ORDER_REFNUM_QUAL_GID
Indicates the values of the invoice refnum qual specified should be validated against the values of this qualifier on the order release
