Definitions - EQUIPMENT_ATTRIBUTE_QUAL

Qualifier attributes to store additional information about the equipment. For e.g., Weight Capacity, Volume Capacity, Dunnage, Tare Weight, Length, Equipment Order Description etc.

EQUIPMENT_ATTRIBUTE_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
EQUIPMENT_ATTRIBUTE_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
