Definitions - SHIP_GROUP_REMARK

The ship_group_remark table contains remarks for the shipment group.

SHIP_GROUP_GID
Unique identifier for the shipment group for which the remark is defined.
REMARK_SEQUENCE
The one-up number to make the remark record unique.
REMARK_QUAL_GID
The remark qualifier, or remark type.
REMARK_TEXT
User entered remark on the shipment group.
