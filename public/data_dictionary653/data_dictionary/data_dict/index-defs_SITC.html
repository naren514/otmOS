Definitions - SITC

This table defines the SITC governmental code and name for the item.

SITC_GID
SITC_GID contains the unique identifier for the Standard International Trade Classification (SITC) of the item being shipped.
SITC_XID
SITC_GID contains the unique external identifier for the Standard International Trade Classification (SITC) of the item being shipped.
SITC_NAME
SITC_GID contains the Standard International Trade Classification (SITC) name for the item being shipped.
