Definitions - BN_RANGE_COUNT

Defines pre-assigned ranges counts

BN_RANGE_GID
ID of a named range
IS_OPEN
Whether or not the range is open for generating a number. Once a range is closed, it cannot be reopened.
POOLED_COUNT
Total numbers available when the range is created.
GENERATED_COUNT
Numbers in generated state if audit is on; numbers generated if audit is off.
ASSIGNED_COUNT
Numbers in assigned state.
DESTROYED_COUNT
Numbers in destroyed state.
