Definitions - BN_GENERATOR

This table is used in conjunction with the Business Number generator scheme. This table stores the generator class (currently only Picture string) The business number module is designed with the following 2 types of generators in mind Pictures string: implemented, this is how all of the current business number rules are defined XSL file: not implemented yet, but easily pluggable.Requires considerable amount of XSL knowledge to define a rule through XSL

BN_GENERATOR_GID
BN_GENERATOR_GID contains the unique identifier for the Business Number (BN) Generator .
BN_GENERATOR_XID
BN_GENERATOR_XID contains the unique external identifier for the Business Number (BN) Generator.
GENERATOR_CLASS
GENERATOR_CLASS contains the Java class used for the Business Number Generator.
