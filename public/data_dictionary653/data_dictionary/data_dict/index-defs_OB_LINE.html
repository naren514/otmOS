Definitions - OB_LINE

This table consists of the individual line items for the order base. For e.g., if the order base has an order of: 500 Nike Shoes, from Philly to N.Y. latest by Jan/1/2001 the line items may contain: 200 Nike Shoes, from Philly to N.Y. 200 Socks, from Nashville to N.Y. Etc. Generally the line item should have the same destination for an order base, however that is not a rule. The tags are used in the line item for tracking purposes. These can be any combination of defined serial numbers. Four tags can be specified for tracking at various levels.

OB_LINE_GID
OB_LINE_GID contains the unique identifier for the order base line item.
OB_LINE_XID
OB_LINE_XID contains the unique external identifier for the order base line item.
ORDER_BASE_GID
ORDER_BASE_GID contains the unique identifier for the order base.
SOURCE_LOCATION_GID
SOURCE_LOCATION_GID contains the unique identifier for the source location.
DEST_LOCATION_GID
DEST_LOCATION_GID contains the unique identifier for the destination location for the order base line item.
TAG1
Additional information about the item. Like the SPLC code etc.
TAG2
Additional information about the item. Like the SPLC code etc.
TAG3
Additional information about the item. Like the SPLC code etc.
TAG4
Additional information about the item. Like the SPLC code etc.
IS_SPLIT_ALLOWED
A true or false flag to define whether the line item can be split during the order release or shipment.
WEIGHT
WEIGHT contains the weight for the order base line item.
WEIGHT_UOM_CODE
Unit of measure for WEIGHT column.
WEIGHT_BASE
WEIGHT in base units.
VOLUME
VOLUME contains the volume for the order base line item.
VOLUME_UOM_CODE
Unit of measure for VOLUME column.
VOLUME_BASE
VOLUME in base units.
DECLARED_VALUE
DECLARED_VALUE contains the stated value of the the order base line item.
DECLARED_VALUE_BASE
DECLARED_VALUE in base units.
DECLARED_VALUE_CURRENCY_GID
DECLARED_VALUE_CURRENCY_GID contains the currency of the stated value for the order base line.
EARLY_PICKUP_DATE
The earliest date when the item can be picked up.
LATE_PICKUP_DATE
The latest date when the item can be picked up.
EARLY_DELIVERY_DATE
The earliest date when the item can be delivered.
LATE_DELIVERY_DATE
The latest date when the item can be picked delivered.
PACKAGED_ITEM_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
LOAD_POINT
source load point
UNLOAD_POINT
dest unload point
PICKUP_IS_APPT
Indicates if the early/late pickup times are appointment times
DELIVERY_IS_APPT
Indicates if the early/late delivery times are appointment times
PLAN_FROM_LOCATION_GID
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
PLAN_FROM_LOAD_POINT
load point associated with plan from location
PLAN_TO_LOCATION_GID
Alternate plan to location.
PLAN_TO_UNLOAD_POINT
unload point associated with plan to location
ROUND_TO
Round to packaging unit quantity (PUQ), layer (LAY), or transport handling unit (THU).
