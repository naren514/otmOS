Definitions - LOCATION_PROFILE

This is an alternate method of grouping the locations by creating enumerated profiles. This allows flexibility to group locations in a profile and specify if they are compatible for delivery.

LOCATION_PROFILE_GID
LOCATION_PROFILE_GID contains the unique identifier for the location's profile.
LOCATION_PROFILE_XID
LOCATION_PROFILE_XID contains the unique external identifier for the location profile.
LOCATION_PROFILE_NAME
LOCATION_PROFILE_NAME contains the name of the location profile.
