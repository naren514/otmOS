Definitions - ORL_SPECIAL_SERVICE

This table specifies the special services required if any for the order release line against the base order. See Special_Services table for further details about the services.

ORDER_RELEASE_LINE_GID
A unique identifier to define the multiple order release lines per order release. The gid has the domain portion.
SPECIAL_SERVICE_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
