Definitions - REPETITION_SCHEDULE

SCHEDULE_TYPE
Determines whether the repetition schedulel is a consol or not. Valid values are DYNAMIC TRIP, STATIC TRIP, FLIGHT, VESSEL, GROUND SERVICE, DETACHABLE TRIP. When this field is populated, it is a consol.
FLIGHT_GID
Should be populated if the schedule type is a FLIGHT.
VOYAGE_GID
Should be populated when the schedule type is a TRIP.
EQUIPMENT_GROUP_GID
Required when the schedule type is a TRIP.
IS_IGNORE_EQUIPMENT_CAPACITY
Determines whether or not to check the equipment's capacity when adding orders to the trip.
MAX_WEIGHT
The weight pre-booked on the specified flight consol.
MAX_WEIGHT_UOM_CODE
The unit of measure for the max_weight.
MAX_WEIGHT_BASE
The weight pre-booked on the specified flight consol in base units.
MAX_VOLUME
The volume pre-booked on the specified flight consol.
MAX_VOLUME_UOM_CODE
The unit of measure for the max_volume.
MAX_VOLUME_BASE
The volume pre-booked on the specified flight consol in base units.
START_TIME_ADJ
Only defined for DYNAMIC TRIP schedule.
START_TIME_ADJ_UOM_CODE
Unit of measure for start_time_adj.
CUTOFF_TIME
Time this schedule is used up to.
