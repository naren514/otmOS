Definitions - LOCATION_ROLE_PROFILE

Detail information about the location role(s). The detail information consists of the minimum stop time at the location, location hours per role. For e.g., the location time for the loading zone can be 8 AM to 5 PM but unloading time may be 12 noon to 7 PM.

LOCATION_GID
LOCATION_GID contains the unique identifier for the location.
LOCATION_ROLE_GID
LOCATION_ROLE_GID contains the unique identifier for the location role.
CALENDAR_GID
CALENDAR_GID contains the unique identifier for the calendar used for the location role profile.
FIXED_HANDLING_TIME_BASE
FIXED_HANDLING_TIME in base units.
VAR_HANDLING_TIME_BASE
VAR_HANDLING_TIME in base units.
PCT_ACTIVITY_BEFORE_LOC_OPEN
This value will be devided by 100 in the application to be used as a percentage. The valid value for this column is >= 0 and <= 100. When it is null, treat as 0 for location role SHIPPING/RECEIVING
PCT_ACTIVITY_AFTER_LOC_CLOSE
This value will be devided by 100 in the application to be used as a percentage. The valid value for this column is >= 0 and <= 100. When null, treat as 100 for location role SHIPPING/RECEIVING
