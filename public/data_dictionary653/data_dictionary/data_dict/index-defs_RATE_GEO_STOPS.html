Definitions - RATE_GEO_STOPS

More granular than RATE_OFFERING_STOPS, this table indicates the number of stops that are included in the rate before the carrier starts to charge extra. While RATE_OFFERING_STOPS has this information for a contract in general, this table relates the information to a particular geo (lane).

RATE_GEO_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PER_STOP_COST_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PER_STOP_COST_BASE
PER_STOP_COST in base units.
