Definitions - COMMODITY

This is a grouping of the items. Multiple items can be part of a commodity. This table may define the transportation mode of the items within the commodity. For e.g., hazardous materials may not be transported via air.

COMMODITY_GID
COMMODITY_GID contains the unique identifier for the commodity.
COMMODITY_XID
COMMODITY_XID contains the unique identifier for the commodity.
COMMODITY_NAME
COMMODITY_NAME contains text for the name of the commodity.
TEMPERATURE_CONTROL_GID
A unqie identifier to define the temperature controls value for the commodity. The gid contains the domian portion.
MODE_PROFILE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
IS_TEMP_CONTROL
IS_TEMP_CONTROL contains a Y for yes if the commodity is temperature controlled or an N if it is not.
REQ_EQPMT_GROUP_PROFILE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
