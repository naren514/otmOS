Definitions - RATE_GROUP

The list of values for rate types. For example, the rates might be standard, or they might be the carrier"s special discount rate. This table is referred to in the RATE_OFFERING table. Example: standard, discount

RATE_GROUP_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
RATE_GROUP_XID
GC3 Unique identifier for the entity. Is used for creating the GID
