Definitions - PROVINCE_CODE_PROFILE

GC3 profile to allow specifying a list of province codes that may be designated compatible or incompatible for applicable business logic

PROVINCE_CODE_PROFILE_GID
Gid to identify the Province Code Profile
PROVINCE_CODE_PROFILE_XID
Xid to identify the Province Code Profile
DESCRIPTION
User description of the province code profile
IS_COMPATIBLE
Indicates if the province codes in the profile are compatible or incompatible when used by logic
