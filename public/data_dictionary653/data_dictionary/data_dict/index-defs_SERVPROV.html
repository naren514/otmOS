Definitions - SERVPROV

This table stores the information about the service providers.

SERVPROV_GID
A unique identifier to define the locations withing GC3. The gid contains the domain portion.
SERVPROV_XID
Similar to the gid except for the domain portion.
TENDER_RESPONSE_TIME
The expected response time by this service provider for shipment tenders.
TENDER_RESPONSE_TIME_UOM_CODE
Unit of measure for TENDER_RESPONSE_TIME column.
TENDER_RESPONSE_TIME_BASE
TENDER_RESPONSE_TIME in base units.
MODE_PROFILE_GID
A unique identifier to define the mode profiles(group) that the service provider uses.
MATCH_RULE_PROFILE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
MATCH_VALID_RULE_PROFILE_GID
profile for the match validation rules that should be tested if the validate match action is run on an invoice for this service provider
AUTO_APPROVE_RULE_PROFILE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ALLOCATION_RULE_PROFILE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
IS_DISPATCH_BY_REGION
A flag to turn on/off the Tender by Locale feature.
OUT_XML_PROFILE_GID
Associates the service provider with an out_xml_profile, which helps define the content of outgoing XML to this service provider.
RATE_SERVICE_GID
Rate Service GID. This will be the default Rate Service used to re-drive the shipment if the service provider does not have a rate offering that matches the shipment
IS_ACCEPT_SPOT_BIDS
Is set to Y(es) if the service provider will participate in spot bid tendering.
IS_ACCEPT_BROADCAST_TENDERS
Is set to Y(es) if the service provider will participate in broadcast tendering.
IS_LOCALIZE_BROADCAST_CONTACT
If "Y", use the preference /language, etc of BROADCAST_SPOT_CONTACT. Otherwise, use shipment's LOGISTICS involved party's preference is used for broadcast notification.
BROADCAST_SPOT_CONTACT_GID
This field can be used to override the shipper's primary contact to be used to determine localization preferences for tender notification
DO_CONDITIONAL_ACCEPTS
used to determing if the servprov can participate conditional booking.
LINE_APPROVE_TOL_PROFILE_GID
Optional; if specified, indicates the line approval tolerance rule profile to apply
SCAC_GID
The Standard Carrier Alpha Code for this Service Provider.
EQUIPMENT_MARK_LIST_GID
The Equipment Mark List for this Service Provider.
IS_FLEET
Only shipments created for fleet service providers can perform fleet related actions.
ALLOW_SPOT_RATING
Flag to indicate that service provider is valid for Spot Rating actions
OBT_XML_PROFILE_GID
Associates the service provider with an online_booking_xml_profile, Which helps to define the content of XML to this service provider for UI display.
