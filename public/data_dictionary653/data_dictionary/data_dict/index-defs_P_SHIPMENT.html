Definitions - P_SHIPMENT

Procurement shipment - a light weight version of the gc3 shipment with minimal information

SOURCE_LOCATION_GID
first stop
DEST_LOCATION_GID
last stop
START_TIME
start time at first stop
END_TIME
end time at last stop
WEIGHT
gross weight
VOLUME
gross volume
NUM_STOPS
total number of stops on the shipment
COST
does not include accessorials
IS_GC3_SHIPMENT
Y if this shipment came from gc3, N otherwise
COMMODITY_GID
Commodity for shipment (only one allowed)
SHIPMENT_CATEGORY
Optional user-defined shipment category text.
MARKED_FOR_PURGE
Identifies the p_shipments that can be purged with the next purge schedule
