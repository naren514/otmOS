Definitions - GTM_TARIFF_RATE

Captures the tariff rate related to a classification code.

GTM_TARIFF_CODE_GID
Contains the identifier of the tariff code.
CLASSIFICATION_CODE
Contains the product classification code GID of the related classification code.
ADDVALOREM_RATE
Contains the rate expressed as percentage.
PER_UNIT_CHARGE
Contains the per unit charge value.
UOM_CODE
Contains the Unit of Measure (UOM) code to use with per unit charge.
GTM_PROD_CLASS_TYPE_GID
Contains the product classification type.
