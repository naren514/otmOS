Definitions - GTM_PERCENTAGE_VALUE_QUAL

Table to store the percentage value qualifiers. these qualifiers will be used to define percentage values for currency and Quantity accross GTM on various objects.

GTM_PERCENTAGE_VALUE_QUAL_GID
contains the unique identifier for the percentage value qualifier
GTM_PERCENTAGE_VALUE_QUAL_XID
contains the XID unique identifier for the percentgae value qualifier
DESCRIPTION
column to add user description about the percentage value qualifier
