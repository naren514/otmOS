Definitions - GTM_TR_ITEM_STRUCTURE

Table to store the result of Structure assignment. Also this will enable user to create a struture specific toa transaction Line item dynamically.

GTM_TR_ITEM_STRUCTURE_GID
Unique ID to represent a transaction Line Item Struture.
GTM_TR_ITEM_STRUCTURE_XID
contains the XID unique identifier for the transaction line item structure.
GTM_TR_ITEM_STRUCTURE_NAME
contains text for the name of the transaction Line item structure
TRADE_DIRECTION
Trade Direction. Used to define import, export or both specific structure
GTM_PROD_CLASS_TYPE_GID
Product Classification Type ID Used to define product classification type specific structure
GTM_PROD_CLASS_CODE_GID
Product Classification Code Used to define product classification code specific structure
GTM_STRUCTURE_GID
GTM_STRUCTURE_GID contains the unique identifier for the trade item structure. This field will be automatically populated when the struture gets assigned automatically.
DESCRIPTION
DESCRIPTION contains text for the description of the trade item structure
MARKED_FOR_PURGE
The Column indicates whether the transaction structure has been marked for purge.
