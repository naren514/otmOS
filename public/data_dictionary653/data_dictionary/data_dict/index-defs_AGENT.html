Definitions - AGENT

AGENT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
AGENT_XID
GC3 Unique identifier for the entity. Is used for creating the GID
DATA_QUERY_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SAVED_CONDITION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
IS_ACTIVE
Indicates whether agent is active or not.
THRESHOLD_GID
A threshold defined at agent level. If the data changes of the lifetime events the agent listens to passed threshold tolerance, the agent will be triggered.
IS_OBJECT_LOCK
If Y, the business object is locked until the agent completes
OBJECT_LOCK_TIMEOUT
Object lock timeout, if IS_OBJECT_LOCK=Y
OBJECT_LOCK_POLLING
If not using JMS notification for cross-machine locks, the polling interval in msecs
IS_LONG_RUNNING_TASK
If Y, this agent is a long running task and should be monitored in the LONG_RUNNING_TASK table.
