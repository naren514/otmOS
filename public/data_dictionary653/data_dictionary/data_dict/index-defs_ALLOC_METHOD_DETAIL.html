Definitions - ALLOC_METHOD_DETAIL

For identifying different allocation methods possible i.e. OR-W20-V80 Allocate to Order Relase by Weight 20% and Volume 80%

ALLOC_METHOD_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ALLOC_METRIC_GID
The metric by which we want to perform the allocation
PERCENTAGE
The weightage given to the metric in performing the allocation
ALLOC_TARGET_GID
Identifies to which allocation target object this metric and method corresponds to, as not all metrics can be given for all of the allocation target objects.
