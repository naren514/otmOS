Definitions - P_BID_ROUND

Represents a bidding round within a project

P_PROJECT_GID
Sequence of the bid round: 1, 2, ...
OPEN_DATE
Date/time that bidding starts - no bids accepted before this date
CLOSE_DATE
Date/time that bidding ends - no bids accepted after this date
SELECTED_SOLUTION_GID
The solution selected by the user for this bid round
MARKED_FOR_PURGE
This column will identify the rows which will be purged
