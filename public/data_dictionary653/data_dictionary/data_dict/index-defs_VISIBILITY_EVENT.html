Definitions - VISIBILITY_EVENT

The data queue for shipment events

Q_EVENT_SEQ
Queue sequence for the shipment event. Sequence name is VISIBILITY_EVENT_SEQUENCE.
Q_PREEMPTION_PRIORITY
A negative sequence used for queue preemption
Q_START
Time the event was last queued. If the event is scheduled, this will equal the scheduled time. If resubmitted, the resubmit time. Otherwise it matches the insert date.
Q_STATE
Queue state of the event (QUEUED, ACTIVE, CANCELED, PROCESSED, FAILED)
Q_TIME
Time the event spent in queue
Q_PROCESS_START
Time the event was last dequeued and passed to the executor for processing
Q_PROCESS_TIME
Time the shipment event took to execute
Q_LOG_PROCESS_ID
ID linking the shipment event execution to the system logs
Q_PROCESS_MACHINE_GID
The application machine that polled and executed the shipment event
Q_RELATED_QUEUE_SEQ
Sequence of the parent queue event that populated this event
P_QUEUE_START_KEY
Index for date-based partitioning
VISIBILITY_EVENT_TYPE_GID
Event type ID
EVENT_DATE
Event date
AVAILABLE_DATE
Available date
SEND_DATE
Send date
EVENT_REMARK
Remark for the event
EVENT_REASON
Event reason
PIECES
Number of package items
EVENT_LOCATION
location generates the event
ORIG_LOCATION
Original Location for the shipment
DEST_LOCATION
Destination location for the shipment
EVENT_DATE_OFFSET
Represents the UTC offset for the EVENT_DATE column
EVENT_FACILITY
Facility where the event occured
