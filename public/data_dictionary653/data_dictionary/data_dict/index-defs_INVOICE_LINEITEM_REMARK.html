Definitions - INVOICE_LINEITEM_REMARK

Can have unlimited remarks per line item per invoice. Additional free format line item information that is not associated to a specific qualifier/refnum.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
LINEITEM_SEQ_NO
LINEITEM_SEQ_NO contains the sequence numebr for the line item on the invoice.
REMARK_SEQ_NO
REMARK-SEQ_NO contains the sequence numebr for the line item remark on the invoice.
REMARK_QUAL_IDENTIFIER
Remark qualifier.
REMARK_TEXT
REMARK_TEXT contains remarks about the line item on the invoice.
