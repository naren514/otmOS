Definitions - DM_TRANSACTION_LINEITEM

It captures the chargeable days and cost for demurrage transaction. Example include for the PVT car, its possible that there are both storage and detention charges applied for a demurrage transaction.

DM_TRANSACTION_SEQ_NO
DM_TRANSACTION_SEQ_NO contains the unique identifier for the demurrage transaction.
DM_TRANSACTION_LINEITEM_SEQ_NO
DM_TRANSACTION_LINEITEM_SEQ_NO contains the unique identifier for the demurrage transaction lineitem.
SPECIAL_SERVICE_GID
Determine the type of charge. Example could be Demurrage - Empty to Load. These are modeled as special services.
DWELL_DAYS
length of time that elapsed from the moment a car reaches the destinations till its released.
CREDIT_DAYS
days that are free of charge
DEBIT_DAYS
dwell days offset by credit days
ADJUSTMENT_REASON_GID
Reason Code for adjusting the chargeable days or charges.
IS_SYSTEM_GENERATED
Is the line item system generated.
RATE_OFFERING_GID
Rate Offering linked to demurrage line item. Its helps to determine demurrage charges.
RATE_GEO_GID
Rate Geo linked to demurrage transaction line item. Its helps to determine demurrage charges.
SHIPMENT_GID
Line item charges is linked as NFRC/demurrage charge shipment.
SECONDARY_CHARGE_RULE_GID
secondary charge rule ID
CHARGE_TYPE
Charge type possible values are NFRC shipment, inbound shipment, outbound shipment and Secondary charge shipment.
COST
Line item Charges from the linked NFRC/Demurrage Charge Shipment.
