Definitions - ACTIVITY_CALENDAR

This table is a link between the Activity and the Calendar entries in the GC3 system. An activity can have many calendar activities like many Pickup times. Similarly a calendar profile can have many activities, like activities during a working hour calendar profile can be Pickup, Dispatch etc. This there is a many to many relationship between calendar profiles and activities. This table resolves the many to many relationship.

ACTIVITY_CALENDAR_ID
ACTIVITY_CALENDAR_ID is the system-generated unique ID for ACTIVITY_CALENDAR. (IDs are internal system indexes; therefore, they are usually not recognizable to users.)
CALENDAR_GID
CALENDAR_GID is the unique identifier for a calendar. A calendar entry defines when certain activities can occur.
ACTIVITY_GID
ACTIVITY_GID is the unique identifier of an activity. This entry defines an activity with the time(s) when that activity can occur.
IS_WORK_ON
IS_WORK_ON is a flag describing whether a specific activity calendar is active. (Valid values are Y for yes (the calendar is active) or N for no (the calendar is not active).
BEGIN
BEGIN contains the time when an activity can start.
DURATION
DURATION contains the length of time an activity can take place.
ACTIVITY_TIME_FACTOR
Acitivity factor provides a multiplication factor for the activity.
