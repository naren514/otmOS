Definitions - CAPACITY_COMMITMENT

Store a service provider's capacity commitment by capacity group, equipment type, lane and time period

CAPACITY_COMMITMENT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
CAPACITY_COMMITMENT_XID
GC3 Unique identifier for the entity. Is used for creating the GID
CAPACITY_GROUP_GID
Unique Identifier for each CAPCITY GROUP. This is a combination of XID and Domain Name. This is part of the compsite PK for this table.
EFFECTIVE_DATE
The date from which the CAPACITY Commitment is applicable. This is part of the compsite PK for this table.
EQUIPMENT_TYPE_GID
A unique identifier to define and group the equipments within the system. The gid contains the domain portion.
MILEAGE_BASE
MILEAGE in base units.
WEIGHT_BASE
WEIGHT in base units.
VOLUME_BASE
VOLUME in base units.
MONETARY_CURRENCY_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
MONETARY_BASE
MONETARY in base units.
COMMITMENT_LANE_OBJECT_TYPE
Should this capacity commitment be recorded by shipment, or by order?
PRIMARY_COUNT_TYPE
This is used to determine which one to be displayed in UI if the commitment has multiple count types
