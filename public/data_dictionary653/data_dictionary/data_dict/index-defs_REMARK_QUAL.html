Definitions - REMARK_QUAL

This table links to the OB_Remark table to further describe the details of the remarks on the base order. The remark qualifiers are the enumerated values for the remarks.

REMARK_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
REMARK_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
REMARK_LEVEL
Level of importance of the Remark.
IS_VISIBLE_IF_MASTER_CARR
Identifies whether the remarks with this qualifier should display on related shipments.
UPDATE_FLAG
Specifies if the remark qualifier can exist on the business object multiple times, or is restricted to occur only once and whether the value is allowed to be updated.
TEXT_TEMPLATE_GID
Default remark text.
DATA_QUERY_TYPE_GID
Remark Qualifier must be visible only for object that you have chosen in Data Query Type
