Definitions - REPORT_COMMON_PROPERTIES

DOMAIN_NAME
Domain of the User. Is used for creating the GID.
INSERT_USER
For audit trail purposes. This field stores the id of the user who first entered the record.
INSERT_DATE
For audit trail purposes. This field stores the date and time when the record was first entered in the database.
UPDATE_USER
For audit trail purposes. This field stores the id of the user who last updated the record.
UPDATE_DATE
For audit trail purposes. This field stores the date and time when the record was updated in the database.
