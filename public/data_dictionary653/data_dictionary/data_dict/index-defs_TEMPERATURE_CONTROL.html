Definitions - TEMPERATURE_CONTROL

This table stores various user defined temperature ranges that can be used to categorize the items and commodities.

TEMPERATURE_CONTROL_GID
TEMPERATURE_CONTROL_GID contains the unique identifier for temperature control information for the item being shipped. Examples include: AMBIENT, INSULATED, REFRIGERATED
TEMPERATURE_CONTROL_XID
TEMPERATURE_CONTROL_XID contains the unique external identifier for the temperature control of the item being shipped.
TEMPERATURE_CONTROL_NAME
TEMPERATURE_CONTROL_NAME contains the temperature control name for the item being shipped.
MIN_TEMP
MIN_TEMP contains the minimum temperature at which the item can be shipped.
MIN_TEMP_UOM_CODE
Unit of measure for MIN_TEMP column.
MIN_TEMP_BASE
MIN_TEMP in base units.
MAX_TEMP
MAX_TEMP contains the maximum temperature at which the item can be shipped.
MAX_TEMP_UOM_CODE
Unit of measure for MAX_TEMP column.
MAX_TEMP_BASE
MAX_TEMP in base units.
