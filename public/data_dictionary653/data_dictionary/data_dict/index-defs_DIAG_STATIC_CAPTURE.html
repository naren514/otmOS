Definitions - DIAG_STATIC_CAPTURE

Grouping of configuration data for static capture

DIAG_STATIC_CAPTURE_GID
Unique identifier for the data capture
JAVA_PLUGIN_GID
Plugin of type CONFIGURATION CAPTURE
DESCRIPTION
Text description of the data
EXTRA_ARGS
For open scripting architecture, a set of extra arguments to pass to the CONFIGURATION CAPTURE plugin. This arguments can be passed to an external script (e.g. generating Unix statistics)
