Definitions - LOGIC_PARAMETER

Parameter values (instances) for logic configuration

LOGIC_SCENARIO_GID
Specifies the scenario where the parameter applies to. e.g., DIRECT/LINEHAUL/CROSSDOCK scenario in MULTISTOP config. Use DEFAULT if there is no other scenario.
PARAM_UOM_CODE
The UOM code such as LB, FT etc, depending on uom_type of the parameter qualifier.
