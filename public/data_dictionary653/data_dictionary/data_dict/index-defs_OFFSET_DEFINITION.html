Definitions - OFFSET_DEFINITION

This table is used to create the offset profile entries for the user company. It specifies the schedules, time periods, holiday events etc. for the entire year for the company. These entries are used by the GC3 engine when scheduling the pickup or delivery of an order

OFFSET_DEFINITION_GID
is the unique identifier for offset.
OFFSET_DEFINITION_XID
is the unique external identifier for offset.
DESCRIPTION
It describes name/purpose of the offset
