Definitions - DISTANCE_SERVICE_TIME

This table is created to allow for GEPS to have a distance band based service commitment Rate Service Type. Example: Service Commitment (Weekend Equal 1 Day) 0 to 500 Miles - 3rd Morning 501 to 1,000 Miles - 4th Morning 1,001 to 1,500 Miles - 5th Morning 1,501 to 2,000 Miles - 6th Morning 2,001 to 3,000 Miles - 7th Morning

RATE_SERVICE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
DISTANCE_VALUE_BASE
DISTANCE_VALUE in base units.
