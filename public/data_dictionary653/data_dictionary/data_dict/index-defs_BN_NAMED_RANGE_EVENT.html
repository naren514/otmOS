Definitions - BN_NAMED_RANGE_EVENT

Defines threshold for different levels of depletion warnings.

BN_NAMED_RANGE_GID
The identifier of the named range to which this warning level applies.
WARNING_LEVEL
The depletion warning level.
DESCRIPTION
The description of the warning level.
WARNING_THRESHOLD
Determines when a depletion warning is raised. An event is raised if the available numbers in the range fall below the threshold.
MAX_WARNING_EVENTS
The maximum number of events to be raised for this level or warning.
ACTUAL_WARNING_EVENTS_COUNT
The actual number of events raised. The number is reset when the named range is modified by adding the new ranges or updating existing ranges.
