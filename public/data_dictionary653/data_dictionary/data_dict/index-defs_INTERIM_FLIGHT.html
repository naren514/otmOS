Definitions - INTERIM_FLIGHT

INTERIM_FLIGHT stores source and destination location details about interim flights. It does not store the information about any intermediate stops. This table is used to store the details of the interim flights.

FLIGHT_GID
FLIGHT_GID contains the unique identifier for the interim flight.
FLIGHT_NUMBER
FLIGHT_NUMBER contains the airline flight number for the interim flight.
SRC_LOCATION_GID
SRC_LOCATION_GID contains the unique identifier for the source location of the interim flight.
DEST_LOCATION_GID
DEST_LOCATION_GID contains the unique identifier for the destination location of the interim flight.
SERVPROV_GID
SERVPROV_GID contains the unique identifier for the service provider for the interim flight.
VEHICLE_TYPE_GID
VEHICLE_TYPE_GID is the vehicle type.
