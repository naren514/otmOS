Definitions - CMS_PARAM_GENERATOR

Specification for the generation of a document parameter.

GEN_TYPE
The type of generation to use for the parameter value. "BN" = Business Number Rule; "CONSTANT" = constant value; "SQL" = SQL statement returning a single value; "JAVA" = a Java plug-in
GEN_PLUGIN_GID
If gen_type="JAVA", the java plugin that generates the parameter value.
GEN_BN_RULE_GID
If gen_type="BN", the business number rule that generates the parameter value.
GEN_SQL
If gen_type="SQL", the SQL select statement that generates the parameter value.
GEN_CONSTANT
If gen_type="CONSTANT", the constant value of the parameter.
