Definitions - LRG_LOCATION_JOIN

This table will be used as a join between location resource group and location.

LOCATION_RESOURCE_GROUP_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
LOCATION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
CALENDAR_GID
The calendar GID that will be applied to the location of the location resource group.
PREFERENCE_LEVEL
This field used to get preference level of location resource group for Appointment scheduling
