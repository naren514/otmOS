Definitions - SERVPROV_ALIAS

This table stores the alias names for the service providers associated with the qualifiers. For example JBHT is the alias for JBHUNT under the SCAC qualifier.

SERVPROV_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SERVPROV_ALIAS_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
