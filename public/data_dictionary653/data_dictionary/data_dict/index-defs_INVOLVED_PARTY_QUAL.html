Definitions - INVOLVED_PARTY_QUAL

INVOLVED_PARTY_QUAL stores information about the parties that get involved in an order, such as the order owner, order supplier and consignee.

INVOLVED_PARTY_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
INVOLVED_PARTY_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
UPDATE_FLAG
Specifies if the qualifier can exist on a business object multiple times, or is restricted to occur only once and whether the value is allowed to be updated. If the value is MANY, then the object can have multiple values for the same qualifier. If the value is either UPDATE_OK or UPDATE_NOT_OK, then the rule is in effect, meaning only one value is allowed for a given qualifier. In the case of UPDATE_NOT_OK, the value may not be modified.
FROM_ADDR_CONTACT_GID
Configurable 'From-Address' contact details.
