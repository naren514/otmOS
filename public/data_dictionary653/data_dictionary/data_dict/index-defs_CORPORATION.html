Definitions - CORPORATION

This table is used in conjunction with the Location table. It contains the additional details about the location that is a corporation. It contains the Billing and Remittance information.

CORPORATION_GID
CORPORATION_GID contains the unique identifier for the corporation.
CORPORATION_XID
CORPORATION_XID contains the unique external identifier for the corporation.
CORPORATION_NAME
CORPORATION_NAME contains the name of the corporation.
IS_SHIPPING_AGENTS_ACTIVE
Indicates whether or not a corporation is active (used with agents)
