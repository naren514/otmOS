Definitions - CURRENCY

This table is used for the currency conversion in GC3. It stores the various countries and their currency names. The table Currrency_Exchange_Rate is used for the currency conversion logic.

CURRENCY_GID
CURRENCY_GID contains the unique identifier for the currency. Examples include USD for US dollars and DEM for Deutsche marks.
CURRENCY_XID
CURRENCY_XID contains the unique external identifier for the currency.
CURRENCY_NAME
CURRENCY_NAME contains the proper name for a currency. Examples include pesos and rupees.
COUNTRY_CODE3_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
