Definitions - BN_RANGE_NUMBER

Keeps track of all the numbers generated from a range instance number if audit is on.

BN_RANGE_GID
The range from which this number is generated.
RANGE_NUMBER
The number generated.
BN_TYPE_GID
The business number type to which the BN number belongs. This is part of a back reference from a BN number to a range number.
OBJECT_GID
The primary object for which the range number is used. For example, the shipment_gid is the object_gid when a shipment_refnum uses range number.
BN_RULE_GID
The Business Rule used to generate the range number, if any.
BN_CONTEXT
Contains the context used for the business number generation.
STATE
The state of the number. Numbers are pooled, generated, assigned, and destroyed.
