Definitions - RATE_VERSION

Includes the dates in which rates are valid. The RATE_OFFERING table references the RATE_VERSION table. This does not appear to be associated with a specific carrier, which means that if all carrier"s rates changed on the day, that you would only need one rate_version entry.

RATE_VERSION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
RATE_VERSION_XID
GC3 Unique identifier for the entity. Is used for creating the GID
