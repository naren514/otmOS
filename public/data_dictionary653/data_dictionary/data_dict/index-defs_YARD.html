Definitions - YARD

Stores detailed information of a slot on a particular row of the yard on a location. Information includes object GID,object type, start time and end time.

YARD_GID
The unique identifier.
LOCATION_GID
The Location Identifier.
ROW_NUMBER
The specific row number of the yard that will be occupied.
SLOT_NUMBER
The specific slot number of the row on the yard that will be occupied.
OBJECT_GID
Specify the object ID that will be occupied at a particular slot of a row in the yard. The OBJECT_GID can be null if the slot is blocked.
DATA_QUERY_TYPE_GID
Specify the object type.
START_TIME
The start time that the slot is occupied.
END_TIME
The end time that the slot is occupied.
