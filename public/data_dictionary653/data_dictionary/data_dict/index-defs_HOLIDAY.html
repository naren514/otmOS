Definitions - HOLIDAY

This table contains the holiday information for the company. This is linked to the Calendar table to define the holiday profiles.

HOLIDAY_GID
HOLIDAY_GID is the unique identifier for a holiday.
HOLIDAY_XID
HOLIDAY_XID is the unique external identifier for a holiday.
HOLIDAY_NAME
HOLIDAY_NAME contains text to describe the name of the holiday.
START_DATE
START_DATE contains the effective date for the holiday.
END_DATE
END_DATE contains the ending date of the holiday.
