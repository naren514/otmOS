Definitions - LINE_APP_TOL_SHIPMENT_REFNUM

Shipment reference numbers that, if present, indicate this rule is valid to test

LINE_APPROVE_TOLERANCE_GID
For the shipment reference numbers specified, this is the rule they are attached to
REFNUM_SEQUENCE
Sequence for uniqueness of multiple qual/value combinations
SHIPMENT_REFNUM_QUAL_GID
If specified with a corrsponding value, the qual-value combination must exist on the shipment in order for the rule to be tested. If no value is specified, only the qualifier must exist on the shipment.
SHIPMENT_REFNUM_VALUE
Optional shipment refnum value; if specified, the value and its corresponding qualifier must exist on the shipment in order to test the rule
