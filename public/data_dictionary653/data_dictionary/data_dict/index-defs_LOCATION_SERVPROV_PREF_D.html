Definitions - LOCATION_SERVPROV_PREF_D

The table is used to store detailed information of time window preferences for appointment on a location for a service provider.

LOCATION_SERVPROV_PREF_GID
The Location Servprov Preference GID
LOCATION_SERVPROV_PREF_D_SEQ
sequence with in a particular location servprov pref gid
DAY_OF_WEEK
The day of week.
BEGIN
The begin time.
PREFERENCE_LEVEL
The preference level.
DURATION
The duration of the time window.
MAX_SLOT
The maximum number of slots during this time.
LOCATION_RESOURCE_GID
Indicates the resource that preference defined on.
IS_STANDING
Indicates if the defined preference is for standing appointment.
