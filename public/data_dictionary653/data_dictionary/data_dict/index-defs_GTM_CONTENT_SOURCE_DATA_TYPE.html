Definitions - GTM_CONTENT_SOURCE_DATA_TYPE

Specifies the mapping from the gtm content data type to the content source provider data type.

GTM_CONTENT_SOURCE_GID
Specifies the unique identifier for the GTM Content Source.
SOURCE_DATA_TYPE
Specifies the content source data type identifier.
GTM_CONTENT_TYPE_GID
Specifies the gtm content type identifier.
IS_SUPPORTED
Specifies if the content source data type is supported in GTM for data retrieval.
