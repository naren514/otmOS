Definitions - ACTION

This table holds the list of all possible user interface actions that can be run in GC3.

DISPLAY_IGNORE_CRITERIA
If 'Y', UI will display "Ignore Criteria" screen.
IS_LONG_RUNNING_TASK
If Y, this action is a long running task and should be monitored in the LONG_RUNNING_TASK table.
LRT_INTERRUPTABILITY
Interruptability support: NONE = task cannot be interrupted; INTERRUPTIBLE = task can be interrupted, likely ending in a failure state; RESUBMITTABLE: task can be resubmitted into it's origin queue for handling by another machine
LRT_INTERRUPT_ON_SQL
If 'Y', a task interruption will force the next SQL statement to fail.
LRT_OBJECT_COUNT_THRESHOLD
The action is only tracked as a Long Running Task if IS_LONG_RUNNING_TASK='Y' and the # of objects passed to the UI action exceeds this threshold.
