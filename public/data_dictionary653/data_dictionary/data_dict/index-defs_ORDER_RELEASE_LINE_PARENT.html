Definitions - ORDER_RELEASE_LINE_PARENT

This table defines the recursive relationship between the order releases. A line order release can be part of another order release. For example 200 Socks order line, 200 Nike shoes order line can be parts of a order line of 200 sports shoes order.

PARENT_ORL_GID
A unique identifier to define the multiple order release lines per order release. The gid has the domain portion.
CHILD_ORL_GID
A unique identifier to define the multiple order release lines per order release. The gid has the domain portion.
