Definitions - ALLOC_TARGET

For identifying the records to which we can assign the allocations. For example Order Relase, Order Line

ALLOC_TARGET_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ALLOC_TARGET_XID
GC3 Unique identifier for the entity. Is used for creating the GID. E.g. OrderRelease, OrderReleaseLine
CLASS_NAME
The Specific java class name that represents and performs operations for this object.
