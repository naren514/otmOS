Definitions - CR_FORECAST

Forecasted volume for a lane and day/shift.

CR_AGGREGATION_GID
The aggregation in which this forecast was created. (Null for user-created forecast)
VOLUME
The forecasted volume (in number of shipments)
CONFIDENCE_FACTOR
Percent representing confidence in the forecasted volume.
START_DAY
Start day in cycle.
START_DOW
Start day-of-week (Sunday=1)
START_TIME
Number of seconds from midnight
END_DAY
End day in cycle.
END_DOW
End day-of-week (Sunday = 1)
END_TIME
Number of seconds from midnight
EQUIPMENT_GROUP_PROFILE_GID
Profile for equipments that can service shipments from this forecast.
MEAN_VOLUME
Average volume among the non-outlier aggregates for this forecast.
MEDIAN_VOLUME
Median volume among the non-outlier aggregates for this forecast.
MAX_VOLUME
Max volume among the non-outlier aggregates for this forecast.
MIN_VOLUME
Min volume among the non-outlier aggregates for this forecast.
STANDARD_DEVIATION
Standard deviation for the non-outlier aggregates for this forecast.
IS_USER_EDITED
Indicates if the forecast has been edited/created by the user.
IS_ACTIVE_FOR_CR_SOLVE
Indicates whether this forecast should be brought into the CR solve.
