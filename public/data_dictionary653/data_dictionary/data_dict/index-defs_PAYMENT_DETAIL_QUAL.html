Definitions - PAYMENT_DETAIL_QUAL

This table stores the payment qualifiers attached to a payment. E.g., Prepay, Pay On Delivery etc.

PAYMENT_DETAIL_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PAYMENT_DETAIL_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
