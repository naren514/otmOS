Definitions - OR_EQUIPMENT

Defines equipments associated with an order release that the user has supplied.

ORDER_RELEASE_GID
The order release that the equipment is being associated with.
OR_EQUIPMENT_SEQ
A one-up number to represent each new equipment_group for the order release.
EQUIPMENT_GROUP_GID
The equipment related to the order release.
EQUIPMENT_TYPE_GID
Can be optionally entered.
