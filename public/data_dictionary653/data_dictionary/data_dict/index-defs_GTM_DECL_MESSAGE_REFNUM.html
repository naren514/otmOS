Definitions - GTM_DECL_MESSAGE_REFNUM

This Table maintains the Reference number for the messages exchanged between the different parties involved in the process of Declaration.

GTM_DECL_MESSAGE_GID
GTM Declaration Message GID is part of the primary key and is a foreign key to the primary key of table GTM_DECL_MESSAGE.
GTM_TRANS_REFNUM_QUAL_GID
It is Transaction Reference Number qualifier ID and is a foreign key to the primary key of GTM_TRANS_REFNUM_QUAL table.
REFNUM_VALUE
Value of reference number.
