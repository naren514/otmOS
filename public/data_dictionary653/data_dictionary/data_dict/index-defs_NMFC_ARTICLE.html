Definitions - NMFC_ARTICLE

This table defines the NMFC governmental name and number if the item is an NMFC article.

NMFC_ARTICLE_GID
NMFC_ARTICLE_GID contains the unique identifier for the National Motor Freight Classification (NMFC) article for the item being shipped.
NMFC_ARTICLE_XID
NMFC_ARTICLE_XID contains the unique external identifier for the National Motor Freight Classification (NMFC) article.
NMFC_ARTICLE_NAME
NMFC_ARTICLE_NAME contains the name of the National Motor Freight Classification (NMFC) article.
