Definitions - INVOICE_PORT

One record per port stop sequence per invoice (the invoice should relate to only one shipment). Since one trip may repeat the same port, the port itself is not part of the key. This table provides supplementary information about the involved ports related to the shipment on the invoice.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
PORT_SEQ_NO
PORT_SEQ_NO is the sequence number for the port on the invoice.
PORT_FUNCTION_CODE
Port function code.
LOCATION_IDENTIFIER
The location identifier.
LOCATION_IDENTIFIER_QUALIFIER
The qualifier associated with the location identifier.
PORT_NAME
PORT_NAME contains the name of the port on the invoice.
PROVINCE_CODE
PROVINCE_CODE contains the province code on the invoice.
COUNTRY_CODE3_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
