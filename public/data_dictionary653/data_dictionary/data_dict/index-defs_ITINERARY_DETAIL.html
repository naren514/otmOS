Definitions - ITINERARY_DETAIL

This table joins the Itinerary table to the Leg table. It is used to resolve the many-to-many relationship between Itinerary and Leg.

ITINERARY_GID
ITINERARY_GID contains the unique identifier for the itinerarty.
LEG_GID
LEG_GID contains the unique identifier for a leg on the itinerary.
