Definitions - GTM_CONTENT_SOURCE_CONFIG

Used to specify the configuration for retrieving the data from the content source provider.

GTM_CONTENT_SOURCE_GID
Specifies the unique identifier for the GTM Content Source.
SOURCE_CONFIG_ID
Specifies the Id given for the source configuration.
COM_METHOD_GID
Specifies the unique identifier for a communication method.
EXTERNAL_SYSTEM_GID
Specifies the unique identifier for an external system.
IS_ACTIVE
Specifies if the source configuration is active.
