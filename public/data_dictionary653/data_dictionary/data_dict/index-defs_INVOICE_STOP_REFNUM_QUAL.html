Definitions - INVOICE_STOP_REFNUM_QUAL

Contains the valid entries for stop reference numbers. Example: Purchase Order Number

GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
INVOICE_STOP_REFNUM_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
INVOICE_STOP_REFNUM_DESC
Reference description.
DEFAULT_REFNUM_BN_TYPE_GID
The BN type used to generate a refnum when the refnum field for a record in the corresponding refnum table is not populated.
