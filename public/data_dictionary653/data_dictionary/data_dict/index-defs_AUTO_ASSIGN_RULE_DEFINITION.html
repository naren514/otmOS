Definitions - AUTO_ASSIGN_RULE_DEFINITION

Defines the values for an actual auto-assignment that may take place

AUTO_ASSIGN_RULE_GID
Auto-assign rule id
DEFINITION_SEQUENCE
Sequence within the rule to separate each auto-assignment
USER_SEQUENCE
User defined sequence to run the assign conditions
IS_DEFAULT
This rule should be applied if none of the other conditions in the rule pass
VALUE1
1st value to auto-assign, when assigning involved parties, may be the contact id
VALUE2
2nd value to auto-assign, when assigning involved parties, may be the contact method
VALUE3
3rd value to auto-assign, when assigning involved parties, may be the involved party qualifier
