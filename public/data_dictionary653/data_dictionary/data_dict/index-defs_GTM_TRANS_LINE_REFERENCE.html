Definitions - GTM_TRANS_LINE_REFERENCE

It is used to store the declaration line id along with reference transaction lines details.

QUANTITY_VALUE
It define quantity value of transaction line that will be aggregated to the declaration line.
QUANTITY_UOM_CODE
It define the unit of measure of the value stored in the quantity_value column.
GTM_QUANTITY_TYPE_GID
It define the value of quantity type , which is a foreign to GTM_QUANTITY_TYPE table.
GTM_TRANSACTION_LINE_GID
It define the declaration line id.
TRANS_LINE_REF_GID
It define the transaction line ids from which declaration line is created.
IS_PRIMARY
Y/N primary transaction line identified by aggregation engine with be stored as Y and data is transferred from this transaction line to declaration line , as per data configuration set up
