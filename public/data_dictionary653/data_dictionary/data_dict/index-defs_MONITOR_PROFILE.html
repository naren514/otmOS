Definitions - MONITOR_PROFILE

A monitor profile can be either a Template (Monitor Profile Template) or an 'instance' (Monitor Profile) created from a template. If it is a Template, it will not have an associated business object. If it is an 'instance', it will have an associated business object.

MONITOR_PROFILE_GID
The unique identier of the monitor profile.
MONITOR_PROFILE_XID
The external identifier for the monitor profile. It is used to be part of the key for a monitor profile template or monitor profile.
MONITOR_PROFILE_NAME
Text, as entered by the user, for familiar reference.
MONITOR_PROFILE_TEMPLATE_GID
Null if this is a monitor profile template -or- the global identifier indicating the monitor profile template from which this monitor profile was created.
IS_TEMPLATE
True if this is a monitor profile template (default) -or- False if this is a monitor profile.
DATA_QUERY_TYPE_GID
The unique identifier referring to the data_query_type table to indicate which type of business object (order_base, shipment, order_release, invoice) the monitor_profile is associated. Note: a Monitor Profile must have a value in this column depending on which value this has, the corresponding column (order_base_gid, shipment_gid, order_release_gid, invoice_gid) must have a value. Thus, the type limits the type of agents you can use and events you can listen for. For example, only Shipment events for a shipment data query type.
