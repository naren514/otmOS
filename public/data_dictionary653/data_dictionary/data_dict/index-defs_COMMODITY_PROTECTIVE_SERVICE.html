Definitions - COMMODITY_PROTECTIVE_SERVICE

THIS TABLE STRUCTURE IS SIMILAR TO THE INVOICE_PROTECTIVE_SERVICE TABLE. PROTECTIVE SERVICE IS USED BY THE RAIL INDUSTRY TO COMMUNICATE TEMPERATURE REQUIREMENTS AND IS WHERE THE USER COMMUNICATES OPTIMAL TEMPERATURE FOR THE RR. THE USER CAN SPECIFY UP TO 5 CODES THAT ARE ALL USED TOGETHER, THIS IS WHY THERE IS A SEQUENCE NUMBER INCLUDED WITHIN THE STRUCTURE.

COMMODITY_GID
A unique identifier to define the commodities in the system.
PROT_SRV_SEQ_NO
THE SEQUENCE NUMBER.
PROT_SRV_CODE
THE PROTECTIVE SERVICE CODE.
PROT_SRV_RULE_CODE
THE PROTECTIVE SERVICE RULE CODE.
PROT_SRV_TEMP
THE PROTECTIVE SERVICE OPTIMAL TEMPERATURE.
PROT_SRV_TEMP_UOM_CODE
THE PROTECTIVE SERVICE TEMPERATURE UNIT OF MEASURE CODE.
PROT_SRV_TEMP_BASE
The protective service optimal temperature in base units.
