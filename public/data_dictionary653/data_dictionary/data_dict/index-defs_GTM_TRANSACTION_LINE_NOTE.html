Definitions - GTM_TRANSACTION_LINE_NOTE

This table is intended to hold the notes that gives information about the result of the actions/events performed on a transaction line.

GTM_TRANSACTION_LINE_GID
Transaction Line ID on which the action is executed.
NOTE_SEQUENCE_NO
Sequence number of notes
TIMESTAMP
Timestamp of the event that is noted.
SUMMARY
High level description about the event.
NOTE
Detailed description about the event.
ENTERED_BY
It takes the action name or the manual intervention that resulted in this note
IS_SYSTEM_GENERATED
Says if the note is generated by the system or manually added by the user.
