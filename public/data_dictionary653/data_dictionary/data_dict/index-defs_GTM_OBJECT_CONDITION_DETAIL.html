Definitions - GTM_OBJECT_CONDITION_DETAIL

To capture condition details like whether it is single/combination/sub-selection condition type and on which object with attribute and parameter details. Example : Simple Condition like "transcation transport mode in air or ocean". Here transaction is the object.Transport mode is attribute ands air, ocean are parameter values.

GTM_OBJECT_CONDITION_GID
Gid.
GTM_OBJECT_CONDITION_SEQ
Condition Sequence.
GTM_OBJECT_CONDITION_TYPE
Condition Type (Simple / Combination / Sub-Select)
GTM_OBJECT_TYPE_GID
Object Type Gid.
GTM_OBJECT_TYPE_ATTRIBUTE_GID
Attribute of Object Type Selected like Transport mode
GTM_OBJ_CONDITION_PARAMETERS
Condition Parameters ( 'Air', 'OCEAN' are parameters for condition on Transport Mode)
