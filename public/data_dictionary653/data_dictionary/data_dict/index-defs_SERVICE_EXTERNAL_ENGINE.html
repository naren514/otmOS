Definitions - SERVICE_EXTERNAL_ENGINE

The SERVICE_EXTERNAL_ENGINE table holds data relevant to third party external engines, including which GC3 class handles that engine. Each entry in the SERVICE_EXTERNAL_ENGINE table refers to an external engine along with (via the SERVICE_EXTERNAL_ENGINE_PARM table) a set of parameters to be passed to that engine in order to calculate service time. .

SERVICE_EXTERNAL_ENGINE_GID
The global identifier for a SERVICE_EXTERNAL_ENGINE. A SERVICE_EXTERNAL_ENGINE refers to a third-party product that calculates service times.
SERVICE_EXTERNAL_ENGINE_XID
The external identifier for a SERVICE_EXTERNAL_ENGINE.
SERVICE_EXTERNAL_ENGINE_DESC
A description of the service_external_engine.
JAVA_CLASS
Name of java class that implements the external engine.
TIME_TOLERANCE
If engine is time sensitive, how close must two start times be to be considered 'close enough'? NULL if engine is not time sensitive.
TIME_TOLERANCE_UOM_CODE
Unit of measure for TIME_TOLERANCE column.
TIME_TOLERANCE_BASE
TIME_TOLERANCE in base units.
