Definitions - SHIPMENT_COST

This table stores the breakdown of the cost associated with the accessorials in the shipment.

SHIPMENT_COST_SEQNO
A system generated unique sequence number to define the shipment costs.
SHIPMENT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COST_TYPE
C, A, B
COST
The cost as it releates to the cost type for the shipment.
COST_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COST_BASE
COST in base units.
ACCESSORIAL_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
BILLABLE_INDICATOR_GID
indicates the billable status of this shipment cost (sell shipment only)
IS_COST_FIXED
Indicates shipment cost is a fixed cost.
PROCESS_AS_FLOW_THRU
Override for record to be copied to the sell side during rating.
ADJUSTMENT_REASON_GID
Reason why this cost was added.
GENERAL_LEDGER_GID
GL code for the shipment cost record.
PAYMENT_METHOD_CODE_GID
Payment method code for the shipment cost
EXCHANGE_RATE_DATE
Indicates the date to be used for any currency conversions; if null, the latest conversion should be used.
EXCHANGE_RATE_GID
Indicates the set of exchange rates to use for currency conversions; if null, the default set should be used
COST_FN
functional currency amount for cost
S_SHIP_UNIT_GID
Points to the ship unit (if any) associated with this shipment_cost, as a package
S_SHIP_UNIT_LINE_NO
Points to the ship unit line (if any) associated with this shipment_cost, as a package
IS_WEIGHTED
Whether or not the cost affects the total weighted cost of the shipment only
TRACKING_NUMBER
Denotes the tracking number associated with a specific charge/cost.
