Definitions - AUTO_ASSIGN_RULE_CRITERIA

User defined criteria which will be used to determine what information to assign. For example, to determine the shipment involved party, look at the shipment mode, and the service provider.

AUTO_ASSIGN_RULE_GID
Auto-assign rule id
AUTO_ASSIGN_CRITERIA_GID
Criteria id that will be used by auto assign logic for this rule
AUTO_ASSIGN_TYPE_GID
Single component of the criteria used to choose what to auto-assign
