Definitions - HAZMAT_GENERIC

THIS TABLE STORES THE HAZARDOUS MATERIALS INFORMATION THAT IS PROPER SHIPPING NAME CENTRIC AND THUS MORE GENERIC IN NATURE. ALL THE TABLE ATTRIBUTES ARE RELATED TO THE PROPER SHIPPING NAME. THIS TABLE ALONG WITH THE HAZMAT_ITEM TABLE AND OTHER ASSOCIATED TABLES MAKE UP THE HAZARDOUS MATERIALS DATABASE. THIS TABLE IS PATTERNED AFTER THE DOT 101 TABLE.

HAZMAT_GENERIC_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
HAZMAT_GENERIC_XID
GC3 Unique identifier for the entity. Is used for creating the GID
PROPER_SHIPPING_NAME
THE PROPER SHIPPING NAME USED FOR AN ITEM.
IDENTIFICATION_NUMBER
HAZARDOUS CARGO CLASSIFICATION NUMBER.
PACKAGING_GROUP
THE PACKAGE GROUP.
SUBSIDIARY_HAZARD
SIMILAR TO HAZARD_CLASS.
IMDG_EMS_NUMBER
INTERNATIONAL MARITIME DANGEROUS GOODS EMERGENCY RESPONSE NUMBER.
