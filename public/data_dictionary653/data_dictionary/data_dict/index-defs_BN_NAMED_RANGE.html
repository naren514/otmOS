Definitions - BN_NAMED_RANGE

Each record represents a collection of pre-assigned ranges.

BN_NAMED_RANGE_GID
The global id of the named range (or range set).
BN_NAMED_RANGE_XID
The external id of the named range.
RECYCLING_POLICY
Determines whether to recycle or audit the range.
USE_BN_CONTEXT
Signifies if the BN_CONTEXT is used in the business number generation.
