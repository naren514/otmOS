Definitions - GTM_TR_SANCTION

This table captures information related to Sanction Controls and Authorization details.

GTM_TR_SANCTION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
GTM_COMPLIANCE_RULE_GROUP_GID
Compliance rule group gid used in sanction screening.
GTM_COMPLIANCE_RULE_GID
Compliance rule gid, based on which sanction controls added.
GTM_CONTROL_TYPE_GID
Captures unique Control Type
CONTROL_CODE
Captures Control Code
GTM_TRANSACTION_LINE_GID
Transaction line gid is a unique gid on which sanction controls applied.
GTM_TR_SANCTION_XID
User entered unique identifier withoout domain.
AUTHORIZATION_TEXT
Authorization Text captures text message to Authorize a transaction line (override of the control).
AUTHORIZATION_ADDED_BY
Authorization added by captures who authorized this transaction line.
AUTHORIZATION_ADDED_ON
Authorization added on captures a date when this authorization happend.
