Definitions - DISTANCE_EXTERNAL_ENGINE

This table defines any external engine that may be used for finding the distance

DISTANCE_EXTERNAL_ENGINE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
DISTANCE_EXTERNAL_ENGINE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
CACHE_CONTROL_TYPE
Controls whether distances should be saved as GC3 lanes. Y= Cache and lookup values from cache; L= Lookup values from cache but do not save them; C=Cache values but do not use the cache for lookup; N=Cache off. Do not save value and do not look them up.
COUNTY_OVERRIDE_GID
County override if county is being used in EDE lookup and requires a translation from OTM to EDE
EXTERNAL_SYSTEM_GID
This flag would specify the external system that needs to be used for rating.
CONNECTION_TYPE
Defines the type of transport mechanism used. In this case, J = javaclass, W= web service which implements REXPublicExternalInterface.java Other mechanisms may be supported in future iterations.
