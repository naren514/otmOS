Definitions - BULK_TRAILER_BUILD

Provides information about what shipments were selected and what shipment groups were created during a given run of bulk trailer build.

BULK_TRAILER_BUILD_GID
Unique identifier for the bulk trailer build process.
BULK_TRAILER_BUILD_XID
External identifier for the bulk trailer build process.
QUERY_NAME
Bulk trailer build input parameter.
START_TIME
Start time of the bulk trailer build.
END_TIME
End time of the bulk trailer build.
NUM_SHIPMENTS_SELECTED
Number of shipments selected for the bulk trailer build.
NUM_SHIPMENTS_BUILT_TO_GROUPS
Number of shipments successfully built into shipment groups by the bulk trailer build.
NUM_SHIPMENTS_NOT_IN_GROUPS
Number of shipments not built into any of the shipment groups by the bulk trailer build.
NUM_SHIPMENT_GROUPS_BUILT
Number of shipment groups generated by the bulk trailer build process.
LOG_PROCESS_ID
The log id for the bulk trailer build process.
APP_MACHINE_GID
Application machine id the process is running on, in order to get the log from the correct machine.
