Definitions - COMPARTMENT_PI_JOIN

This table associates the equipment compartments with the packaged items. In GC3 users have the flexibility of defining the compatible Packaged items for a compartment.

EQUIPMENT_GROUP_GID
Equipment Group Id for the compartment the packaged Item is linked to
COMPARTMENT_NUM
Compartment Number for the compartment the packaged Item is linked to
PACKAGED_ITEM_GID
Packaged Item that is linked to the compartment
