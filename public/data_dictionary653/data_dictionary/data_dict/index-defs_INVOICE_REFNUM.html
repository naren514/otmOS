Definitions - INVOICE_REFNUM

Can be used to store qualified reference information about an invoice. The qualifier/description of the information is stored in the table INVOICE_REFNUM_QUAL. Example of a reference qualifier would be BILL OF LADING.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
INVOICE_REFNUM_QUAL_GID
INVOICE_REFNUM_QUAL_ GID contains the unique identifier for the reference number (PONUMBER, VATNUMBER, CUSTPONUMBER).
ISSUE_DATE
ISSUE_DATE contains the date the invoice was issued.
