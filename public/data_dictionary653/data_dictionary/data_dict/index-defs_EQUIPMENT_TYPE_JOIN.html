Definitions - EQUIPMENT_TYPE_JOIN

This table associates the equipment types with the equipment groups. In GC3 users have the flexibility of defining the equipments as types or groups. I both are defined, this table can be used to join them.

EQUIPMENT_TYPE_GID
EQUIPMENT_TYPE_GID is the unique identifier for the equipment type.
EQUIPMENT_GROUP_GID
EQUIPMENT_GROUP_GID is the unique identifier for the equipment group.
