Definitions - OUT_XML_PROFILE

This table and OUT_XMl_PROFILE_D table together specify the outbound xml profiles that can be used to limit the size of xml documents pushed out from gc3 to external systems. The Out_xml_profile table contains the header record for the profile and defines the "gid" for that profile. The out_xml_profile_d table provides the details which comprise a given profile. Each record in the out_xml_profile_d table indicates a specific "chunk" of xml that should be excluded from the outbound document.

OUT_XML_PROFILE_GID
Globally unique identifier for an outbound xml profile.
OUT_XML_PROFILE_XID
Domain unique identifier for an outbound xml profile.
DEFAULT_MODE
Specifies whether the default mode for generating XML should be MIN or MAX. When not specified, the property is the default.
SHOULD_VALIDATE
Specifies if the XML produced should conform to the schema.
XML_TEMPLATE_GID
XML template ID
USE_TEMPLATE
If it equals "Y", integration will use template to build the XML
