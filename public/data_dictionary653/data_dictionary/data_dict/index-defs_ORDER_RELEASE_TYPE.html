Definitions - ORDER_RELEASE_TYPE

The types of orders defined by the shipper. For e.g., Customer Order, Manufacturer Order, Purchase Order, Distribution Order etc.

ORDER_RELEASE_TYPE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ORDER_RELEASE_TYPE_XID
GC3 Unique identifier for the entity. Is used for creating the GID
