Definitions - CURRENCY_EXCHANGE_RATE

This table is used for the currency conversion in GC3. It stores the conversion rates for every country with effective dates. It is used along with the Currency table to calculate the currency conversion rates.

EXCHANGE_RATE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
EFFECTIVE_DATE
EFFECTIVE_DATE contains the date on which the currency exchange rate is applicable.
FROM_CURRENCY_GID
FROM_CURRENCY_GID contains the unique identifier for the currency to be calculated into a different currency.
TO_CURRENCY_GID
TO_CURRENCY_GID contains the unique identifier for the currency into which the original currency was calculated.
TRIANGULATION_CURRENCY_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
EXCHANGE_RATE
EXCHANGE_RATE contains the actual exchange rate.
