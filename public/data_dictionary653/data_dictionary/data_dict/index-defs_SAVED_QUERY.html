Definitions - SAVED_QUERY

SAVED_QUERY_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SAVED_QUERY_XID
GC3 Unique identifier for the entity. Is used for creating the GID
USE_IN_FINDER
Identify if the saved query can be displayed in the finder or saved query screen. It is used to support direct SQL for agent saved condition. N means finder should not display the query, and saved query manager screen should display it as pure sql statement.
IS_HIDDEN_FOR_MILESTONES
If "Y", this saved query is used by milestone monitors and should not be displayed in finder result page.
