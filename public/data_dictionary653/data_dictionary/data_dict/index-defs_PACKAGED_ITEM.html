Definitions - PACKAGED_ITEM

PACKAGED_ITEM_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PACKAGED_ITEM_XID
GC3 Unique identifier for the entity. Is used for creating the GID
ITEM_GID
A unique identifier to define all teh items in teh system. The gid contains the domain portion.
PACKAGE_SHIP_UNIT_WEIGHT_BASE
PACKAGE_SHIP_UNIT_WEIGHT in base units.
CORE_DIAMETER
For cylindrical objects, this defines the diameter of the core
CORE_DIAMETER_UOM_CODE
For cylindrical objects, unit of measure for the diameter of the core
CORE_DIAMETER_BASE
For cylindrical objects, the base value of the diameter of the core
BB_HANDLING_TYPE
Indicates the Batch Balancer handling type for the packaged item. Valid handling types are conveyable, non-conveyable, and repack. Batch Balancer logic uses this info to determine potential batches for order release line assignment in the event an order release needs to be split across multipe batches.
PACKAGED_ITEM_TYPE_GID
Type of packaged item.
LOAD_CONFIG_RULE_RANK
During 3D loading of mixed items, the PI with the highest rank will be used to find the matching stackability rule and load config rule.
THU_PROFILE_GID
Defines all valid THUs the PI can be packed into.
