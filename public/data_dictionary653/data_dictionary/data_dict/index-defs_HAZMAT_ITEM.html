Definitions - HAZMAT_ITEM

THIS TABLE STORES THE HAZARDOUS MATERIALS INFORMATION THAT IS ITEM CENTRIC. ALL THE TABLE ATTRIBUTES ARE RELATED TO AN ITEM. THIS TABLE ALONG WITH THE HAZMAT_GENERIC TABLE AND OTHER ASSOCIATED TABLES MAKE UP THE HAZARDOUS MATERIALS DATABASE.

HAZMAT_ITEM_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
HAZMAT_ITEM_XID
GC3 Unique identifier for the entity. Is used for creating the GID
REGION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
HAZMODE_GID
Reference to the transportaion mode. I.e.) AIR, TRUCK, RAIL etc.
HAZMAT_GENERIC_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
HAZMAT_INDICATOR
Y/N, INDICATES IF HAZARDOUS MATERIAL.
HOT_INDICATOR
Y/N, INDICATES HOT TEMPERATURE.
RESIDUE_INDICATOR
Y/N, INDICATES LEFT OVER PRODUCT.
DOT_EXEMPTION_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
ADDITIONAL_TRANSPORT_MSG_GID_1
Reference to additional transport message instruction.
ADDITIONAL_TRANSPORT_MSG_GID_2
Reference to additional transport message instruction.
STCC_49_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COMPETENT_AUTHORITY_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
IS_ALL_PACKED
Flag to indicate the pkg is 'All packed in one'
AUTHORIZATION
The authorization for the Dangerous Goods shipment.The authorization field is only required to state special governmental authorizations that are required because of the provisions or packing materials shipped with.
CHEMICAL_FORMULA
The basic chemical description of the substance
CONCENTRATION_PERCENT
Required if the Dangerous Goods UN Number is 2031, 1796, 1826, or 1873. Enter the percentage of DG concentration if the above condition applies.
CRIT_SAFETY_INDEX
A measurement for packages containing fissile material that is determined by dividing the number 50 by the smaller of the numbers determined using the rules in section 10.6.2.4.3 of the DG Regulation Guide
IDG_ADD_DESC_INFO
Hazmat IDG additional description information
MAX_QUANTITY
Maximum passenger volume or the Maximum cargo volume for the item (Used in Q- Value calculation)
OUTER_PACKAGING_TYPE
Outer packaging type if packing type = All packed in one
OUTER_PACKING_COUNT
Outer Packing count
IS_OVERPACK
Flag to indicate the pkg is an 'overpack'
PACKAGE_TYPE
Hazmat Package type of the Dangerous Goods shipment.
PACKING_COUNT
Packing Count (number of packaged units)
PACKING_INSTRUCTIONS
Required for dangerous goods shipments, unless the DG class is radioactive or Not Restricted.See Table 4.2 columns G, I, or K of the IATA DG Regulations.
PHYSICAL_FORM
A description of the physical form of the radioactive substance : S(p)ecial (S)olid (L)iquid (G)as
Q_VALUE
Ratio used to indicate that the combined quantities in 'all packed in one' package are within the maximum allowable limits.
HAZ_QUANTITY
Hazmat item quantity
RADIOACTIVE_LABEL
The type of label on the shipment. W1 (White I) Y2 (Yellow II) Y3 (Yellow III)
RAD_PACKAGING
A description of the type of packaging the substance is contained in. Acceptable package types are listed in section 10.5 of the DG Regulations Guide: Expected ( Expected ) Industrial ip1 ( Industrial Type 1(IP-1) ) Industrial ip2 ( Industrial Type 2(IP-2) ) Industrial ip3 ( Industrial Type 3(IP-3) ) Type a ( Type A ) Type b(m) ( Type B(M) ) Type b(u) ( Type B(U) ) Type c ( Type C )
RADIOACTIVITY
A measurement of the activity of the shipment. For fissile material, the weight of the material in grams or kilograms may be substituted for the activity measurement
RADIOACTIVITY_UNITS
The units that the activity were mesaured in. This must be in becquerel units (or multiples thereof) : Bq (becquerel) kBq (kilobecquerel) MBq (megabecquerel) GBq (gigabecquerel) TBq (terabecquerel) PBq (petabecquerel)
RADIONUCLIDE
The radionuclide assigned to the substance being shipped. A table of basic radionuclides is available in the DG Regulation Guide section 10.4.A. For example, Aluminium (13) has a radionuclide Al-26
SUBSTANCE_NUMBER
Hazardous Materials substance number.
SURFACE_READING
The surface reading of the radiation exposure for the package. This reading must be measured in mrem/h
TRANSPORT_INDEX
A number based on the radiation exposure of a package. This is determined based on the maximum measurement of the radiation in mrem/h at a distance of 1 meter from the package surface
MATERIALS_TYPE
Hazardous Materials Type. Valid values are: l = Limited Quantities e = 500 KG Exemption f = Fully Regulated (Purolater)
UNITS
IATA units for Non UPS shipments (L=>Liters g=>Grams kg=>Kilograms kg G=>Kilograms Gross).
EXTERNAL_CODE
Item Hazmat code as configured in the TNT EM. (For carrier TNT)

Store a user-defined icon that will apear within the user interface to help visually identify attributes for the item

Store a user-defined icon that will apear within the user interface to help visually identify attributes for the item
