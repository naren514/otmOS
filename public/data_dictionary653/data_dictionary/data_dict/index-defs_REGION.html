Definitions - REGION

This table contains the user defined regions. The details of the regions are in the REGION_DETAIL table.

REGION_GID
REGION_GID contains the unique identifier for the region for the lane.
REGION_XID
REGION_XID contains the unique external identifier for the region for the lane.
REGION_NAME
REGION_NAME contains the name for the lane's region.
REPRESENTATIVE_LOCATION_GID
Location that can be used to stand in for the region when location data is needed.
