Definitions - ASK_OTM_SAVED_QUERY

SAVED_QUERY_GID
Gid for the Saved Query. The gid for the migrated data will contain the user id and query name combination.
SAVED_QUERY_XID
Xid for the Saved Query.
QUERY_NAME
This is the name of the query class
SAVED_QUERY_XML
It can store XML/HTML for content for the saved query. For the migrated data this field can be formed by combining 2 -3 records.
