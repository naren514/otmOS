Definitions - DM_TRANSACTION_EVENT

It captures all events and their corresponding event date for a particular demurrage transaction. for example a demurrage transaction (load to empty) of a equipment movement at a location, there could be multipe events like arrival, order out, released, pulled events.

DM_TRANSACTION_SEQ_NO
DM_TRANSACTION_SEQ_NO contains the unique identifier for the demurrage transaction.
BS_STATUS_CODE_GID
An external event ID. Within OTM external events are referred by status_code_gid.
EVENT_DATE
Date and Time of the event.
