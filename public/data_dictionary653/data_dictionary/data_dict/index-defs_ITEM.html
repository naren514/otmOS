Definitions - ITEM

This is the item master table. It defines the various items in the customer"s database. The item can be defined in various governmental classification categories. GC3 includes the categories to represent NMFC, STCC, HTS and SITC. Other details like, minimum, maximum shipping units, pallet size etc. are also defined in this table.

ITEM_GID
ITEM_GID contains the unique identifier for the item.
ITEM_XID
ITEM_XID contains the unique external identifier for the item.
ITEM_NAME
ITEM_NAME contains text for the name of the item.
COMMODITY_GID
COMMODITY_GID contains the unique identifier for the commodity.
NMFC_ARTICLE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
NMFC_CLASS_GID
This field allows the user to define the item's NMFC code.
STCC_GID
STCC_GID contains the unique identifier for the Standard Commodity Code (STCC).
HTS_GID
HTS_GID contains the unique identifier for the Harmonized Tariff System (HTS).
SITC_GID
SITC_GID contains the unique identifier for the Standard International Trade Classification (SITC).
DESCRIPTION
ITEM contains a description of the item being shipped.
USER_DEFINED_COMMODITY_GID
USER_DEFINED_COMMODITY_GID contains the unique identifier for the user-defined commodity.
UDC_CLASSIFICATION_LIST_GID
List of User Defined commodities for item.
MARKED_FOR_PURGE
A boolean field. When it is set to "Y", this row and its child data will be purged.
PRIORITY
This column defines the priority associated with an Item
COUNTRY_OF_ORIGIN_GID
Specifies the country of origin for the item.
