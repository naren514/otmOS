Definitions - LEG_FLEX_COMMOD_PLIST

This table is used to specify that a given leg can (or cannot) provide service for a given set of commodities.

LEG_GID
LEG_GID contains the unique identifier for the leg.
FLEX_COMMODITY_PROFILE_GID
FLEX_COMMODITY_PROFILE_GID contains the unique identifier for the flex commodity profile for the leg.
