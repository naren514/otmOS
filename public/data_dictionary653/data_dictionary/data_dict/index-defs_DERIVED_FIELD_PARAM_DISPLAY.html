Definitions - DERIVED_FIELD_PARAM_DISPLAY

Defines how the derived field parameter is configured for display.

DERIVED_DATA_OP_IMPL_GID
Contains the unique identifier for the derived data operations implementation.
FIELD_NAME
Field name
PARAMETER_NAME
Parameter name
IS_REQUIRED
Field is required. For future use.
IS_DROP_LIST
Field is a drop list.
IS_MULTIPART_DROP_LIST
Field is a multi-part drop list.
IS_DROP_LIST_WITH_DESCRIPTION
Field is a drop list displaying description.
ALLOW_NEGATIVE
Allow negative numbers
ALLOW_NEW
Allow new button. For pick lists.
ALLOW_LIST
Allow list button. For pick lists.
ALLOW_VIEW
Allow view button. For pick lists and drop lists.
ALLOW_FIND
Allow find button. For pick lists.
MIN_VALUE
Minimum number value.
MAX_VALUE
Maximum number value.
VALID_VALUE_TYPE_GID
Contains the unique identifier for the valid value type. Used to create a fixed drop list.
DEFAULT_OPERATOR
Default query operator.
SHOW_DAYS
Show days. For durations.
SHOW_HOURS
Show hours. For durations.
SHOW_MINUTES
Show minutes. For durations.
QUERY_NAME
Alternate query name.
ALLOW_MIXED_CASE
Allow mixed case strings.
