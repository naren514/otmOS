Definitions - DRIVER_CALENDAR

This table captures the Driver work profile definition, that includes the Activity calendar and the Home location.

DRIVER_GID
The Driver GID in OTM
HOME_LOCATION_GID
The Home location of the Driver defined for this work profile
ACTIVITY_CALENDAR_GID
The Activity Calendar of the Driver defined for this work profile
EFFECTIVE_DATE
The effective date for this Driver work profile
EXPIRY_DATE
The expiry date for this Driver work profile
