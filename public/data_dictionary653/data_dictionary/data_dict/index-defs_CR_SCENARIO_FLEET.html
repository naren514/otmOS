Definitions - CR_SCENARIO_FLEET

Table linking a CR Scenario with the fleets to be used for that scenario.

NUM_RESOURCES
Number of resources available for this scenario.
NUM_RESOURCES_USED
Number of resources used on CR Routes for this scenario.
NUM_RESOURCES_COMMITTED
Number of resources used on committed CR Routes for this scenario.
AVAILABLE_START_DAY
Indicates which day begins the period during which the fleet is available.
AVAILABLE_TIME
Indicates the start time-of-day for the available start day (number of seconds from midnight).
AVAILABLE_DURATION
The length of the period during which the fleet is available.
DAILY_START_TIME
The daily start time-of-day (number of seconds from midnight)
DAILY_END_TIME
The daily end time-of-day (number of seconds from midnight)
DEPOT_RETURN_INTV
Depot Return Interval: Duration after which the fleet resource needs to revisit the start depot.
DEPOT_TURN_TIME
Depot Turnaround Time: Duration that the fleet resource needs to wait after revisiting the start depot.
