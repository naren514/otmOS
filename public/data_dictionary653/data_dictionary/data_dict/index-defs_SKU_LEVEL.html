Definitions - SKU_LEVEL

SKU stock levels.

SKU_GID
SKU id.
SKU_QUANTITY_TYPE_GID
The sku_quantity_type to which this level applies.
SKU_LEVEL_TYPE_GID
SKU inventory alert level type.
SKU_LEVEL_VALUE
SKU inventory alert level. For example, when the inventory fall below the minimum level, a request can be sent out for replenishment.
