Definitions - APPT_RULE_SET

This table used to add rules for appointments.It is used for both location resource and location resource group.

APPT_RULE_SET_GID
The unique identifier.
APPT_RULE_SET_XID
The ID of the Appointment Rule Set.
APPT_RULE_SET_NAME
Name of rule set.
EFFECTIVE_DATE
Effective date of rule set.
EXPIRY_DATE
Expiration date of rule set.
FLEX_COMMODITY_PROFILE_GID
The flex commodity profile ID that associates to the rule set.
EQUIPMENT_GROUP_PROFILE_GID
The equipment group profile ID that associates to the rule set.
MODE_PROFILE_GID
The mode profile ID that associates to the appointment rule set.
SERVPROV_PROFILE_GID
The service provider profile ID that associates to the rule set.
CONTACT_PROFILE_GID
The contact profile ID that associates to the rule set.
FLEX_COMMODITY_CHECK_OPTION
Flex Commodify check option, such as CHECK ALL, CHECK BY LARGEST WEIGHT and CHECK BY LARGEST VOLUME.
SOURCE_REGION_GID
The source region ID that associates to the rule set.
DESTINATION_REGION_GID
The destination region ID that associates to the rule set.
APPT_SS_ACTIVITY_PROF_GID
The appointment Shipment shot activity profile ID that associates to the rule set.
APPOINTMENT_ACTIVITY_TYPE
Apointment activity Type. Drop/Live
