Definitions - LOCATION_GROUP

Multiple locations can be grouped into a location group. This table stores the information about the groups.

LOCATION_GROUP_GID
LOCATION_GROUP_GID contains the unique identifier for the location group.
LOCATION_GROUP_XID
LOCATION_GROUP_XID contains the unique external identifier for the location group.
LOCATION_GROUP_NAME
LOCATION_GROUP_NAME contains the name for the location group.
