Definitions - COMMERCIAL_INVOICE

EXCHANGE_RATE_DATE
Indicates the date to be used for any currency conversions; if null, the latest conversion should be used.
EXCHANGE_RATE_GID
Indicates the set of exchange rates to use for currency conversions; if null, the default set should be used; this only applies if IS_OVERRIDE_EXCHANGE_RATE is 'N' (false)
EXCHANGE_FROM_CURRENCY_GID
When a conversion rate specific to this commercial invoice is specified (EXCHANGE_RATE field), this field should be populated with the FROM currency gid for that exchange rate
IS_OVERRIDE_EXCHANGE_RATE
If true, indicates a specific conversion rate for this commercial invoice is entered in the EXCHANGE_RATE field and should be used
COMMERCIAL_INVOICE_NUMBER
This column will be set by BN Generator.
PRODUCT_AMOUNT_FN
functional currency amount for product_amount
OTHER_CHARGE_FN
functional currency amount for other_charge_amount
TOTAL_INVOICE_FN
functional currency amount for total_invoice_amount
