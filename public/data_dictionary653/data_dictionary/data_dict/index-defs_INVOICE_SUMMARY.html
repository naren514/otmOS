Definitions - INVOICE_SUMMARY

INVOICE_SUMMARY contains the summary of the INVOICE_LINEITEM and INVOICE_SUMMARY_DETAIL tables.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
FREIGHT_CHARGE
FREIGHT_CHARGE contains the total freight charge for the invoice.
FREIGHT_CHARGE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
FREIGHT_CHARGE_BASE
FREIGHT_CHARGE_BASE contains the total freight charge for the invoice in the base currency.
PREPAID_AMOUNT
PREPAID_AMOUNT contains the dollar amount already paid on the invoice.
PREPAID_AMOUNT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PREPAID_AMOUNT_BASE
Prepaid amount in base units.
COMMERCIAL_UNIT_PRICE
Commercial unit price.
COMMERCIAL_UNIT_PRICE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COMMERCIAL_UNIT_PRICE_BASE
Commercial unit price in base units.
UNIT_COUNT
UNIT_COUNT contains the number of units on the invoice.
WEIGHT
WEIGHT contains the total weight of all items shipped on this invoice.
VOLUME
VOLUME contains the total volume of items shipped for this invoice.
WEIGHT_UOM_CODE
Weight unit of measure.
VOLUME_UOM_CODE
Volume unit of measure.
WEIGHT_BASE
Weight in base units.
VOLUME_BASE
Volume in base units.
WEIGHT_QUALIFIER
Weight qualifier.
INVOICE_TOTAL
INVOICE_TOTAL contains the count of the invoice's line_items.
EXCHANGE_RATE_DATE
Indicates the date to be used for any currency conversions; if null, the latest conversion should be used.
EXCHANGE_RATE_GID
Indicates the set of exchange rates to use for currency conversions; if null, the default set should be used
FREIGHT_CHARGE_FN
functional currency amount for feeight_charge
PREPAID_AMOUNT_FN
functional currency amount for prepaid_amount
INVOICE_SERVICE_CODE_GID
INVOICE_SERVICE_CODE_GID indicates the type of service that was performed. Foreign key to INVOICE_SERVICE_CODE. Summary level charges on invoice for service level. This is generally for air and ground.
