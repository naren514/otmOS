Definitions - DM_TRANSACTION_STATUS

One record per status per demurrage transaction. Contains the statuses of an demurrage transaction. For example, whether the demurrage was not reviewed, calculated, reviewed, settled etc.

DM_TRANSACTION_SEQ_NO
DM_TRANSACTION_SEQ_NO contains the unique identifier for the demurrage transaction.
STATUS_TYPE_GID
STATUS_TYPE_GID contains the unique identifier for the status type on the demurrage transaction.
STATUS_VALUE_GID
STATUS_VALUE_GID contains the unique identifier for the status value on the demurrage transaction.
