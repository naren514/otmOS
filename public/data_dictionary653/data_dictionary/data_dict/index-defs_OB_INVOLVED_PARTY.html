Definitions - OB_INVOLVED_PARTY

This table contains the link to the involved parties (shippers, service providers, consignee etc.). This is used to establish the communication with the parties during the order delivery. The Workflow set of tables determine when and what to communicate.

ORDER_BASE_GID
ORDER_BASE_GID contains the unique identifier for the order base.
INVOLVED_PARTY_QUAL_GID
INVOLVED_PARTY_QUAL_GID contains the unique identifier for the involved party qualifier for the order base.
INVOLVED_PARTY_CONTACT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COM_METHOD_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
