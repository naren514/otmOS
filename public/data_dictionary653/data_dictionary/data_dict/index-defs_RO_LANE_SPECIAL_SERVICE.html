Definitions - RO_LANE_SPECIAL_SERVICE

Mapping of arbitrary (ground move) lanes to Special Service codes for a rate offering

RO_LANE_SPECIAL_SERVICE_GID
Unique ID for this arbitrary lane
RO_LANE_SPECIAL_SERVICE_XID
Unique ID within domain for this arbitrary lane.
RATE_OFFERING_GID
Unique ID of associated rate offering.
LANE_SPECIAL_SERVICE_TYPE
Enumerated type; only possible value is ARBITRARY for now
X_LANE_GID
Geography for which this special service applies.
SOURCE_REGION_GID
Limits applicability of special service to cases where the shipment source is within the specified region
DEST_REGION_GID
Limits applicability of special service to cases where the shipment destination is within the specified region
SPECIAL_SERVICE_CODE_GID
Special service this lane maps to
RATE_DISTANCE_GID
The rate distance to be used while finding the distance for this arbitrary. This value will overrride the rate distance from the main rate offering.
ROUTE_CODE_GID
The route code to be used while finding distance for this arbitrary.
