Definitions - AUTO_ASSIGN_TYPE

Content provided by GC3...lists the valid types of auto-assignments that can take place such as shipment involved parties, shipment text templates, etc.

AUTO_ASSIGN_TYPE_GID
Type of auto-assignment
VALUE1_DATA_TYPE
Data type for field; S/B/I/N/D/G/U/C/P/H/L/F for String, Boolean, Integer, Number, Double, Gid, Uom, Char, Percentage, Geo hierarchy, drop-down list, Fixed values. If null, no field should be shown
VALUE1_GID_QUERY_CLASS
Gid query class if DATA_TYPE1 is a gid or drop-down list.
VALUE1_VALID_VALUE_TYPE_GID
If DATA_TYPE is F, indicates the set of fixed values to display in a picklist.
VALUE1_TRANSLATION_GID
Translation gid for field label
VALUE2_DATA_TYPE
Same as VALUE1_DATA_TYPE, used if there is a second field for the auto assign type; example: inv parties require 3 fields.
VALUE2_GID_QUERY_CLASS
Same as VALUE1_GID_QUERY_CLASS, used if there is a second field for the auto assign type; example: inv parties require 3 fields.
VALUE2_VALID_VALUE_TYPE_GID
Indicates the set of fixed values to display in a picklist.
VALUE2_TRANSLATION_GID
Same as VALUE1_TRANSLATION_GID used if there is a second field for the auto assign type; example: inv parties require 3 fields.
VALUE3_DATA_TYPE
Same as VALUE1_DATA_TYPE, used if there is a second field for the auto assign type; example: inv parties require 3 fields.
VALUE3_GID_QUERY_CLASS
Same as VALUE1_GID_QUERY_CLASS, used if there is a second field for the auto assign type; example: inv parties require 3 fields.
VALUE3_VALID_VALUE_TYPE_GID
Indicates the set of fixed values to display in a picklist.
VALUE3_TRANSLATION_GID
Same as VALUE1_TRANSLATION_GID, used if there is a second field for the auto assign type; example: inv parties require 3 fields.
ASSIGNER_CLASS
Java class handles the actual auto-assignment
