Definitions - SHIPPING_AGENT_CONTACT

Used to define the agent contacts for a given agent.

SHIPPING_AGENT_CONTACT_GID
Internal Id of the shipping agent contact
SHIPPING_AGENT_CONTACT_XID
External Id of the shipping agent contact
SHIPPING_AGENT_GID
Id of the related shipping agent
INVOLVED_PARTY_QUAL_GID
The role of the agent
INVOLVED_PARTY_CONTACT_GID
The primary contact id of the agent
COM_METHOD_GID
The preferred communication method of the agent
IS_ACTIVE
Indicator of whether or not the agent is active
EFFECTIVE_DATE
Date the agent becomes effective
EXPIRATION_DATE
Date the agent expires
PRIMARY_LEG_MODE_PROFILE_GID
Mode profile to be compared to transport mode of primary leg sell/buy shipment depending on agent type
DEST_LOC_REGION_GID
Location profile to be compared with dest location of primary leg sell/buy shipment depending on agent type
SHIPPER_LOCATION_PROFILE_GID
Location profile to be compared with location related to 'SHIPPER' involved party on the order
CONSIGNEE_LOCATION_PROFILE_GID
Location profile to be compared with location related to 'CONSIGNEE' involved party on the order
IS_PRIMARY
Indicator of whether or not the agent is considered the primary in the corresponding geography
IS_ALLOW_HOUSE_COLLECT
Indicator of whether or not the house bill is allowed to have a 'COLLECT' payment method
MAX_HOUSE_COLLECT_AMT
Maximum amount of freight charges on the house bill that can be 'COLLECT'
MAST_BL_TYPE
Type of Master B/L ('Express', or 'Original')
MAST_BL_CONSIGNEE_TYPE
Determines who to make the Master B/L Consignee party ('Consol Agent', 'House B/L Consignee', House B/L Notify Party')
MAST_BL_CONS_CONTACT_GID
Contact Id of the Master B/L Consignee if assigned by the 'Consol Agent'.
MAST_BL_CONS_COM_METH_GID
Communication Method of the Master B/L Consignee.
MAST_BL_NOTIFY_TYPE
Determines who to make the Master B/L Notify party ('Consol Agent', 'House B/L Consignee', 'House B/L Notify Party')
MAST_BL_NOTIFY_CONTACT_GID
Contact Id of the Master B/L Notify if assigned by the 'Consol Agent'.
MAST_BL_NOTIFY_COM_METH_GID
Communication Method of the Master B/L Notify.
MAST_BL_A_NOTIFY_TYPE
Determines who to make the Master B/L Also Notify party ('Consol Agent', 'House B/L Consignee', 'House B/L Notify Party')
MAST_BL_A_NOTIFY_CONTACT_GID
Contact Id of the Master B/L Also Notify if assigned by the 'Consol Agent'.
MAST_BL_A_NOTIFY_COM_METH_GID
Communication Method of the Master B/L Also Notify.
IS_INTERNAL
Indicator of whether or not the 'Sales Agent' is an internal or external sales agent
NFRC_RULE_GID
This field will be used to create NFRC shipment when run assign shipping agent to job action.
SHIPPING_AGENT_TYPE_GID
Indicates which shipping agent type this agent is.
