Definitions - ORDER_RELEASE_TEXT

Text that is associated with this order. An order can have more than one set of text, and the text can be shared by more than one order.

ORDER_RELEASE_GID
The order release.
TEXT_TEMPLATE_GID
Foreign key to represent the fixed set of text associated with the order.
DOCUMENT_DEF_GID
Document that should hold text
TEXT_OVERRIDE
Transactional override for the template on the specific document
