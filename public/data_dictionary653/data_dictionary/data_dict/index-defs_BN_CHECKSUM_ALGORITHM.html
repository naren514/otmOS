Definitions - BN_CHECKSUM_ALGORITHM

Stores algorithm that is used to calculate the checksum.

BN_CHECKSUM_ALGORITHM_GID
Unique identifier of the algorithm.
BN_CHECKSUM_ALGORITHM_XID
External identifier of the algorithm.
ALGORITHM_SUMMARY
A summary of the algorithm.
ALGORITHM_DESCRIPTION
A detailed description of the algorithm.
ALGORITHM_CLASS
Name of the java class implementing the algorithm.
ALGORITHM_CLASS_PARAMETERS
Free format parameters to the java class.
