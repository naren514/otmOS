Definitions - CORPORATION_PROFILE

This table is used to create enumerated groups of corporations. It creates a qualifier for the corporation to group them accordingly. Multiple corporations can be a part of a single or more corporation profiles. This table also stores the compatibility information between corporation profiles.

CORPORATION_PROFILE_GID
CORPORATION_PROFILE_GID contains the unique identifier for the corporation profile.
CORPORATION_PROFILE_XID
CORPORATION_PROFILE_XID contains the unique external identifier for the coroporation profile.
CORPORATION_PROFILE_NAME
CORPORATION_PROFILE_NAME contains the name for the corporation'sprofile.
