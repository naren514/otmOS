Definitions - LOCATION_RESOURCE

The resource defined for a location resource type.

LOCATION_RESOURCE_GID
The unique identifier.
LOCATION_RESOURCE_XID
The ID of the location resource.
LOCATION_GID
Location GID of the resource. Need to create a unique constraint on LOCATION_GID, RESOURCE_TYPE_GID and ID.
RESOURCE_TYPE_GID
Resource Type Gid of the resource. Need to create a unique constraint on LOCATION_GID, RESOURCE_TYPE_GID and ID.
LOCATION_RESOURCE_NAME
The name of the resource. Need to create a unique constraint on LOCATION_GID, RESOURCE_TYPE_GID and ID.
DESCRIPTION
The description of the resource.
CALENDAR_GID
The calendar GID that will be applied to the resource.
FLEX_COMMODITY_PROFILE_GID
The flex commodity profile ID that associates to the location resource.
EQUIPMENT_GROUP_PROFILE_GID
The equipment group profile ID that associates to the location resource.
MODE_PROFILE_GID
The mode profile ID that associates to the location resource.
SERVPROV_PROFILE_GID
The service provider profile ID that associates to the location resource.
CONTACT_PROFILE_GID
Contact Profile GID
FLEX_COMMODITY_CHECK_OPTION
Flex Commodify check option, such as CHECK ALL, CHECK BY LARGEST WEIGHT and CHECK BY LARGEST VOLUME.
DOMAIN_NAME
Domain of the user.
PREFERENCE_LEVEL
This field used to get preference level of resource for Appointment scheduling
