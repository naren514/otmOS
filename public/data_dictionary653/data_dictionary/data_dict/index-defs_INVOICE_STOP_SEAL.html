Definitions - INVOICE_STOP_SEAL

INVOICE_STOP_SEAL contains the serial number of the security device that relates to the stop.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
STOP_SEQ_NO
STOP_SEQ_NO identifies the stop number for which the seal was applied.
SEAL_SEQUENCE
SEAL_SEQUENCE is a one-up number to allow several seals being placed on a piece of equipment at the same stop.
SEAL_NUMBER
SEAL_NUMBER identifies the number of the seal that was placed on the piece of equipment.
