Definitions - MONITOR_AGENT_TYPE

This table describes the different monitor Agent types which can be one of the five Assignment, Execution, Planned, Unplanned and Exception. There can be only one Assignment and Execution Agent for a single Profile.

MONITOR_AGENT_TYPE_GID
The unique global identifier for the monitor agent type.
MONITOR_AGENT_TYPE_XID
The external identifier for the monitor agent type.
MONITOR_AGENT_TYPE_NAME
The description of the agent type.
