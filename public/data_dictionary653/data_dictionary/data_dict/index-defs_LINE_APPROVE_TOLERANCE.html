Definitions - LINE_APPROVE_TOLERANCE

Tolerance thresholds for individual shipment cost/invoice lines. For given shipment cost/invoice line attributes such as cost type and accessorial code, users may specify tolerance thresholds to prevent auto approval of invoice lines that are too much higer or lower than the corresponding shipment cost.

LINE_APPROVE_TOLERANCE_GID
ID for the line approval tolerance rule
IS_INVOICE_REFNUM_MATCH_ALL
If true, all invoice reference numbers in the LINE_APP_TOL_INVOICE_REFNUM table must exist on the invoice to evaluate this rule; if false, only one must exist
IS_SHIPMENT_REFNUM_MATCH_ALL
If true, all shipment reference numbers in the LINE_APP_TOL_SHIPMENT_REFNUM table must exist on the shipment to evaluate this rule; if false, only one must exist
EFFECTIVE_DATE
The Effective Date for this Line Approval Tolerance Rule.
EXPIRATION_DATE
The Expiration Date for this Line Approval Tolerance Rule.
