Definitions - MATCH_VALID_RULE_PROFILE

Groups a set of match validation rules together so that several may be assigned to a service provider.

MATCH_VALID_RULE_PROFILE_GID
Id for the match validation rule group or profile
DESCRIPTION
Optional user description for the profile
IS_DEFAULT
If this is checked (value=Y), this profile will be used if the validate actio is run on an invoice and the service provider on the invoice does not have a profile specified.
