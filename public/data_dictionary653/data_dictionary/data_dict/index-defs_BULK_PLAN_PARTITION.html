Definitions - BULK_PLAN_PARTITION

TOTAL_NUM_STOPS
average # of stops of all shipments in this partition
TOTAL_COST
total cost of all shipments built in this partition
STATE
Execution state of the bulkplan partition. INTERRUPTED when a user has requested termination but the partition is still running. TERMINATED when the partition completes termination.
SINCE
Time of last state change
APP_MACHINE_GID
If scaled, the application machine running the partition
TOTAL_SHIPMENT_TIME_SECS
Sum of all shipments times (end-start) in seconds
TOTAL_WEIGHT_CAPACITY
Sum of all shipment weight capacities (based on equipment used)
TOTAL_VOLUME_CAPACITY
Sum of all shipment volume capacities (based on equipment used)
NUM_OF_ORDERS_THRU_POOL
Number of orders that went through a pool point
NUM_OF_ORDERS_THRU_XDOCK
Number of orders that went through a crossdock
TOTAL_BEST_DIRECT_COST
Total best direct cost of planned orders
NUM_OF_WA_CREATED
Number of Work assignments created during the Bulk plan partition process.
