Definitions - DRIVER_LOCATION_PREFERENCE_D

This table captures the actual prefer / avoid preference condition for locations / regions / lanes that are applicable for a Location Preference GID

LOCATION_PREF_SEQ_NO
The auto-generated sequence number makes the primary key for this table
LOCATION_PREF_GID
The Location Preference GID in OTM
LOCATION_PREF_TYPE
An Enumerate that indicates the preference condition. Values - 1 - Indicates 'Prefer' 2 Indicates 'Avoid'
LOCATION_GID
Refers to the preferred / not-proferred Location GID
REGION_GID
Refers to the a preferred / not-proferred Region GID
LANE_GID
Refers to the a preferred / not-proferred Lane GID
LOCATION_PREF_VALUE
Indicates the weightage for this location preference
