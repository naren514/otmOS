Definitions - INVOICE_INVOLVED_PARTY

One record per involved party per invoice. The related parties to the invoice. This can include the shipper, consignee, bill to, etc. This table maps to INVOLVED_PARTY_QUAL, which is the table that contains the description of the involved party.

INVOICE_GID
INVOICE_GID is the unique identifier for the invoice.
INVOLVED_PARTY_QUAL_GID
INVOLVED_PARTY_QUAL_GID is the unique identifier for an involved party.
INVOLVED_PARTY_CONTACT_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
COM_METHOD_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
