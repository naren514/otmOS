Definitions - ORACLE_ERROR

This table maps Oracle codes to a translation id, in order to provide better information to the user on how to fix a problem when an Oracle error arises.

ERROR_CODE
The oracle error code.
CONSTRAINT_ID
A constraint name, such as the foreign key name, if applicable.
ERROR_TRANSLATION_GID
The translation id that the oracle error code is associated with. Allows the ability to add customized translations for an Oracle error.
SOLUTION_TRANSLATION_GID
The translation id that provides a user-friendly solution for the Oracle error.
