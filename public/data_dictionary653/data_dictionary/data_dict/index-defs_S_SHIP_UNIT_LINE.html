Definitions - S_SHIP_UNIT_LINE

This table stores the line item entry for each shipment ship unit entry for S_SHIP_UNIT table.

S_SHIP_UNIT_GID
A unique pointer to define the shipment ship unit. The gid contains the domain portion.
S_SHIP_UNIT_LINE_NO
The unique sequence within the ship unit to define the individual line items.
WEIGHT
The weight of the item.
WEIGHT_UOM_CODE
Unit of measure for WEIGHT column.
WEIGHT_BASE
WEIGHT in base units.
VOLUME
The volume of the item.
VOLUME_UOM_CODE
Unit of measure for VOLUME column.
VOLUME_BASE
VOLUME in base units.
TRACKING_TAG1
Additional information about the item. This could be any tracking numbers, SKU codes, SPLC codes etc.
TRACKING_TAG2
Additional information about the item. This could be any tracking numbers, SKU codes, SPLC codes etc.
TRACKING_TAG3
Additional information about the item. This could be any tracking numbers, SKU codes, SPLC codes etc.
TRACKING_TAG4
Additional information about the item. This could be any tracking numbers, SKU codes, SPLC codes etc.
OR_LINE_GID
A unique identifier to define the multiple order release lines per order release. The gid has the domain portion.
ORDER_RELEASE_GID
A unique identifier created by the system for the order release. the gid contains the domain portion.
ITEM_PACKAGE_COUNT
the number of packages in this S_SHIP_UNIT_LINE for one ship unit ( its S_SHIP_UNIT may represent more than one same ship units). (e.g., for 20 ship units that contains 4 cases of 24 cans of coca, each S_SHIP_UNIT_LINE row will have ITEM_GID: coca, ITEM_COUNT 4x24, ITEM_PACKAGE_SPEC_GID: case-of-24-cans, ITEM_PACKAGE_COUNT: 4. In the S_SHIP_UNIT table, the SHIP_UNIT_COUNT will be 20)
DECLARED_VALUE_BASE
DECLARED_VALUE in base units.
DECLARED_VALUE_CURRENCY_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
PACKAGED_ITEM_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
HAZMAT_ITEM_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
RECEIVED_WEIGHT_BASE
RECEIVED_WEIGHT in base units.
RECEIVED_VOLUME_BASE
RECEIVED_VOLUME in base units.
RECEIVED_PACKAGING_UNIT_COUNT
Packaging Unit Count Recieved
RECEIVED_COUNT_PER_SHIP_UNIT
Count per Ship Unit Received
T_ORDER_RELEASE_GID
This is used as a non-foreign key reference to the order release.
T_OR_LINE_GID
This is used as a non-foreign key reference to the order release line.
T_SHIP_UNIT_GID
This is used as a non-foreign key reference to the order release ship unit line.
T_SHIP_UNIT_LINE_NO
This is used as a non-foreign key reference to the order release ship unit line.
