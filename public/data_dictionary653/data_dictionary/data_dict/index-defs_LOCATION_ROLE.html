Definitions - LOCATION_ROLE

The role(s) played by the location. For e.g., a location can be a supermarket, loading station and unloading station.

LOCATION_ROLE_GID
LOCATION_ROLE_GID contains the unique identifier for the location role. Examples of roles include airport, carrier and port.
LOCATION_ROLE_XID
LOCATION_ROLE_XID contains the unique external identifier for the location role.
LOCATION_ROLE_NAME
LOCATION_ROLE_NAME contains the name for the location role.
