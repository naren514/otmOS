Definitions - OB_REMARK

Free form entry text for the order base. This is a space provided for the customer/user to enter any additional remarks for the base order. For e.g., a remark may say that the "Driver may not wear a beard while delivering this order due to hazardous chemicals"

ORDER_BASE_GID
A unique identifier to define the order base in the system. It is system generated. The gid contains the domian portion.
REMARK_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
