Definitions - BULK_PLAN_MILESTONE

BULK_PLAN_MILESTONE_SEQ
A globally unique ID for this bulk plan milestone
BULK_PLAN_GID
Bulk plan ID
PLANNING_MILESTONE_GID
Planning milestone type (e.g. Order Bundling)
PARENT_MILESTONE_SEQ
Parent milestone (if any)
START_TIME
Start time of milestone
END_TIME
End time of milestone
STATE
Execution state
SINCE
Time of last state change
ALGORITHM
Algorithmic choice (e.g. 2-opt, DP)
NUM_THREADS
For multi-threaded milestones, the # of threads allocated to the milestone
PROGRESS
Milestone-dependent progress amount (e.g. iterations completed, bytes processed)
MAX_PROGRESS
Milestone-dependent problem size (e.g. total # of iterations to complete, total # of bytes to process)
PROGRESS_DESCRIPTION
A text description to place after 'xx of yy'. E.g. 'orders processed'
PARTITION_GID
If part of a bulk plan partition, the partition ID
LOG_PROCESS_ID
Log process ID
