Definitions - INVOICE_STOP_REASON

Indicates the reason for shipment stops associated with the invoice. For example, "PICKUP", "DELIVERY", etc.

INVOICE_STOP_REASON_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
INVOICE_STOP_REASON_XID
GC3 Unique identifier for the entity. Is used for creating the GID
INVOICE_STOP_REASON_DESC
The description for the stop.
