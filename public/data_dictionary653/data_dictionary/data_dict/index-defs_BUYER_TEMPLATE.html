Definitions - BUYER_TEMPLATE

It is a buy template with transport mode, move perspective and order release template info.

BUYER_GID
Along with buy_template_seq is the primary key of this table.
ORDER_RELEASE_TEMPLATE_GID
It is order_release_gid which has is_template='Y' in order_release table.
IS_ACTIVE
Define if this template is active or inactive.
BUY_TEMPLATE_SEQ
The sequence number with buyer_gid will be the primary key of this table
