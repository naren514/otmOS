Definitions - GL_KEY_COMPONENT_TYPE

Defines possible types of components for hierarchy used to define general ledger lookup keys

GL_KEY_COMPONENT_TYPE_GID
Global identifier for the component type of the key.
GL_KEY_COMPONENT_TYPE_XID
External identifier for the component type of the key.
JAVA_CLASS
Java class to obtain the information.
GL_CODE_ASSIGN_TYPE
This will indicate whether the component type is to be part of an Order lookup key or a shipment lookup key or a invoice or a bill.
