Definitions - XML_BUILDER

This table enumerates the xml builder classes used to generate outbound xml documents. Each xml_builder defines a "chunk" of xml that may be excluded from an outbound xml document. Each out_xml_profile_d record points to a given xml builder.

XML_BUILDER_GID
Globally unique identifier for a given xml builder class. GID and XID are the same in this case, since only public data resides in this table.
XML_BUILDER_XID
Domain unique identifier for a given xml builder class.
JAVA_CLASS_NAME
Fully qualified java class name for an xml builder class.
