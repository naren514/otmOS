Definitions - Q_LIFETIME_EVENT

The data queue for lifetime events

Q_EVENT_SEQ
Queue sequence for the lifetime event. The sequence name is Q_LIFETIME_EVENT_SEQUENCE.
TOPIC_DESCRIPTION
A short description of the topic, displayed on the lifetime events finder
TOPIC_CLASS_NAME
Workflow topic class of the event
TOPIC_PARAMETERS
Workflow parameters for the event
Q_PREEMPTION_PRIORITY
A negative sequence used for queue preemption
Q_START
Time the event was last queued. If the event is scheduled, this will equal the scheduled time. If resubmitted, the resubmit time. Otherwise it matches the insert date.
Q_STATE
Queue state of the event (QUEUED, ACTIVE, CANCELED, PROCESSED, FAILED)
Q_TIME
Time the event spent in queue
Q_PROCESS_START
Time the event was last dequeued and passed to the executor for processing
Q_PROCESS_TIME
Time the event took to execute
Q_LOG_PROCESS_ID
ID linking the event execution to the system logs
Q_PROCESS_MACHINE_GID
The application machine that polled and executed the event
Q_RELATED_QUEUE_SEQ
Sequence of the parent queue event that populated this event
P_QUEUE_START_KEY
Index for date-based partitioning
