Definitions - BULK_PLAN_BY_MODE

Bulk plan summary information break down by transport mode

NUM_ORDERS_PLANNED
number of orders planned on this transport mode
NUM_ORDER_MOVEMENTS_PLANNED
number if order movements planned on this transport mode
NUM_SHIPMENTS_BUILT
number of shipments built on this transport mode
TOTAL_COST
total cost of all shipments built on this transport mode
TOTAL_NUM_STOPS
average number of stops of all shipments built on this transport mode
TOTAL_SHIPMENT_TIME_SECS
Sum of all shipments times (end-start) in seconds
TOTAL_WEIGHT_CAPACITY
Sum of all shipment weight capacities (based on equipment used)
TOTAL_VOLUME_CAPACITY
Sum of all shipment volume capacities (based on equipment used)
