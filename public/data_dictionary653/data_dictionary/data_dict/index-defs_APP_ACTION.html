Definitions - APP_ACTION

IS_OBJECT_LOCK
If Y, the business object is locked during the action call. The may lock multiple business object for UI/planning actions.
OBJECT_LOCK_TIMEOUT
Timeout for object lock, if IS_OBJECT_LOCK=Y
OBJECT_LOCK_POLLING
If not using JMS notification for cross-machine locks, the polling interval in msecs
APPLICATION_NAME
The name of Application. E.g. Dock.
DATA_QUERY_TYPE_GID
The unique identifier of data query type.
IS_POWER_ACTION
An indicator whether this is a power action.
POWER_ACTION_NAME
The name of power action.
