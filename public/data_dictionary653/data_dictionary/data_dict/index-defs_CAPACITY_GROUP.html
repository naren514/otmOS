Definitions - CAPACITY_GROUP

A capacity group consists of a list of rate offerings

CAPACITY_GROUP_GID
Unique Identifier for each CAPCITY GROUP. This is a combination of XID and Domain Name
CAPACITY_GROUP_XID
GC3 Unique identifier for the entity. Is used for creating the GID
SERVPROV_GID
This is set up so service providers can edit their own capacity usage records.
