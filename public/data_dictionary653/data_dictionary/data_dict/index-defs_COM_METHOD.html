Definitions - COM_METHOD

This is a master table that stores the various communication methods that can be used to communicate with the locations. Some examples include: EDI E-Mail Fax Phone HTTPPOST etc.

COM_METHOD_GID
COM_METHOD_GID contains the unique identifier for the communication method for the location.
COM_METHOD_XID
COM_METHOD_XID contains the unique external identifier for the communication method for the location.
MAX_RETRIES_ON_ERROR
Maximum number of retries upon delivery failure.
ALTERNATE_COM_METHOD_GID
Alternate communication method to delivery message when retries are exhausted.
