Definitions - JOB

IS_MEMO_BL
Indicate if the documentation process should create one BL doc for all the shipments in the job. If Y, use MEMO_BL_SHIPMENT_GID to generate the BL.
MEMO_BL_SHIPMENT_GID
Indicate which shipment will be used to generate B/L. This field can not be empty when IS_MEMO_BL is Y.

EXCHANGE_RATE_DATE
Indicates the date to be used for any currency conversions; if null, the latest conversion should be used.
EXCHANGE_RATE_GID
Indicates the set of exchange rates to use for currency conversions; if null, the default set should be used
