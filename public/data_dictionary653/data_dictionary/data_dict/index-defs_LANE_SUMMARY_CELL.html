Definitions - LANE_SUMMARY_CELL

Time periods which make up a lane summary

TIME_PERIOD_NUM
The number of the time period: 1, 2, 3, ...
SHIPMENT_COUNT
The number of shipments that fall into this time period
START_DATE
The start date for this time period
END_DATE
The end date for this time period. Ex: for 1 week time periods, the start_date and end_date will be 7 days apart for each time period
