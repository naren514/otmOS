Definitions - VAT_ANALYSIS

EXCHANGE_RATE_DATE
Indicates the date to be used for any currency conversions; if null, the latest conversion should be used.
EXCHANGE_RATE_GID
Indicates the set of exchange rates to use for currency conversions; if null, the default set should be used
VAT_BASIS_AMOUNT
The portion of the invoice amount that will be taxed at the rate for the given VAT code
VAT_BASIS_AMOUNT_CURRENCY_GID
currency of VAT_BASIS_AMOUNT
VAT_BASIS_AMOUNT_BASE
base amount for the VAT_BASIS_AMOUNT
VAT_RATE
tax rate that the basis amount will be taxed
TAX_AMOUNT_FN
functional currency amount for tax_amount
VAT_BASIS_AMT_FN
functional currency amount for vat_basis_amount
