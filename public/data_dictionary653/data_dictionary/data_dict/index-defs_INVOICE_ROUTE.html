Definitions - INVOICE_ROUTE

One record is created for every carrier (related to the shipment included on the invoice) per invoice. The difference between an invoice stop and an invoice route is that a new route indicates a change in transportation carriers (and the location where the interchange is made), while a stop is simply a stop to pick up or deliver goods.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
ROUTE_SEQ_NO
ROUTE_SEQ_NO contains the sequence number for the route on an invoice.
TRANSPORT_MODE_IDENTIFIER
The transport mode identifier.
SERVPROV_ALIAS_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SERVPROV_ALIAS_VALUE
Service provider associated with the route.
INTERMODAL_SERVICE_CODE
Intermodal service code.
JCT_CITY_CODE
Junction city code.
