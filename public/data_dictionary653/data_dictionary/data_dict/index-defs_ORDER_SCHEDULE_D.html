Definitions - ORDER_SCHEDULE_D

Order delivery schedule details.

ORDER_SCHEDULE_D_ID
A system generated number for this detail record.
ORDER_SCHEDULE_GID
The ID of the parent order schedule.
DATE_DAY_OF_WEEK
The number of days to offset the early delivery date from the input date (i.e. +2).
EARLY_DELIVERY_DATE_OFFSET_VAL
The number of days to offset the early delivery date from the input date (i.e. +2).
EARLY_DELIVERY_TIME
The early delivery time of day.
LATE_DELIVERY_DATE_OFFSET_VAL
The number of days to offset the late delivery date from the input date (i.e. +3).
LATE_DELIVERY_TIME
The late delivery time of day.
ORDER_RELEASE_TEMPLATE_GID
The ID of the related order release template.
TEMPLATE_COPY_TYPE
Identifies the copy behavior of the template (Copy if NULL or Replace)
