Definitions - INVOICE_SUMMARY_REMARK

Remarks to summarize an invoice. One record per remark per invoice.

INVOICE_GID
INVOICE_GID contains the unique identifier for the invoice.
REMARK_SEQ_NO
REMARK_SEQ_NO contains a sequence number for a remark on an invoice.
REMARK_QUAL_IDENTIFIER
REMARK_QUAL_IDENTIFIER contains the qualifier for the remark.
REMARK_TEXT
REMARK_TEXT contains the text of the remarks on the invoice.
