Definitions - GTM_STR_COMPONENT_CURRENCY

A generic key/value representation to capture all possible amount/currency information for example declare value, fees, freight cost etc

GTM_STRUCTURE_GID
GTM_STRUCTURE_GID contains the unique identifier for the trade item structure.
GTM_STRUCTURE_COMPONENT_SEQ
Trade Item Structure Component Sequence Number
GTM_VALUE_QUALIFIER_GID
A pointer to the currency type. For example: declared value, invoice value etc
VALUE
Value for the specified currency type.
