Definitions - P_SHIPMENT_SET

Defines selection criterion for building a shipment set

BEGIN_DATE
Only shipments whose start date is after the begin_date are selected
END_DATE
Only shipments whose end date is before the end_date are selected
SAVED_QUERY_GID
Only shipments that meet the criterion specified in this saved query are selected
GEO_HIERARCHY_PROFILE_GID
Defines the "good" geographies so we know which lanes to bring over to procurement
