Definitions - E_KPI_TARGET_VALUE_STAGE

This table in combination of its detail table, stores the USER_ENTERED target data. The table contains data for only those KPIs that have a TARGET_VALUE_TYPE of USER_ENTERED

KPI_TARGET_VALUE_TYPE_ID
With this column the KPI_TARGET_VALUE_STAGE table is folded under the E_KPI_TARGET_VALUE_TYPE table when the Target Value Type is 'User Entered'
