Definitions - HTS

This table defines the HTS governmental code and name for the item.

HTS_GID
HTS_GID contains the unique identifier for the Harmonized Tariff System (HTS).
HTS_XID
HTS_XID contains the unique external identifier for the Harmonized Tariff System (HTS).
TERTIARY_RQ_UOM_CODE
Tertiary UOM.
TERTIARY_RQ_UOM_TYPE
Tertiary UOM Type.
