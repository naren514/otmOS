Definitions - DM_TRANSACTION_NOTE

Unlimited notes per demurrage transaction. Freeform, timestamped notes that can be used to indicate explanations/information throughout the lifetime of the demurrage transaction.

DM_TRANSACTION_SEQ_NO
DM_TRANSACTION_SEQ_NO contains the unique identifier for the demurrage transaction.
NOTE_SEQ_NO
NOTE_SEQ_NO contains the unique identifier for the DM_TRANSACTION_NOTE.
SUMMARY
Note Summary
NOTE
NOTE contains the note(s) about the demurrage transaction.
IS_SYSTEM_GENERATED
Indicates whether this is system generated or not.
