Definitions - ITEM_ACCESSORIAL

This table has a link to the accessories table. It defines the accessories required to ship an item. For e.g., a truck with refrigerated unit may be required to ship Ice Creams. The accessories differ from the special services, in that the customer has to pay for the accessories.

ITEM_GID
ITEM_GID contains the unique identifier for the item being shipped.
ACCESSORIAL_CODE_GID
ACCESSORIAL_CODE_GID contains the unique identifier for the accessorial code.
