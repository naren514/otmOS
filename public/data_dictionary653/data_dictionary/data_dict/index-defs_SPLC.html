Definitions - SPLC

The Postal Standard Point Location Code is a code used in Geopolitical numbering systems for locations.

SPLC_GID
The unique Postal Standard Point Location Code.
SPLC_XID
The Postal Standard Point Location Code.
DESCRIPTION
The Description of the Postal Standard Point Location Code.
SERVICE_TYPE
The Service Type of the Postal Standard Point Location Code. P = Postal, R = Rail, B = Both
CITY
City.
