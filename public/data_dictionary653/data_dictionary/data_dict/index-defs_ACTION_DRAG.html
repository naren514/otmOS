Definitions - ACTION_DRAG

This table defines the drag and drop behavior in the workspace. It defines which drags are legal and what action to run.

FROM_QUERY_TABLE_GID
origin query of a drag and drop
TO_QUERY_TABLE_GID
target query of a drag and drop
ACTION_GID
action to run when performing a drag and drop
STATE
action_def_state to invoke for a drag and drop
