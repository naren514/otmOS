Definitions - BN_RANGE

Defines pre-assigned ranges, which are instances of named ranges.

BN_RANGE_GID
The unique identifier for the range.
BN_RANGE_XID
The external identifier for the range.
BN_NAMED_RANGE_GID
The named range to which this range belongs.
BN_RULE_GID
The business number rule used to generate numbers for the range. The only segment types allowed are BN sequences, either alphabetical or numerical. If the field is null, the set of numbers defined in the bn_range_number table is used.
SEQUENCE_NO
Sequence number used for ordering ranges in a named range.
