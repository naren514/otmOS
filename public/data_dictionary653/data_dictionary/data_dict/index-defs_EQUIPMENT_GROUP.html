Definitions - EQUIPMENT_GROUP

This table defines the categories that can be used for grouping the equipments. For example, the equipments can be grouped into: 42FT Covered Hopper, 60FT Flatcar, Frozen, Heated etc. The user can also record the size (length, height, width), weight and volume. A link to the defined temperature controls can also be established.

EQUIPMENT_GROUP_GID
EQUIPMENT_GROUP_GID is the unique identifier for the equipment group.
EQUIPMENT_GROUP_XID
EQUIPMENT_GROUP_XID is the unique identifier for the equipment group.
EQUIPMENT_GROUP_NAME
EQUIPMENT_GROUP_NAME contains text to describe the group of equipment.
TEMPERATURE_CONTROL_GID
A unqie identifier to define the temperature controls value for the commodity. The gid contains the domian portion.
EFFECTIVE_WEIGHT
EFFECTIVE_WEIGHT contains the weight of the equipment.
EFFECTIVE_WEIGHT_UOM_CODE
Unit of measure for the effective_weight column.
EFFECTIVE_WEIGHT_BASE
EFFECTIVE_WEIGHT contains the weight of the equipment in base units.
EFFECTIVE_VOLUME
EFFECTIVE_VOLUME contains the capacity for the equipment.
EFFECTIVE_VOLUME_UOM_CODE
Unit of measure for the effective_volume column.
EFFECTIVE_VOLUME_BASE
EFFECTIVE_VOLUME contains the capacity for the equipment in base units.
WIDTH
WIDTH contains the width of the equipment group.
WIDTH_UOM_CODE
Unit of measure for the width column.
WIDTH_BASE
WIDTH contains the width of the equipment group in base units.
LENGTH
LENGTH contains the length of the equipment group.
LENGTH_UOM_CODE
Unit of measure for the length column.
LENGTH_BASE
LENGTH contains the length of the equipment group in base units.
HEIGHT
HEIGHT contains the height of the equipment group.
HEIGHT_UOM_CODE
Unit of measure for the height column.
HEIGHT_BASE
HEIGHT contains the height of the equipment group in base units.
DOOR_HEIGHT
Height of the door on the equipment group. Used to check against the height of a ship unit.
CURVE_GID
Defined the curve function of the curved roof of the equipment group.
MAX_LHS_WEIGHT
The max weight allowed on the left hand side of the equipment.
MAX_RHS_WEIGHT
The max weight allowed on the right hand side of the equipment.
MAX_SIDE_WEIGHT_DIFF
The max weight difference allowed between two sides (left and right) of the equipment to prevent load inbalance.
ALLOW_LIFO_ONLY
Indicates whether the equipment is for LIFO only.
COMB_EQUIP_GRP_STRUCT_SEQ
If equipment group is a composite equipment group, it will have this comb_equip_grp_struct_seq populated. It contains the list of equipment groups that belong to a double or triple.
COMPARTMENT_TYPE_GID
A unique identifier to define the compartment type
HAS_METER
Indicates if the compartment has METER or not(The METER is a device that is used the measure how much product is discharged at any given stop).
COMMODITY_GID
Commodity that is compatible with the Equipment Group
IS_BULK
Indicates if it is a BULK. If Y then BULK If N then PACKAGED
EQUIPMENT_TYPE_CODE
Alphanumeric code used to designate a specific type of equipment and attributes.
MECHANICAL_DESIGNATION
Alphabetic code assigned to the physical description of the unit.
EQUIPMENT_OWNER_TYPE
The type of company that owns the stenciled mark on the equipment. For example private or carrier.
EQUIP_MAX_GROSS_WT
The maximum permissible weight of the equipment and the load.
FLOOR_HEIGHT
Actual height of the platform or floor above the ground or the rail.
EMPTY_CG_HEIGHT
When empty, indicates the height from Top of Rail to the Center of Gravity
