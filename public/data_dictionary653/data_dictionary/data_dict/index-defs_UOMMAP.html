Definitions - UOMMAP

This table stores the conversion/ map from one Unit of Measure to another Unit Of Measure. For example: For Length the Meter To Feet conversion is 3.2808399, Meter to Yard is 1.0936133 etc.

TYPE
TYPE contains the type for the unit of measure for localization. Examples include: DISTANCE DURATION LENGTH SPEED TEMPERATURE VOLUME WEIGHT
FROM_UOM_CODE
FROM_UOM_CODE contains an abbreviated name for the original unit of measure being converted.
TO_UOM_CODE
TO_UOM_CODE contains an abbreviated name for the unit of measure to which another unit of measure is being converted.
CONVERSION_CLASS
CONVERSION_CLASS contains the Java class name that is used to convert a temperature from one unit of measure to another. The units of measure include Celsius, Fahrenheit and Kelvin.
CONVERSION_FACTOR
CONVERSION_FACTOR contains the mathematical factors for converting one unit of measure to another.
