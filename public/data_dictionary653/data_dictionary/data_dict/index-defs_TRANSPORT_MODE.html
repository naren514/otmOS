Definitions - TRANSPORT_MODE

This is the master table that stores the various transportation modes that can be used for shipping the goods. Some examples include: Air Train Truck Ship etc.

TRANSPORT_MODE_GID
A unique identifier to define the transportation modes. (includes the domain name as part of the key)
TRANSPORT_MODE_XID
A unique identifier to define the transportation modes. (does not include the domain name as part of the key)
TRANSPORT_MODE_NAME
The name/description of the transportation mode. For example: Air, Barge, Drayage, Truckload, Rail etc.
TRANSPORT_MODE_QUALIFIER
DISTINGUISHES THE TRANSPORTATION MODE AS EITHER A STANDARD TRANSPORT MODE, A HAZARDOUS TRANSPORT MODE OR BOTH. VALUES: B=BOTH, S=STANDARD TRANSPORT MODE, H=HAZARDOUS TRANSPORT MODE.
IS_MASTER_CARR_REF_REM_VISIBLE
Identifies whether the master carrier reference remark is displayed on the 'View Related Remarks' screen for this particular transportation mode.
IS_CONSIDER_COST
Indicate if cost should be considered (therefore rating is invoked) during ship unit building (packing).
THU_PROFILE_GID
Specifies all compatible THUs that can be used during ship unit building (packing).
