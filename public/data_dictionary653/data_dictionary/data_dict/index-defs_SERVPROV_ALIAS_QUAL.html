Definitions - SERVPROV_ALIAS_QUAL

This table stores the qualifier for the service provider alias. For example SCAC.

SERVPROV_ALIAS_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
SERVPROV_ALIAS_QUAL_XID
GC3 Unique identifier for the entity. Is used for creating the GID
