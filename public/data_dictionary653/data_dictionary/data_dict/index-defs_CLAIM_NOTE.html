Definitions - CLAIM_NOTE

Holds notes that are entered by a user or by GC3 to record a history of claim activity

CLAIM_GID
Id of the claim that this note is linked to
CLAIM_NOTE_SEQ_NO
Sequence number to keep notes unique
TIMESTAMP
Timestamp for note
NOTE
Note text
IS_SYSTEM_GENERATED
Indicates if this note was added to the claim by GC3.
