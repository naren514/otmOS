Definitions - GTM_STRUCTURE

Define possible structures for the Trade Item. These structures would vary by trade direction, product classification type, product classification code, supplier and/or lot/batch

GTM_STRUCTURE_GID
GTM_STRUCTURE_GID contains the unique identifier for the trade item structure.
GTM_STRUCTURE_XID
GTM_STRUCTURE_XID contains the XID unique identifier for the trade item structure.
GTM_STRUCTURE_NAME
GTM_STRUCTURE_NAME contains text for the name of the trade item structure
DESCRIPTION
DESCRIPTION contains text for the description of the trade item structure
IS_ACTIVE
When N, indicates trade item structure is inactive and should not be used by the algorithm.
EFFECTIVE_DATE
The Effective Date of this trade item structure
EXPIRATION_DATE
The Expiration Date of this trade item structure
SOURCE
source of this trade item structure. Its it created within GTM or imported from out system.
ITEM_GID
Trade Item ID, this represents the trade item for which this structure is defined.
TRADE_DIRECTION
Trade Direction. Used to define import, export or both specific structure
GTM_PROD_CLASS_TYPE_GID
Product Classification Type ID Used to define product classification type specific structure
DEFAULT_COUNTRY_OF_ORIGIN
Default country of origin. Any component that doesn?t have country of origin specified will use this value as country of origin
