Definitions - DATA_QUEUE_DEF

Defines a data queue

DATA_QUEUE_DEF_GID
Global ID for the data queue definition
DATA_QUEUE_DEF_XID
Data queue definition ID
DATA_QUEUE_TABLE_GID
Data table GID
DATA_QUEUE_POLLER_GID
Data queue poller GID
DATA_QUEUE_EXECUTOR_GID
Data queue executor GID
THREAD_COUNT
Number of threads to allocate to pollers
BATCH_SIZE
Number of events to retrieve in a polling batch. If null, all events are retrieved.
POLLING_FREQUENCY
Duration between queue polls
FINDER_SET_GID
Finder set for Event Queue viewer
IS_ACTIVE
If N, the queue is inactive and ignored by GC3
LRT_RELATED_EVENT_QUEUE
Event queue associated with the data queue. This allows the server to check event queue resources when pulling of records from the queue table.
LRT_SCA_REQUIREMENTS
For long running tasks, determines if the task should be pulled out of the data queue only when resources are available to process it. NONE=task is pulled regardless of resources; THREAD=task is pulled only if an event thread in the queue is available; MEDIATOR=task is pulled only if an event thread is available and the task will not be blocked by the mediator. Note this column is only used when LRT_RELATED_EVENT_QUEUE is not null.
