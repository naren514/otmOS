Definitions - CLAIM

Information for a single damage claim in GC3

CLAIM_GID
Unique ID for damage claim
DAMAGE_INFORMATION
Text description of damage
REPORT_DATE
Date incident is reported
INCIDENT_DATE
Date incident occurred
DAMAGE_NOTIFY_POINT_GID
Code for notification point for damage claim, user defined in power data
DAMAGE_CAUSE_GID
Code for cause of damage, user defined in power data
DAMAGE_NATURE_GID
Code for nature of damage, user defined in power data
DAMAGE_VALUE
Financial loss (value) of only the damaged goods; may be calculated by GC3 or specified by user
IS_DAMAGE_VALUE_FIXED
If Y, GC3 will not calculate damage value and will keep the value entered by user
SIZE_OF_LOSS
Financial loss for the entire claim including all losses incurred and credits; may be calculated by GC3 or specified by user
IS_SIZE_OF_LOSS_FIXED
If Y, GC3 will not calculate size of loss and will keep the value entered by user
SHIPMENT_GID
Id of the shipment relating to the claim, if applicable
INDICATOR
Claim indicator value
USER_DEFINED1_ICON_GID
User defined icon for claim
USER_DEFINED2_ICON_GID
User defined icon for claim
USER_DEFINED3_ICON_GID
User defined icon for claim
USER_DEFINED4_ICON_GID
User defined icon for claim
USER_DEFINED5_ICON_GID
User defined icon for claim
NOTIFICATION_DATE
Date of notification for the claim
NOTIFY_PARTY_CONTACT_GID
Party that notified of the damage
LIABLE_PARTY_CONTACT_GID
Party responsible for causing damage
IS_SELF_RECONDITION
Indicates if the claim is self-reconditioning
