Definitions - GL_LOGIN_HISTORY

Contains the history of attempted GC3 logins.

LOGIN_SEQUENCE
A sequence number used as primary key.
GL_USER_LOGIN_ATTEMPT
The attempted GL_USER.
LOGIN_STATUS
Whether the attempt was successful, and if not, what was the reason.
LOGIN_MODE
Whether this was interactive or batch.
IP_ADDRESS
The IP address of the client. It is the IP address of the machine which has the web browser.
LOGIN_TIME
Time of the login attempt.
PARTITION_KEY
A key (from 1-4) based upon the insert_date which helps purge data periodically.
