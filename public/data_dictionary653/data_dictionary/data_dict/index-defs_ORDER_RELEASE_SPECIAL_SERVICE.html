Definitions - ORDER_RELEASE_SPECIAL_SERVICE

This table links to the special services required for the order release against a base order. The special services differ from the accessories in that the customer does not pay for special services.

ORDER_RELEASE_GID
A unique identifier created by the system for the order release. the gid contains the domain portion.
SPECIAL_SERVICE_CODE_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
TIME
Duration required for this special service on this order.
FOR_SOURCE_DEST
Define if this special service is required for source and/or destination location of this order.
