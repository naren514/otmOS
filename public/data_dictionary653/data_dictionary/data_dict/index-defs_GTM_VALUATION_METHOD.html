Definitions - GTM_VALUATION_METHOD

This table is used for identification of different methods of custom valuation.

GTM_VALUATION_METHOD_GID
This field represents unique valuation method ID accross all domains.
GTM_VALUATION_METHOD_XID
This field represents unique valuation method ID in a particular domain.
GTM_VALUATION_METHOD_NAME
Name for a valuation method.
DESCRIPTION
This field describes the valuation method.
GTM_VALUATION_METHOD_CODE
This field describes the valuation method code. which will vary between 1 to 6 based on taxation and custom guidelines i.e., "SAD Guidelines - C43".
