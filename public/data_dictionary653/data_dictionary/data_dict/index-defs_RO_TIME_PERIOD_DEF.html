Definitions - RO_TIME_PERIOD_DEF

RO_TIME_PERIOD_DEF_GID
Unique identifier
START_DATE
Identifies first day of data collection and cycle. If cycle_type is M or Y, rounds to beginning of the calendar entity, but data collection does not actually start until the specified date
CYCLE_TYPE
D = Daily, W = Weekly (7-day cycle), M = Monthly (calendar months), Y = Yearly (calendar years), O = Month to Month, E = Year to Year
CYCLE_COUNT
Number of time-periods as defined in cycle_type per cycle. 2 = 2 Days, 2 Weeks, etc.
DESCRIPTION
Long form description of time period definition
