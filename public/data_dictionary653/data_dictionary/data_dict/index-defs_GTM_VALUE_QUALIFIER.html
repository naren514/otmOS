Definitions - GTM_VALUE_QUALIFIER

Define Value Qualifier which will help use to define different value representation like total value, cost, duties and taxes etc...

GTM_VALUE_QUALIFIER_GID
Value Qualifier ID
GTM_VALUE_QUALIFIER_XID
Value Qualifier ID
GTM_VALUE_QUALIFIER_NAME
Value Qualifier Name
DESCRIPTION
Value Qualifier Description
GTM_VALUE_TYPE_GID
Value Type ID
GTM_VALUE_QUALIFIER_CLASS
It will use to define the different value representation like total value, cost, duties,per unit value and taxes etc... of value qualifier.
IS_MANDATORY_INDICATOR
This field is used by the landed cost adhoc screen to indicate if a value qualifier is mandatory or not
VISIBLE_ON_SIMULATOR
This field is used to render the value qualifier in the landed cost simulator adhoc screen
