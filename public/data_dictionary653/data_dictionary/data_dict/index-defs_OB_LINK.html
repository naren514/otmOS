Definitions - OB_LINK

The table holds associations between two order base objects. The relationship can not be circular i.e. it should not be possible to have a relationship OB1 - OB2 and OB2 - OB1.

PREV_ORDER_BASE_GID
Order Base Gid of the previous order in the life cycle
NEXT_ORDER_BASE_GID
Order Base Gid of the next order in the life cycle
