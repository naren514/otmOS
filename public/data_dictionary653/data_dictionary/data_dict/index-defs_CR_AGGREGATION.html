Definitions - CR_AGGREGATION

Summary for a Cooperative Routing aggregation run for a CR Project.

START_TIME
Time that the aggregation process started.
END_TIME
Time that the aggregation process ended.
NUM_SHIPMENTS
Number of shipments brought into the aggregation.
NUM_AGGREGATES
Number of Lane Volume Aggregates created during the aggregation.
NUM_OUTLIERS
Number of Lane Volume Aggregates that were identified as outliers during the aggregation.
NUM_FORECASTS
Nuimber of Lane Volume Forecasts created during the aggregation.
