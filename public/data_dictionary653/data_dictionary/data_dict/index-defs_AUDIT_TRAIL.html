Definitions - AUDIT_TRAIL

Table used to store Audit data

AUDIT_TRAIL_GID
Unique identifier of Audit_Trail table.
AUDIT_TRAIL_XID
DB Sequence.
NOTIFY_SUBJECT_GID
Workflow Notify Subject. E.g. ORDER - CREATED
NOTIFY_FUNCTION_GID
Workflow Notify Function. E.g. ORDER_EVENT.Object GID for which the lifetime event was generated
GL_USER_GID
User who made change to the data. Do not make GL_USER_GID a Fk. We need to keep the record in the audit table when the user is no longer a valid glog user
AUDIT_TIME
Local server time when the audit is recorded
MARKED_FOR_PURGE
A boolean field. When it is set to "Y", this row and its child data will be purged.
