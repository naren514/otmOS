Definitions - SKU_EVENT_QUANTITY

Optional quantity change attached to a SKU event.

SKU_EVENT_GID
SKU event id.
SKU_QUANTITY_TYPE_GID
SKU quantity type id.
SKU_QUANTITY_VALUE
SKU quantity update amount.
QUANTITY_UPDATE_TYPE
SKU quantity update type. Valid values are 'REPLACEMENT', 'INCREMENT', 'DECREMENT'.
