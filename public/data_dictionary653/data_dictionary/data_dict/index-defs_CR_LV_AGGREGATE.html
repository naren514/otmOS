Definitions - CR_LV_AGGREGATE

Individual aggregation of lane volume by date/shift.

CR_FORECAST_GID
The LV Forecast to which this aggregate contributes. This link exists even if the aggregate is an outlier.
CR_AGGREGATION_GID
The ID of the aggregation in which this aggregate was created. Null for user-created data.
START_DATE
Start date for shipments in the aggregate.
START_DOW
Start day of week (1 = Sunday)
START_DAY
Start day within cycle.
AVG_START_TIME
Number of seconds from midnight
END_DATE
End date for shipments in the aggregate.
END_DOW
End day-of-week (1 = Sunday)
END_DAY
End day within cycle.
AVG_END_TIME
Number of seconds from midnight
TOTAL_VOLUME
Total number of shipments for this aggregate.
IS_USER_EDITED
Indicates whether this aggregate has been edited/created by the user.
IS_OUTLIER
Indicates whether this aggregate is excluded as an outlier.
