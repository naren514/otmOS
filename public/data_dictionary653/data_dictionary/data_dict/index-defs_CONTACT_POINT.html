Definitions - CONTACT_POINT

Defines a contact point, with a communication method and a communication address, for a contact

CONTACT_GID
Contact associated with this contact point.
COM_METHOD_GID
Communication method ID.
COM_ADDRESS
Communication address; e.g. email address or phone number.
ALIAS
Display alias for the contact point.
IS_VALID_ADDRESS
If the address is invalid.
DELIVERY_WINDOW_PROFILE_GID
override the default delivery time window profile on the communication method
