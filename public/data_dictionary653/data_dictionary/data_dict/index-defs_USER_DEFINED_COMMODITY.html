Definitions - USER_DEFINED_COMMODITY

This table is related to the Item table. This is yet another way to define and categorize the Items in GC3.

USER_DEFINED_COMMODITY_GID
USER_DEFINED_COMMODITY_GID contains the unique identifier for the user-defined commodity that is being shipped.
USER_DEFINED_COMMODITY_XID
USER_DEFINED_COMMODITY_XID contains the unique external identifier for the user-defined commodity that is being shipped.
USER_DEFINED_COMMODITY_NAME
USER_DEFINED_COMMODITY_NAME contains the name of the user-defined commodity that is being shipped.
