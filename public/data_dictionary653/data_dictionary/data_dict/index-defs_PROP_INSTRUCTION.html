Definitions - PROP_INSTRUCTION

Property instruction

PROP_INSTRUCTION_SET_GID
Property Set GID
PROP_SEQUENCE_NUM
Sequence of instruction within the property set.
INSTRUCTION
Instruction type: SET = set a single or multi-valued property; REMOVE = remove a property value; INCLUDE = include another property set
KEY
Property key. For INCLUDE, this is the Property Set GID being included.
VALUE
Property Value. For REMOVE, this can be omitted. It should be set to remove a single value from a multi-valued proeprty. For INCLUDE, this is ignored.
DESCRIPTION
Description of the property instruction
