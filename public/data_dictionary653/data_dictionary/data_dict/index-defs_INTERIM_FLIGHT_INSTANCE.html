Definitions - INTERIM_FLIGHT_INSTANCE

This table is similar to the Flight_Instance table. It stores the actual flight information (at day level) associated with the Interim_Flight table.

FLIGHT_NUMBER
FLIGHT_NUMBER contains the airline flight number for the interim flight.
LEAVE_TIME_STAMP
LEAVE_TIME_STAMP contains the departure time of the interim flight.
ARRIVAL_TIME_STAMP
ARRIVAL_TIME_STAMP contains the date of the arrival of the interim flight.
SRC_LOCATION_GID
SRC_LOCATION_GID contains the unique identifier for the source location for the interim flight.
DEST_LOCATION_GID
DEST_LOCATION_GID contains the unique identifier for the destination location for the interiim flight.
SERVPROV_GID
SERVPROV_GID contains the unique identifier for the service provider.
VEHICLE_TYPE_GID
VEHICLE_TYPE_GID references the vehicle_type table.
