Definitions - TRANSPORTATION_WORKBENCH

LAYOUT_GID
The Layout GID provided by the user. This needs to be unique. What this means is that two users cannot share the same layout gid.
GL_USER_GID
gl_user_gid is deprecated as of 6.4.3
LAYOUT_NUMBER
This corersponds to the internal layout being used for a layout id for a user. gl_user_gid and layout_number form a unique constraint
LAYOUT_SEQ_NUM
A unique sequence number to identify the layout.
