Definitions - BULK_PLAN_PARTITION_BY_MODE

summary information of bulk plan partition break down by modes

NUM_ORDERS_PLANNED
number of orders planned in this partition and this transport mode
NUM_ORDER_MOVEMENTS_PLANNED
number of order movements planned in this partition and this transport mode
TOTAL_COST
total cost of all shipments built in this partition and mode
TOTAL_NUM_STOPS
average # of stops of all shipments in this partition and mode
TOTAL_SHIPMENT_TIME_SECS
Sum of all shipments times (end-start) in seconds
TOTAL_WEIGHT_CAPACITY
Sum of all shipment weight capacities (based on equipment used)
TOTAL_VOLUME_CAPACITY
Sum of all shipment volume capacities (based on equipment used)
