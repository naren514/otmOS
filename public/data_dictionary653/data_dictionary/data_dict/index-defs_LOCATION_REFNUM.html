Definitions - LOCATION_REFNUM

A reference to a location among the various application systems within a company, which may be either a location global identifier or a new location definition. For example, a location may be referenced as XYZ in the ERP app, 123 in the JDE app etc within the same company.

LOCATION_GID
LOCATION_GID contains the unique identifier for the location.
LOCATION_REFNUM_QUAL_GID
GC3 Global identifier for the entity. Is created by concatenating Domain Name,"." and XID.
