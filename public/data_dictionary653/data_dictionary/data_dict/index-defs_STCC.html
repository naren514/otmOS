Definitions - STCC

This table defines the STCC governmental code and name for the item.

STCC_GID
STCC_GID contains the unique identifier for the Standard Commodity Code (STCC) for the item being shipped.
STCC_XID
STCC_XID contains the unique external identifier for the Standard Commodity Code (STCC) for the item being shipped.
STCC_NAME
STCC_NAME contains the Standard Commodity Code (STCC) name of the item being shipped.
IS_49_CODE
Y/N, THE STCC CODE IS A 49 CODE AND APPLIES TO A HAZARDOUS MATERIAL .
