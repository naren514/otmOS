Definitions - ORDER_SCHEDULE_OVERRIDE

Override table for order delivery schedules.

ORDER_SCHEDULE_OVERRIDE_ID
System generated number for this schedule override.
ORDER_SCHEDULE_GID
ID for the parent order schedule.
OVERRIDE_NAME
Name of the override.
DESCRIPTION
Description of the override.
EFFECTIVE_DATE
Date the override becomes effective.
EXPIRATION_DATE
Date the override becomes ineffective.
