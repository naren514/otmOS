Definitions - WORKFLOW_TIME_WINDOW

A hierarchical table that specifies a duration of time. The workflow_granularity_gid specifies the time interval qualifier (such as day, minutes, seconds) while the duration specifies how long. Used by the PROCESS_CONTROL_REQUEST table.
